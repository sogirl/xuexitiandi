💡小白前置认知

在动手之前，必须搞懂的 6 个基础概念——搞懂这些，后面 14 节课就不会懵。

1. GPT-image2 到底是什么？

GPT-image2（也叫 Image 2 / GPT Image 2）是 ChatGPT 的图片生成能力。本套课几乎所有"出图"动作都靠它。它最大的两个杀手锏：

  • 文字渲染清晰——很多 AI 出图工具（如早期 Midjourney）生成的文字是糊的、乱码的，而 GPT-image2 生成的海报、PPT、UI 上的文字非常清晰可用（详见课 06、09）。
  • 能听懂"大白话"——你不用写复杂的英文提示词，一句"做一张天猫主图，细节多一点"就能出图（详见课 07、13）。
认知关键GPT 不只是画图工具！它同样能回答问题、生成文字、反推提示词。很多人只用它画图，浪费了一大半能力（课 05 重点强调）。

2. 为什么必须用"5.5 模型"？

课程反复强调：必须选择最新的 5.5 模型（出现在 01、02、03、04、05、06、09、11、13 等几乎所有课）。

  • 旧模型无法拆分图层 / 无法导出 PSD（课 01 的核心痛点）。
  • 旧模型在角色一致性、文字清晰度、细节丰富度上都明显更弱。
  • 5.5 模型 + 进阶模式，才能挂上 PS 插件导出分层文件。
新手第一大坑很多人卡在"分层失败""图糊""角色不一致"，根本原因就是没选 5.5 最新模型。

3. 为什么必须选"进阶"思考模式？

在 GPT 创建图片界面，"思考程度"有标准和进阶两档。课程几乎每一节都要求选进阶。

  • 进阶模式理解力更强：能更好保持角色一致性、风格统一、复杂构图。
  • 标准模式下，角色容易"换脸"、风格容易漂移、细节不够。
新手第二大坑（最坑！）每次刷新页面后，"进阶"会自动重置回"标准"！所以每一步操作前都要重新确认思考程度是"进阶"。这是 01、02 两节课反复强调的铁律。

4. 比例（画幅）速查表

不同场景要用不同比例，选错比例 = 出图方向全错。这套课里用到的所有比例：

比例用途出处

5. "8 张"是硬上限

GPT-image2 一次最多只能生成 8 张图片（课 02、05、06、09 反复提及）。这是工具的硬限制。

  • 需要超过 8 张？用"延展"指令突破：发一句"按照生成的图片延展说下面的另外 5 张内容"（课 05）。
  • 详情页用"往下面文章五张画面内容"延展（课 09）。
  • 建议一次生成 3~5 张做横向对比择优，不必每次顶满 8 张（课 03）。

6. 核心工作思路：让 AI 写提示词

整套课最精髓的方法论（课 05 首次提出，贯穿后半程）：不要自己憋提示词，让 AI（Gemini/GPT/豆包）帮你写。

你的需求（一句话） ↓ 丢给 Gemini/GPT/豆包 专业的结构化提示词 ↓ 粘贴给 GPT-image2 高质量出图
这是从"会用工具"到"精通"的分水岭新手自己写提示词又累又不出效果；高手让 AI 写提示词，自己只改"类目""风格"等关键字段。

01
一键海报生成 + 分层 PSD 文件导出
海报类 · 基础工具型
📍系列第 1 节🎯 学会 GPT-image2 基础 + PS 联动🛠️ GPT-image2 / PS / KS

本节目标

在 GPT-Image 2 里生成一张海报图后，导出为 可编辑的 PSD 分层文件——拿到 PS 里能随意调整图层顺序、文字颜色、字体。

核心工具

  • 主工具：GPT（GPT-image2 图像生成）
  • 辅助：Adobe Photoshop（PS）、KS（图层/素材工具）
  • 导出：PSD 分层文件 + ZIP 压缩包（含单独素材）

详细操作步骤

  1. 新建窗口设参数：复制课件提示词 → 新建窗口 → 模型选 5.5 → 思考程度选 进阶 → 选比例（3:4/9:16 等）→ 上传参考图点"生成"。
  2. 拆分图层：把生成的海报复制进去，粘贴下面这段"拆分提示词"（见提示词 A），等待执行。
  3. 安装 PS 插件：GPT 主界面 → "更多" → "应用" → 找到 Adobe Photoshop (add-on) → "加入链接"。
  4. 生成 PSD：回到拆分好的图片 → "更多"打开 → 输入提示词 → 勾选 logo → 选进阶 → 粘贴拆分提示词 → 生成。
  5. 下载编辑：下载 PSD 分层文件 → 拖进 KS 或 PS → 调整图层顺序 / 文字颜色 / 字体。

提示词原文

提示词 A · 拆分图层（核心）把生成的海报拆成若干个图像，每个元素成分一样，不要更改对应的一个位置。

易踩坑

三大失败原因① 没选 5.5 模型 → 无法分层；② 没选进阶模式 → 分层差或失败；③ PS 插件链接没挂 → 存不出真 PSD。
刷新陷阱刷新页面后"进阶"会自动变回"标准"，每一步都要重新确认。

ZIP vs PSD 怎么选？

文件类型用途
PSD 分层文件需在 PS 精细编辑、调图层/颜色/字体时用
ZIP 压缩包不需编辑、只要单独提取某个素材时用

知识逻辑

生成海报图（5.5 + 进阶 + 尺寸） ↓ 提示词 A：拆分图层 ↓ 安装 PS 插件（Adobe Photoshop add-on） ↓ PS 插件输入提示词 + 勾选 logo + 进阶 ↓ 下载 PSD / ZIP → 拖入 PS/KS 编辑
02
10 分钟搞定角色一致性 IP 提案到场景海报
海报/IP 类 · 角色一致性
📍系列第 2 节🎯 掌握"母版图 + 主题词"工作流📦 一次最多 8 张

本节目标

基于已有的 IP 角色母版图，10 分钟内快速生成多个主题场景海报，并衍生出三视图 / 表情包 / 手机壳 / 图标等整套 IP 物料。

核心思路：母版驱动

先有一张 IP 母版图，所有延展都基于这张图。进阶模式比标准模式在保持一致性上更强。

详细操作步骤

  1. 准备 IP + 提示词：起点是已有一张 IP 角色图。提示词要点——基于角色形象 + 角色名字（不要文字就不写名字）+ 场景主题。
  2. 上传 + 设参数：角色图粘贴进 GPT → 选"创建图片" → 模型选最新 + 进阶 → 写主题词 → 选比例（9:16）→ 开启生成。
  3. 体验物料延展：想切换成 IP 体验物料（三视图/表情包/手机壳/图标/徽章），用提示词 B + 改色调（亮/暗/绿色系）。
  4. 刷新卡住处理：刷新后"进阶"会重置为"标准"，必须重新改为进阶。
  5. 比例控制：100% 锁定比例用"创建图片"功能强制规定；不规定就沿用上次比例。

主题词决定场景方向

提示词写法效果
"刺激海报" / "主题海报"一次生成 4 张多样化场景
"沙漠海报"一次生成 1 张单一场景
不写角色名字画面不出现文字
"亮色调" / "暗色调"控制整体配色

提示词原文

提示词 A · IP 主题海报基于这一个角色形象，然后这个角色名字叫[名字]。 （不要文字就不写名字；想要多场景写"刺激/主题海报"；指定单场景写"沙漠海报/雪地/森林"等）
提示词 B · IP 体验物料延展亮色调（可改为暗色调/绿色系），可增删需要/不需要的内容。 扩展：把这张图片做成手机壳封面 / 图标 / 徽章都可以。
核心要点三大控制维度：场景（沙漠/雪地/森林）+ 色调（亮/暗/绿）+ 比例（9:16/16:9/3:4）。进阶模式 = 一致性保障。
03
GPT-Image2 快速交付 Logo 到品牌整套视觉方案延展
品牌类 · 一体化生成
📍系列第 3-4 例🎯 Logo + 字体 + 配色 + 视觉手册一套搞定📦 推荐 3~5 张

整套方案能交付什么

过去需要分开做的 Logo 设计、字体规范、配色规范，GPT-Image2 一次性输出：

Logo 标志本体

品牌核心图形

字体系统

字体大小、粗细规范

配色方案

品牌主辅色

视觉方案/手册

整体风格 + 物料延展

提示词核心结构（可编辑字段）

  1. 品牌信息：名字 + 品牌个性（专业/时尚/北方…）+ 额外要求
  2. 目标受众：明确方向（如卖化妆品 → 15~35 岁女性；潮玩 → 年轻化）
  3. Logo 方向偏好：列举 3 个喜欢的方向 + 讨厌的方向 + 视觉风格偏好
  4. 生成数量：默认 1，最多 8，推荐 3~5 个

两段提示词接力（核心工作流）

  1. 第一段 → Logo 视觉：复制提示词 → GPT 创建图片（5.5 + 进阶）→ 粘贴 → 得到含 Logo+字体+配色的视觉方案图。
  2. 截图 Logo：在 PS 打开方案图 → 选中想要的图标 → 复制。
  3. 第二段 → 品牌视觉手册：新建 GPT 窗口 → Ctrl+V 粘贴图标 → 创建图片 → 粘贴第二段提示词 → 得到品牌视觉手册（物料/包装/模特延展）。

后期调整（无需重新上传）

自然语言微调比例不喜欢 → 直接调比例；色调不喜欢 → 告诉 AI"换成深色系/简约时尚国际感"；要更多 → 要求"同样尺寸同样风格继续往下做"。换色时 Logo 主体保持不变，只切换色调风格。
实战经验给客户提方案，一次让 AI 输出 3~5 个版本横向对比择优；同一 Logo 可复用出横版/竖版/不同色调/不同延展，"图1+图2"堆叠即一整套品牌物料。
04
用 AI 重塑年代感照片 · 老照片修复案例实战
修图类 · 人物融合
📍系列第 5 例🎯 年代感生成 + 老照片修复 + 人物融合🔑 图一/图二顺序

能力一：年代感照片生成（门槛极低）

适用：街头/教室/学校/家庭/山区/群像/老电影感。操作：5.5 + 标准进阶 → 选比例（4:3/16:9）→ 写提示词 → 生成。

关键技巧给一个具体时间锚点（如"1996 年前后"），让 AI 有时代参考物。简单描述（"普通家庭的图片"）也能出效果。还可以反推提示词：给参考图让 AI 反推，再用反推提示词生成类似图，实现风格复用。

能力二：老照片修复（一句话搞定）

复制任意老照片 → 选标准进阶 → 点击生成。用自然重建技术英文提示词，一键修复面部细节。

能力三：人物融合 / 替换（重点实操）

想让图二人物融入图一场景、替换没到场的人、保持人物跨场景一致——核心就一句话：

核心 · 人物替换将图二的人物融入到图一的场景

AI 会把图二人物"抠出来"替换到图一位置，自动匹配图一光源（光线/明暗/色彩氛围），融合自然。

顺序非常重要！参考图（目标场景）在第 1 张 → 写"将图二人物融入图一"；参考图在第 2 张 → 改为"将图一人物融入图二"。流程：先生成目标场景图 → 下载/截图 → 再上传人物源图 → 粘贴提示词 → 发送。

高阶玩法：保留原人物 + 新增人物

不想完全替换，想保留图一原人物 + 新增图二人物？追加一句：

完整 · 保留+新增+指定位置将图二的人物融入到图一的场景，图一人物保留不改变，坐在后面左侧
  • "图一人物保留不改变" → 系统保留原图一人物原位置不动
  • "坐在后面左侧 / 在右侧 / 在左侧后面" → 指定图二人物位置动作
  • 不写位置 → AI 自动理解；写了 → 按指令摆放

知识逻辑

[年代感生成] 简单/详细/反推提示词 → 选比例 → 生成 [老照片修复] 上传 → 英文修复提示词 → 自动重建面部 [人物替换] 图一(场景) + 图二(人物) → "图二融入图一" [保留叠加] + "图一人物保留不改变" [指定位置] + "坐在后面左侧" 等自然语言
05
告别复杂提示词 · GPT-image2 快速生成专业 UI 界面
UI 类 · 方法论奠基课 ⭐
📍系列第 6 例🎯 提出"让 AI 写提示词"核心方法论📦 ≤8 张 + 延展

核心方法论（全套课的精髓！）

用 AI 写提示词，用 GPT-image2 出图，把用户从"自己写复杂 UI 提示词"中解放出来。

[ 类目想法 ] ↓ 用 Gemini/GPT/豆包 生成 UI 提示词（一句话模板） [ 专业结构化提示词 ] ↓ 粘到 GPT-image2（5.5/进阶/9:16/图片创建图片） [ 首张 UI 图 ] ↓ 加"延展 N 张 / 色调一致"（N ≤ 8） [ 一套 UI 组件库 ] ↓ 不够？再发"接着延展" [ 完整 UI Kit ]

端到端 SOP（四步法）

  1. 让 AI 写 UI 提示词：把模板句式丢给 Gemini，替换类目词（健身/生活/电商/处方/天气预报）。AI 输出一段专业 UI 布局描述。

关键心法① 模板化——提示词 1 是可换主题万能模板；② 一致性约束——"色调保持一致"是批量风格统一的关键；③ 迭代机制——"接着延展"突破数量限制；④ 工具组合——提示词工厂 + 图像生成器的组合拳。

两种方案（提示词 90% 相同）

方案一 · 复杂版

大量功能点图标、细节丰富，"卖点轰炸"式详情页

方案二 · 简约版

简洁、主体清晰，几乎不堆砌图标（仅多一句密度控制）

复杂版核心提示词（4 个要点）

详情页主提示词（要点拆解）[场景图] + [参考图]（可跨类目，手表+打气筒都行） 关键 1：输出的画面每一张每一屏的比例必须得是竖版（9:16） 关键 2：生成 8 张（GPT 一次最多 8 张） 关键 3：生成的图片不要出现页数（否则出现 1/8 水印） 关键 4：写明产品名（手表/智能手表/手机/耳机…） 配置：GPT-Image 2 + 5.5 思考模型 + 进阶 + 创建图片 + 9:16

延展 + 换色（不重传）

延展咒语（风格不漂移）往下面文章五张画面内容 （风格元素完全保持一致；不够再追加"再延展五张画面"，逐轮累加）
整体换色调（基础内容不变）把一开始生成的图片（8 张图片）变成为亮色调 后面延长的 5 张图片也变成亮色调 （也可换紫色调、可爱风、粉色调）

简约版唯一差异

简约版控制密度（单屏单重点）细节内容不需要太多，不用太多的图标，不用太多的功能介绍，尽量每一瓶只讲一个重点信息。

后期收口：筛选 + PSD + ZIP

三大原则① 比例硬约束——竖版不写，GPT 不按竖版输出；② 批量化思维——8 张起生成、5 张延展、ZIP 打包；③ 二次编辑兜底——不完美单帧走 PSD 拆分改字删元素最稳。
批量下载（极实用）把所有的图片打包成 ZIP 压缩格式给我。
一句话方法论GPT-Image 2 一次 8 张竖版分镜 + 一句"延展 5 张"，复杂版堆卖点、简约版单屏单重点，整体换色不重传，改字删元素走 PSD，批量下载一句 ZIP。

16 拍分镜实现精准动作分解与视频生成

📍系列第 11 例🎯 舞蹈/动漫/运动动作分解 + 转视频🎬 即梦/可灵

应用场景

16 拍分镜 = 通用动态动作分解工具，可做舞蹈、动漫、滑雪、足球等任意场景。

准备：模特图 + 三视图（建议）

模特图可用自己生成的或任意现成图（人物/IP/漫画）。三视图不强制但有更好——分镜出现背面/侧面动作时，有三视图效果更佳。

方法一：直接输入舞蹈名称

16 拍分镜主提示词[舞蹈名称/动作描述] 16 拍动作分解 四层四网格排版 16 个连贯动作
不知道舞蹈名？让 AI 帮你改写帮我修改提示词，换一个其他的 [国家/风格] 舞蹈。 数量不用改动，只需要修改文档内容和名称。
方法一限制只适用于"知名舞蹈"——名字写不出，模型也编不准。

方法二：上传视频 → AI 拆解动作（适合任意自创动作）

视频动作拆解（方法二核心）图一是我生成的图片分镜图，十六动作分解，四层四网格排版，16 个连贯动作。 你不用管图一，图一只是给我看我生成的图片， 我需要你帮我分析 [参考视频] 里面的动作，然后写出对应的提示词。 我只需要生成动作描述，不需要生成图片，也不需要出现视频里面的其他元素。

拿到 16 拍动作描述后，回填到分镜 prompt——去掉原"舞蹈名称"字段（否则强行套回原舞蹈动态），下方追加视频拆解的动作描述。

分镜图转视频（核心参数）

视频生成四必填项（铁律）① 图 1（16 拍分镜图）= 动作蓝图；② 图 2/3/4/5（三视图）= 人物蓝图；③ "精准还原 [动作名]"——锁定动作；④ "9:16"——锁定竖版。时长 12 秒。
即梦视频生成（老师主推 · 角色一致性强）参考图一（16 拍动作分镜图）的动作，结合图 2/3/4/5（三视图）， [追加 16 拍具体动作内容 / 具体舞蹈名称]， 作为蓝图进行 12 秒舞蹈演示。 比例 9:16。
可灵（VO/GVU）视频生成参考图一的动作，结合图 2/3/4/5， 精准还原 [具体舞蹈/动作名称]， 进行 12 秒演示。 比例 9:16。

三个视频模型对比

模型特点
即梦（推荐）角色一致性强、姿态还原度高，人物/角色基本不变
可灵 VO强制写实化（3D 也会变写实）
可灵 GVU保留原图风格（3D 不变写实），适合风格化分镜
三大工程化技巧① 提示词外包——拿 GPT 帮你改写舞蹈名+描述；② 蓝图+三视图双图输入——分镜给动作、三视图给人物，缺一不可；③ "精准还原[动作名]"必填。翻车兜底：太久就刷新一次重新拉取。

电影级分镜图生成与多视角角色一致性控制

📍系列第 12 例🎯 电影分镜 + 多视角角色一致🛠️ 即梦2.0/GPT5.5/豆包

两种用法（核心区别：是否提供参考素材）

用法一 · 纯文字

不提供任何素材，角色随机（男女老少、各种外观）。适合概念分镜、灵感验证。

用法二 · 三视图+参考图

角色保持比例/外张/身高一致。适合 IP 角色、品牌故事、固定主角。

核心工作流（5 步）

  1. 打开提示词模板：蓝/白/红分区——蓝色系统固定不改、白色可改、红色占位替换。
  2. 填写角色+动作：模板"你帮我按照图片中的文案重新描述一个[角色]"，填宇航员/快递员/F1赛车手等 + 动作（"在窗外"）。
  3. GPT5.5 生成分镜图：5.5 + 进阶 + 创建图片 + 比例 16:9 → 粘贴 → 发送。
  4. 用图生视频提示词转视频：把图发回 GPT 生成视频提示词（正标题必填、副标题/负面可选、视频比例必填）。
  5. 即梦2.0 生成视频：字数多只写正面提示词，字数少可加副标题/负面。

提示词模板

模板 1 · 纯文字角色描述你帮我按照图片中的文案重新描述一个[角色] （角色 = 宇航员/快递员/F1赛车手；可追加"这个宇航员在窗外"）
模板 3 · 三视图+游戏故事（用法二专用）图2 TUBG的一个游戏（不知道具体游戏可写"图二"） 让他去帮你生成一个宣传广告故事 / 游戏故事 / 某一个行业的一个内容
避坑① 即梦2.0 提示词有 2000 字上限，长提示词要精简；② 字数取舍优先级：正面提示词 > 副标题 > 负面提示词；③ 没有角色要求不要硬塞三视图，会适得其反；④ 模板蓝色部分（系统设定）不要改。

快速制作 360 度全景图与动态视频
特殊形态 · 全景图
📍系列第 13 例🎯 全景图 + 旋转视频 + HTML 查看器📐 16:9 → 1:2

核心价值

一张图即可让用户从任意角度观察场景——适合室内展示、游戏环境、虚拟漫游、产品 360 展示。老师称这是"目前做全景图最简单最方便的方式"，无需 3D 建模能力。

标准流程（5 步）

  1. 准备提示词/参考图：可用课件提示词，或大白话写，或用反推提示词。
  2. GPT 生成 16:9 初始图：上传提示词/参考图 → 选"创建图片" → 比例 16:9 → 生成。（注：反推提示词时不要选创建图片）
  3. 反推提示词（可选推荐）：上传图 → 选"我们的"（反推模式）→ 提交 → AI 返回对应提示词，可复用风格。
  4. 扩展为 360 全景比例：加上核心扩图提示词，把图扩成 1:2（2:1）宽图，只有 1:2 才能形成 360 环视。
  5. 清晰度增强（可选）：GPT 官网图偏糊时，用增强提示词基于原图重新生成更清晰版本。

三段核心提示词

提示词 B · 扩图提示词（关键中的关键）将此场景或者此图片转换为360度的据柱状投影图像 （变体：将此场景转换为360度的一个柱状投影；比例改 2:1）
关键避坑① 初始生成用 16:9，不要一开始就 1:2（内容会被压扁）；② "360 度柱状投影"是触发全景的关键短语，没有它就是普通横图；③ 2:1 和 1:2 是同一比例不同写法，指横向长图。

查看方式优点缺点

室内设计与多风格效果图生成
特殊形态 · 装修全流程 ⭐
📍系列第 14 例🎯 平面图 → 11 步装修全套图🔑 风格作上下文变量

核心思想

一张平面图 + 一套流程 + 灵活切换风格 = 一整套装修设计。把过去需要 3D 建模、手动渲染、多软件协作的流程，简化为纯自然语言对话。所有图基于同一套流程：上传平面图 → GPT 分析 → 依次输出各类图。

前置设置

必须① 选最新 5.5 模型；② 思考模式选进阶；③ 上传图时附一句"分析这个房子的户型"——模型会结合上下文分析采光/动线/功能区。

11 步完整工作流

  1. 户型分析图："画一张户型分析图，标注采光动线功能区"——直观看到户型优缺点。
  2. 全套施工图："生成一套全套的施工图"——简单描述即可。
  3. 风格效果图："北欧/日式/新中式…风格效果图"——风格变量从此注入上下文，输出含外观/颜色/材质/瓷砖。
  4. 价格清单图："基于 2025 年（指定年份）计算场景中所用价格"——显式声明年份（2026 未结束）。
  5. 分房间渲染图："根据前面的 X 风格，对每个房间进行设计效果，需要 5 张"——风格一致。
  6. 元素/风格解析图："场景放左边，文案放右边"——明确写出布局要求，控制版式。
  7. 局部布局图："设置局部布局图，室内家具和空间匹配"——知道买多大家具/预留位置。
  8. 3D 立体效果："生成 3D 立体效果"——过去要 3D 建模，现在用 GPT。
  9. 爆炸图："生成分析爆炸轴测图"——展示第一层/第二层/第三层及面积。
  10. 水电图："水电基础工程布置示意图"——电路/智能家居，颜色标记清晰。
  11. 多风格切换（高阶）：直接对话追加"根据前面的北欧风格，帮我换成另外 3 张风格（新中式、美式、地中海），一共需要 3 张"——风格完全不同，场景完全相同。

风格清单

北欧、日式、泰式、潮汕、中式、新中式、欧美、美式、地中海。切换时只替换"风格名称"字段，其他描述保持不变。

核心技巧风格作为上下文变量贯穿整轮对话；用"场景放左、文案放右"等明确版式指令控制版式；用"宽高比"控制画幅；用"指定年份"保证价格数据合理。

GPT + 即梦 2.0 生成的 3A 级游戏交互 Demo
特殊形态 · 互动视频
📍系列第 15 例🎯 几张静态图 → 3A 游戏交互 Demo🎬 GPT + 即梦2.0

核心思想

不自己写提示词让 AI 写；不自己拍视频用静态图 + 即梦 2.0 合成。案例：PUBG 枪战 Demo，可替换为赛博朋克、仙侠、萌宠等任何风格。

Step 1：GPT 生成风格化场景图（分页式）

让 AI 帮你分析每个页面的提示词我想要做一款上一级的 PUBG 游戏 demo 效果，我需要你帮我分析每一个页面的提示词，可以提供我参考 （可选物料：角色图/模特图、参考视频、纯文案或纯图片）
  • 前置设置：最新模型 + 进阶 + 16:9 横屏。
  • 分页生成：页面 1 → 2 → … → 6。每张都喂一张图（哪怕同一张），让风格在上下文向下延展。建议 5~6 张。
  • 内容修正：自动生成的页面若偏向视频内容，要求"重新帮我描述，让它更偏向枪战类游戏场景"。

Step 2：GPT 生成即梦 2.0 角色图提示词

生成即梦2.0 角色图提示词根据我前面上传的 6 张图片生成的图片，我要来做视频，根据我第 7 张图片的文案来进行一个观看
即梦2.0 硬限制① 字数 ≤ 2000 字（超了让 GPT 改短）；② 时长 ≤ 15 秒（提示词出现"16"等数字要改成"15"）。

Step 3：即梦 2.0 合成视频

  1. 前期设置：即梦 2.0 → 比例（竖 9:16/横 16:9）→ 秒数 15 秒。
  2. 上传参考图：依次上传 6~7 张图，按顺序上传（系统自动识别顺序），手动逐张点击确保位置准确。
  3. 输入提示词：直接复制粘贴，只需角色描述部分，删掉"角色开始"这行。
核心技巧用 AI 反写提示词；每张图都喂给 GPT 维持风格统一；即梦2.0 字数≤2000、时长≤15 秒；按顺序上传；删掉模板头只留角色描述正文。

⚠️十大踩坑点（新手必背）

这 10 个坑踩中任何一个，都可能直接导致出图失败——背下来，每次操作前默念一遍。

  1. 刷新页面会重置"进阶"为"标准"——必须每次确认（课 01、02）。这是最坑的坑，没有之一。
  2. 不挂 PS 插件 = 无法导出 PSD——要做分层 PSD，必须先在 GPT 应用里挂上 Adobe Photoshop 链接（课 01）。
  3. 不写比例 = AI 沿用上次比例——想 100% 锁定比例，用"创建图片"功能强制规定（课 02、09）。
  4. 不写"细节多一点" = 主图过简——生成电商主图时，这句是必加项（课 07）。
  5. 不写"5 张"或"8 张" = AI 漏页——生成 UI/PPT/详情页，明确写数量避免漏页（课 05、06）。
  6. 不写"不要页数水印" = 详情页出现 1/8——生成详情页分镜，必须显式写"生成的图片不要出现页数"（课 09）。
  7. 图片顺序与提示词不一致 = 人物融合失败——图一/图二顺序要对，提示词中的指向要匹配（课 04）。
  8. 不写"蓝图"+"精准还原" = 视频生成动作失真——图转视频时，这两个关键词必须出现（课 10）。
  9. 超过即梦 2.0 的 2000 字限制 = 视频生成失败——长提示词要精简，或让 GPT 改短（课 11、14）。
  10. 全景图初始就用 1:2 = 内容被压扁——先生成 16:9，再用扩图提示词转 1:2（课 12）。

  6. 06：Branch 把已有 UI 作锚点 + AI 写 8 段文案 + GPT-image2 出 PPT（4:3 或 16:9）。
  7. 07：主图生成 = "细节多一点" + 结构化提示词 + 局部保护换色。

  10. 10：16 拍分镜（4×4 网格）+ 双图（分镜+三视图）+ 即梦/可灵 → 9:16 / 12 秒视频。
  11. 11：电影级分镜（5 步流程）+ 即梦 2.0 + 2000 字提示词限制 + 三视图保一致性。
