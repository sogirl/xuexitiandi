PART ONE

广告底层逻辑与全面解析

课程第1篇 · 约27,095字符 · 整个课程的理论基石

1.1 广告的两大作用

★ 作用一：付费引流，增加曝光点击

广告的本质就是"付费引流"四个字。新品刚上架，没有流量，客户不知道我们，算法也没有收录，此时必须通过广告付费引流。直接效果链：增加曝光 → 增加点击 → 带来订单 → 带来信心。

★ 作用二：隐形订单（1+1>2效应）

广告除了直接带来的订单外，还能通过付费流量撬动自然流量。循环机制：广告带来流量 → 流量形成订单 → 提升关键词排名和类目排名 → 自然流量增加 → 自然订单增多 → 广告表现越来越好。

1.2 流量预算递增定律

预算 = A → 流量 = A  |  预算 = 2A → 流量 = 2A + B（B为额外增量）
阶段操作系统反馈
起步给$10预算（$1/流量=10个流量）花完了 → 说明产品受欢迎
第一次加加$5 → 总预算$15系统感知产品受欢迎 → 给17-18个流量
继续递增花完再加流量持续递增
✔ 正确做法：预算从$10开始，花完了再加$5，循序渐进。
✘ 错误做法：直接给$30预算 → 系统给30个流量 → 但长期花不完 → 系统推劣质流量 → 广告表现差。

核心原则：初始值循序渐进加，花完了再加预算，而非一开始就给足。

1.3 广告马太效应

广告会出现两极分化：

  • 表现差的广告 → 流量越来越少 → 订单越来越少 → 越来越差
  • 表现好的广告 → 流量越来越多 → 质量越来越高 → 订单越来越多 → 越来越好
实操指导：广告调了2-3次仍调不好 → 直接关掉，重新开一个新的。不要纠结"原广告有权重" → 表现差的广告只有差的权重。 "没有很差的广告，调了两三次都调不好，直接关了重新开"

1.4 不适合做CPC广告的五类产品

类型核心问题
超蓝海市场头部不开广告，价格压极低，后面链接无利润
客单价超低（<$7）ACOS = CPC/(客单价x转化率)，客单价低 → ACOS必然高
FBM自发货没有购物车，转化率极差
转化率超低低价标品<15%/非标<8%/高客单<3%
没有广告资格烟草类/纯佣金/当天受限/仓内受限
判断转化率是否达标：查看商机探测器中的均值x10作为参考基准。对比"同赛道商品"（同样材质、产品、场景、受众、功能、相近价格段）。

1.5 三个30%黄金原则

原则标准说明
毛利润≥30%(售价-财务成本-运费-最小折扣-佣金-配送费)/售价
广告订单占比≤30%（稳定期）广告订单:自然订单 = 3:7。降低占比靠推排名，不是削减预算
ACOS按品类控制高客单<30%、低客单<45%、推排名可50-60%

1.6 广告搜索排位法则

位置曝光能力
第一页首页（特别是首行）100%
第二页30%
第三页15%
第四页及以后几乎不曝光

核心结论：广告打到前三页才有效，自然排名同理。

1.7 广告实际扣费与质量得分机制

实际CPC = 下一位出价 x 下一位质量得分 / 你的质量得分 + 0.01
质量得分维度权重影响因素
相关性—产品/关键词/节点相关性
点击率★ 80%主图+价格，20%标题主图吸引力、价格竞争力
转化率—价格、促销、页面质量、A+内容
广告历史表现—表现越好 → 权重越高
关键认知：质量得分决定了你的实际CPC（与出价无关）。点击率是质量得分中最重要的维度。

1.8 广告投放结构（单词/多词配比）

关键词类型ABA排名每活动投词数策略
流量大词<2万1个词单词单活动单组
流量中等词2万-10万5个词左右筛词测词
流量小词>10万10个词左右多词一活动

否定词判断：点击次数达到出单次数的2倍仍不出单 → 可否定。

1.9 广告的五大目的

目的对应广告类型特点
推排名SP关键词精准广告ACOS可能高，但自然排名上升
拓词拓流SP词组/广泛、自动广告低ACOS，但不推排名
提升关联流量SP/SB/SD的ASIN定位关联内部流量
盈利低竞价捡漏、低预算游击战ACOS漂亮但自然流量少
关键认知：全是推排名广告 → ACOS很烂；全是盈利广告 → 自然流量起不来。必须各种目的都有 → 实现均衡。

1.10 瀑布流打法与简单打法

  • 瀑布流打法：同一种广告活动给不同竞价 → 覆盖不同流量入口 → 快速测试最合适竞价
  • 第二种定价策略：$0.02竞价 + 动态降低提高 + 加成900% + 预算$10 → 等效$0.2

2. SP广告5种高阶打法

课程第2篇 · 约24,533字符

前置：标品、非标品与半标品

类型特征出单特点案例
标品规格型号明确、外观相似出单词集中水杯、笔、键盘

半标品有款式属性但出单集中出单较集中，吃款式头巾、墙纸

五大打法核心对比

维度高效激进稳健全面流量全覆盖渐进式捡漏
预算$150+$100左右$100+$50+$5-10/组
适合产品标品/半标品所有产品所有产品CPC贵/预算低所有产品
推品速度最快（15天首页）较快（2个月内）辅助较慢不适用
ACOS表现较差较好取决于测试好很好
能否做全部架构✓✓✗只能辅助✓✗只能辅助

打法一：高效激进打法

架构：自动广告（1组）+ 手动精准（2组）
      关键词1（核心大词）+ 关键词2（核心大词）
操作：预算$150+，关键词:自动=8:2
     花完加$5/天，连加一周 15天上首页
     上首页后降预算，匹配给Q3/Q4，开词组拓词
     第6-7周报BD冲排名

打法二：稳健全面打法

架构：核心大词精准2组(推排名,目标前3页)
      长尾关键词精准3组(高转化+推排名)
      自动广告1组 + ASIN定位1组 + 类目定位1组
操作：预算$100左右，大词推前3页即可，ACOS更好看

打法三：流量全覆盖打法

  • 同一词x不同匹配方式（精准/词组/广泛）
  • 同一词x不同竞价（高/中/低）
  • 注意 只能作为辅助，不能作为广告架构全部

打法四：渐进式打法

  • 先打长尾词 → 积累权重 → 再打大词（CPC降低）
  • 分两阶段各5个长尾词 → 约30天 → 再开大词
  • 适用：CPC贵（$3+）+ 预算有限 + 可接受较长周期

打法五：捡漏打法

  • 低竞价$0.2-0.3 → 精准/词组/广泛/自动均可
  • 每个活动$5-10 → 花完不加 → 表现好也不加
  • 唯一考核ACOS → 不看点击率转化率
  • 重要 必须配合推排名广告

选择打法的决策树

你的产品是什么类型？
+-- 标品/半标品 + 预算$150+ + 追求速度 高效激进
+-- 所有产品 + 预算$100 + 追求平衡 稳健全面
+-- CPC很贵($3+) 或 预算有限 渐进式
+-- 需要测试最佳配置 流量全覆盖（辅助）
+-- 想要纯盈利辅助 捡漏打法（辅助）

5. 出单词集中和分散广告架构搭建策略

课程第5-6篇

SP广告六大目的

  1. 引流增量
  2. 推排名（精准匹配最有效）
  3. 自动广告跑词
  4. 拓词拓流（词组、广泛）
  5. 打关联（ASIN定位、类目定位）
  6. 盈利性广告（捡漏、品牌广告、SD、ASIN定位）

关键词分类标准

分类ABA排名分类ABA排名
大词2万以内中下等词5万-10万
中上等词2万-5万长尾词10万以上

按相关性：查看前10个自然位中相关产品数量，≥6个→强相关；3-5个→一般；1-3个→弱。

按修饰词：功能词、花色词、场景词、形状词、材质词等。

出单词集中（标品）四种打法

  1. 高效激进（$150+）→ 15天上首页
  2. 稳健全面（$100左右）→ 均衡架构
  3. 流量全覆盖（$100-150+）→ 注意否定规则
  4. 渐进式（预算少）→ 先小词后大词

否定规则：开了精准→把精准词本身在词组和广泛中否定掉；开了词组→把词本身在广泛中否定掉。

出单词分散（非标品）三种打法

标准打法（$150+）约14-15个组
  1. 自动广告（紧密+同类）
  2. 手动精准2组
  3. 手动词组2组（5词/组）
  4. 手动广泛2组
  5. ASIN定位（价高10个+评分低10个）
  6. SP商品集（垂直细分才开）
  7. SBV必须开（非标品即使没评论也可能产生购买）
  8. SD广告（VCPM获取广泛曝光）

预算少的打法（$40-50/天）：自动+手动词组+ASIN定位+广泛组=4个活动。

关键操作原则

  • ★ 分时调预算，不调竞价（调竞价会破坏广告模型）
  • 否词原则：少量多次，不要一下子否太多
  • 互补产品词：前期否掉，稳定阶段不要否

类型核心操作

拓词拓流核心方法

  • 自动广告：只留紧密匹配+同类商品
  • 关键词词组和广泛

否词原则：词根跟产品没关系→词组否定；词根有关系但变体不匹配→精准否定；别人品牌名形成转化→不用否定。

出单词再投放：ABA排名<2万→3-5个一组；5万-10万→10个一组；>10万→10-20个一组。

两个极端都不可取：全部精准推排名→ACOS差但排名上去了；全开盈利型→ACOS好但没有自然排名，广告占比70%是危险信号。正确做法：要平衡推排名组和盈利型组。

PART SIX

9周推品SOP

14. 标品9周SOP

核心打法：自动+2个手动精准。出单词集中，推核心词自然排名到首页。

周操作
W1中高价+40-50%优惠券，自动+2手动精准，起量20单
W2降价触发红标（连续7天+超20单+降价>10%）
W3优惠券下调到20%，拓展广泛+长尾+ASIN
W4优惠券降到10%，ACoS控制+出单词再投放
W5拓词+竞价优化，精细化运营
W6七天秒杀：Day1爆单，后3天不动广告链接
W7秒杀收尾+30%优惠券，出单词再投放
W8限时秒杀+BD，SB/SD扩展
W9-12三项核心循环持续优化

15. 非标品9周SOP

核心打法：广泛匹配+ASIN定位。加预算容易变差，宁可多开活动不加预算。

周操作
W1自动瀑布流3个+词根广泛2个+长尾2个+ASIN定位2个
W2拓展更多大词广泛+ASIN扩展
W3手动精准入场+大词广泛+50个长尾广泛
W4ACoS控制+出单词再投放+精准否定
W5竞价优化+自动出单词再投放
W6七天秒杀（约2倍，非5-10倍）
W7预算不动+出单词再投放
W8SB视频+SD+SB商品集扩展
W9-12三项核心循环
★ 任何活动预算不超$15-20 | 堆词加速词收录 | 30-40单足够 | 十几单有利润
CORE COGNITIONS

最重要的30条核心认知

  1. 广告预算花完了再加，不要一开始给足
  2. 调不好的广告（2-3次）直接关新开
  3. 广告打前三页才有效，自然排名同理
  4. 质量得分决定了你的实际CPC（与出价无关）
  5. 健康架构必须有推排名+拓词+关联+盈利
  6. 降低广告占比的正确方式是推排名
  7. 毛利润≥30%是广告盈利的前提条件
  8. SP高效激进打法15天核心词可上首页
  9. 打法选择取决于产品类型和预算
  10. 渐进式核心：长尾词积累权重大词CPC自然下降
  11. 捡漏必须配合推排名广告
  12. SBV是转化最好的广告类型
  13. 品牌广告广泛匹配加修饰词"+"号
  14. 品牌广告时机：评论50+且核心词到首页
  15. SD是主动型广告，内容相关投放效果最好
  16. vCPM的漂亮ACOS不真实
  17. 交叉销售：低价投高价可以，高价投低价不行
  18. 标品集中打精准，非标品分散打广泛
  19. 分时调预算，不调竞价
  20. SBV+广泛+加号=ACOS可控很低
  21. 转化率不达标什么都不选
  22. 低价标品+CPC高+转化低=翻不了盘
  23. 多表交叉验证
  24. 搜索词展示量份额报告最重要
  25. 六色标注法6类词对应明确动作
  26. 合理ACOS=大词建议竞价x出一单点击/客单价
  27. 标品核心：自动+2个手动精准
  28. 非标品加预算易变差宁可多开活动
  29. 标品秒杀翻5-10倍非标品约2倍
  30. 三项核心循环每周执行
