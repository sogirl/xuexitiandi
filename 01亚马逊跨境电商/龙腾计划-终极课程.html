1.2 自然排位规律与超级爆款打造

BSR排名与自然位的关系

BSR排名首页曝光位数第二页曝光位数特征

71-400名--约230个产品争6个位置

关键词自然排位核心因素

  1. 点击转化率 → 最重要的因素
  2. 关键词下出单量 → 飞轮效应：出单越多→排名越靠前→曝光越多→出单越多

关键结论

  • 进不到前三页 = 吃不到流量（第一页100%→第二页30%→第三页15%→第四页后几乎无）
  • 最低要求：推到第三页，给广告预算推动
  • 进前20名是类目布局的门槛
  • 进前20名后品牌会被收录到左侧参数栏品牌筛选
  • 前20名可利用BD（秒杀）加速

Review获取

  • 前20名必须关注review模块
  • 锁屏邮件方式：通过ERP工具在客户收到产品5天时发送邮件
  • 不要提review/feedback内容
  • 明信片方式已爆雷，不建议使用

其他流量入口

  1. 四星以上产品标识 → 仅自动广告能匹配，仅时尚品类，要求评论数100+、四星以上
  2. Amazon's Choice标识 → 在该词下出单最多的产品
  3. 气候友好标识（绿标） → 费用约万把块钱，提升转化率5%-10%

1.3 关联流量体系化梳理

关联流量是仅次于搜索流量的第二大块流量。

关联流量分类

类型具体形式可控性
自然关联看了又看、买了又买、Compare with Similar低（需长期积累）
付费关联ASIN定投(SP/SB/SD)高（可主动操作）

各关联流量入口详解

1. 看了又看(Customers Also Viewed)

  • 操作：筛选相关性高的竞品 → 做ASIN定投 → 长期积累（从50→100+）

2. Compare with Similar

  • 条件：评论数接近 + 价格相近 + 参数相似 + 势均力敌
  • 头部链接只与头部相互Compare

3. 买了又买(Frequently Bought Together)

  • 核心特征：互补关系
  • 关键规则：低客单不要定位高客单（如卖手表不要定位钢化膜），应反向操作
  • 找服务商操作加购效果最快

4. Make It a Bundle（虚拟捆绑）

  • 可自己操作！后台设置
  • 虚拟捆绑商品矩阵：新品A与10个老品B相互虚拟捆绑 → 10个老品都为新品引流

5. 商品页面第一屏的广告位

  • 所有广告类型都会出现在此位置（不仅ASIN定投）

ASIN定投守护流量策略

  • 垂直细分领域：自己定投自己（ABCDEF链接相互定投）→ 守流量护城河
  • 自己定投自己的ASIN → CPC更低

SD广告的关联流量位置

SD广告90%的曝光出现在三个位置：

  1. 标题上方
  2. 五点描述下方
  3. 购物车下方

1.5 类目布局方法论

类目布局意识

  • 类目布局 = 类目垄断
  • 一个品类至少布局5-6个链接
  • 实现类目垄断至少需要16个链接
  • 风险意识：排名越靠前越危险 → 面临恶搞

不同排名位置的布局策略

排名区间布局数量目的
1-5名1个占坑位，可能不盈利
5-20名1-3个头部后备链接
20-50名2-3个稳定盈利
50-100名若干稳定盈利

差异化布局策略

  1. 数量差异化：如红丝圈做五件套变体
  2. 尺寸差异化：迷你/小号(低价引流) + 大号(高利润)
  3. 款式差异化：不同外观给不同喜好客户更多选择
  4. 功能差异化：单一功能 vs 多功能

价格布局策略

⚠️ 顺序：先中等客单 → 再低客单 → 再高客单
  1. 先中等客单入局：参考同类商品大部分定价
  2. 低客单冲排名：不盈利，纯粹冲排名 → 转化率极高 → 快速上排名后弱化
  3. 高客单后布局：等至少5个链接成熟后 → 联系供应商开模差异化

流量闭环与垄断

  • 多个链接通过广告+自然关联形成相互引流 → 商业流量闭环
  • 前4个广告位被自己占据 → 竞品无法进入

1.7 大件商品布局与大卖模式

标准件布局

  • 3个运营 + 3个店铺 + 同时推同一产品
  • 1个链接推起来后 → 复制再推到5个 → 形成SOP

大件商品的双拼模式（洛阳7亿级大卖案例）

📊 核心架构：
  • 工厂团队与运营团队分开
  • 工厂加5-8个点卖给运营团队 → 工厂获得固定利润
  • 产品发到海外后 → 工厂再赚退税（十几个点）
  • 运营团队目标：销量提成

运营操作：

  • 10个运营同时上架10个店铺
  • 每个运营分400套
  • 赛马机制：表现好的集中资源，其他重新选款
  • 统一定价权和促销权
  • 风险对冲：分销平台+多电商平台兜底

2.1 产品分类体系——标品、半标品、非标品的定义与判断

标品的三大核心特征

特征说明案例
外观千篇一律同质化严重鼠标键盘、茶杯、3C类
功能属性>个人喜好购买注重使用功能不会因为喜欢不喜欢而决定
客户有清晰认知明确购买意向知道产品长什么样

流量特征：流量入口有限、出单词集中、CPC高（>1美金）、受众广泛

非标品的三大核心特征

特征说明案例
外观千差万别款式非常多衣服、鞋子、假发、珠宝
个人喜好>功能属性先看好不好看买裙子先看款式
客户无清晰认知无明确购买意向只知道场景/人群

流量特征：流量入口多、出单词分散、CPC低（<1美金，0.6居多）、受众细分

半标品的判断标准

维度特征
搜索结果千差万别（非标特点）
出单词集中（标品特点）
本质兼具标品和非标品属性

典型案例：台灯(table lamp)、碗/茶具、吸顶灯、桌布、袜子

快速判断流程

产品 → CPC>1? → 出单词集中? → 搜索结果相似?
  → 全是 → 标品
  → CPC低 + 出单词分散 + 千差万别 → 非标品
  → CPC高 + 出单词集中 + 千差万别 → 半标品
🔑 核心意义：不同产品类型 → 完全不同的广告策略和推品思路

2.3 非标品的三阶段广告架构

⚠️ 核心区别：非标品不能开精准 → 以词组/广泛为主。
原因：非标品客户个人喜好，转化率天然就低，CPC也低 → 精准效果差

第二阶段（第一周后）广告架构

广告组操作匹配方式目的
表现好的自动广告增加预算-扩大流量
接力赛广告开同内容新广告活动，错峰投放同上解决加预算变差
核心大词词组或广泛广泛优先拓词拓流
长尾词（三类分法）分类投放词组或广泛高转化+推排名
ASIN定投出单/BSR前100定投关联流量

接力赛广告详解

📌 定义：一个广告活动表现好，加预算后效果变差 → 恢复原预算 → 开一个一模一样的广告活动，从原广告花完的时刻启动 → 两个广告交替运行

操作步骤：

  1. 自动中等竞价表现好 → 加预算 → 发现花得快但单量没增加 → 恢复原预算
  2. 看预算报告发现投放活跃时间不到70%（比如美国时间下午5点花完）
  3. 开一个一模一样的广告活动 → 从下午5点启动
  4. 两个广告互不影响 → 增加总曝光
  5. 链接多时用ERP工具分时调预算

非标品长尾词三类分法

  1. 带同义修饰词的长尾词 → 分类投放
  2. 带相同词根的长尾词 → 分类投放
  3. 带相同关键词的长尾词 → 分类投放

非标品持续拓词

动作具体操作
跑出的新词出单ASIN投放
自动广告分组主脉变体：一个变体一个自动；非主脉：3-5个变体一个自动
持续拓词筛词非标品可能有20-30个广告活动
🔑 非标品核心策略：广泛 > 词组 > 精准（精准最差）

2.4 广告优化方法论 ⭐核心

优化决策树

场景操作

不推排名 + 高ACOS下调竞价0.1或10%
下调竞价后花不出去恢复原状 → 降低预算10%-20%
无ACOS + 点击少(预算少)加预算到"应出一单所需预算"
无ACOS + 点击少(预算充足)竞价过低 → 加竞价到至少80%的花费率
无ACOS + 高点击不出单点击次数≥应出单点击次数×2 → 否定该词

关键公式

🔑 应出单点击次数 = 1 ÷ 转化率
例：转化率10% → 应出点击次数=10；转化率5% → 应出点击次数=20

高低ACOS临界点

🔑 公式：毛利润率 + 15% = 高低ACOS临界点

例：毛利润30% → 临界点 = 45%
• ACOS < 45% → 正常/盈利性广告
• ACOS > 45% → 高ACOS需要优化

为什么加15%？
• 广告本质是付费引流 → 需要容错率
• 有些组ACOS 30%，有些60-70% → 综合考虑给15%偏差值
• 如果所有>毛利润的都调 → 广告会被调坏

推起来的定义与稳定期

  • 推起来 = 上首页（或达到预期目标位置）
  • 稳7天左右再降预算
  • 降预算后会掉 → 正常，但不会掉到第二页第三页以后

  • 效果好定义：达到最低库存的2-3倍

秒杀期间的操作节奏

  1. 秒杀前：开大额优惠券（提前2小时开）
  2. 前2小时：进度条跑到90%
  3. 跑90%后：关掉/调小优惠券 → 给买家紧迫感
  4. 盯库存进度条：跑到95% → 加3-5个库存
  5. 盯一晚上

第三篇：根据广告目的搭建广告架构

📌 课程：003-根据广告目的搭建广告架构 🎯 定位：七大广告目的体系——每个广告活动都必须有明确目的 ⚠️ 前置：第二篇 标品/非标品广告架构

3.1 广告的作用与本质

广告的基本作用——付费引流

曝光 → 点击 → 订单 → 销售额

  • 订单就是信息：新品刚上架时，广告订单给卖家推品信心

流量预算递增定律

📌 定律：广告预算从低到高递增 → 花完说明产品受欢迎 → 加预算后亚马逊推额外流量 → 2A预算带来2A+B的流量

• 预算花完 → 系统认为产品受欢迎 → 额外推流
• 预算花不完 → 系统不会额外推流
• 核心原则：预算一定要从低到高递增（5→10→20→30）

广告的马太效应

两极分化：表现越好→流量越多→CPC可能降低；表现越差→流量越少→ACOS越高

广告权重问题：

  • 表现好的广告活动权重上升（0→1→2→3）
  • 表现差的广告活动权重可能是负值
  • 核心建议：调了3次效果依然差的广告活动 → 果断关掉 → 新开

3.2 不适合做广告的产品类型

超蓝海市场小类

  • 第五名单量不到30单
  • 头部链接非常稳定 → 几乎不开广告
  • 头部把价格压得很低
  • 结论：这类品类没办法搞

客单价<10美金的超低客单产品

客单价9.7美金，CPC=0.5，转化率=8.7%

ACOS = 0.5 ÷ 9.7 ÷ 0.087 = 59% → 没法做

各客单价转化率门槛

客单价区间标品最低转化率非标品最低转化率
<20美金(低价)≥15%较低但仍需关注
20-80美金(中等)≥8%≥5%
>80美金(高客单)≥3%≥3%

其他不适合的情况

  • FBM非大件产品 → 转化率极差
  • 链接质量低（文案未极致优化）
  • 产品质量问题/客诉 → 差评/评星低于4.0
  • 没有购物车

3.3 七大广告目的体系 ⭐最核心框架

🔑 这是整个课程最核心的框架之一——每个广告活动都必须有明确目的！
序号广告目的对应广告类型说明
1付费引流/提升销量所有广告类型最基本目的
2推动排名SP关键词精准最有效；SB微乎其微
3拓词拓流SP词组/广泛、自动广告让自动跑出更多词
4提升关联流量ASIN定投、类目定位定位出单/强者/势均力敌/弱者ASIN
5盈利类目定位(CT)、SBV、低竞价捡漏一石三鸟
6扩大流量出单ASIN投放、出单词投放表现好的扩大
7搜索流量关联防护品牌名+关键词防止竞品抢占品牌搜索结果

各目的详解

目的2：推动排名

  • 仅SP商品推广的关键词广告有效
  • 精准匹配 > 词组 > 广泛
  • 点击率>0.7%说明推排名有效；<0.3%说明大部分展示在商品页面（无效）

目的4：提升关联流量

ASIN类型定义优先级
出单ASIN广告报告中出单的ASIN最高优先
强者ASINBSR榜单前100名中比我们强的高优先
势均力敌ASIN前后排位10-20名以内中优先
弱者ASIN出单不如我们的低优先但也要做

目的5：盈利型广告

  • 类目定位(CT) → 同时实现关联+盈利+增量三种目的
  • 低竞价捡漏广告：竞价0.02 + 首行加成900% → 最多0.2
  • 捡漏广告开4-5个组 → 有2-3个能捡到就很漂亮

目的7：搜索流量关联防护

  • 品牌名+关键词投放 → 守护搜索结果页面
  • 自己用品牌名+关键词投放 → 占据前4广告位 → 防止竞品进入

七大广告目的的层次关系

广告目的体系
├── 基础目的：付费引流/提升销量（所有广告都实现）
├── 增长目的：推动排名 / 拓词拓流 / 提升关联 / 扩大流量
├── 盈利目的：类目定位 / SBV / 低竞价捡漏
└── 防护目的：品牌名+关键词防护
🔑 核心认知：
  • 同一广告可实现多种目的（如类目定位=关联+盈利+增量）
  • 任何广告评判标准是"有没有达到目的"而非ACOS
  • 广告架构 = 七大目的的有机组合

3.4 广告目的实操分析

广告分析的标准流程

  1. 按订单降序排列 → 识别增量型广告（中流砥柱，不准动）
  2. 标注每个广告的目的 → 判断是否实现
  3. 看ACOS优化 → 只动非中流砥柱
  4. 看点击率判断位置 → 判断推排名是否有效
  5. 看数据量 → 数据小继续观察
🔑 增量型广告的核心原则：
"不管发生什么天崩地裂的事情，不准动增量型广告。主要贡献出单的广告一旦调整，差的结果谁都承受不了。"

诊断方法

  1. 看点击率 → >0.7%在搜索结果页面(有效) / <0.3%在商品页面(推排名无效)
  2. 看转化率 → 与平均线比较
  3. 看数据量 → 数据太小不具说明性

4.4 标品广告三阶段（大件家居版）

阶段一(SP)：关键词权重建立

二级词(最精准长尾) → 一级词(中等精准) → 大词(精准) → 自动+防御+进攻
  • 从二级词开始（最精准长尾词）→ 用广泛匹配 → 测试基础CPC
  • 加大流量 → 一级词 → 同样广泛匹配
  • 打核心大词 → 精准匹配 → 加重权重
  • 自动紧密 + 防御(SP投自己) + 进攻(打竞对ASIN)
📌 为什么先打长尾词？
  1. 成本更低
  2. 长尾词收录帮助在大词获得权重
  3. 新品直接打大词 → 基础CPC竞价就高 → 后续打其他词也偏高

关键数据：

  • SP广告买顶部位置：新品6-7美金/点击，老品5美金多
  • 核心产品日广告预算：3000-4000美金/天

阶段二(SB)：拉升单量冲排名

  • SB CPC比SP低30-70%
  • SBV置顶约1.7美金(SP需5-6美金)
  • 缺点：出单点击对关键词自然权重提升不多

阶段三(SD)：转化率提升

  • 核心理念：20%转化率 = 80%流量跳失率 → 流失流量再打捞
  • 浏览再营销（7天/14天/30天）
  • 防御：SD投本品链接ASIN → 占据五点描述旁广告位
  • 复购投放：复购率高(20-30%) → 打60-90天

🔑 优先级排列：测款 > 推新品 > 冲排名 > 清库存 > 补单量 > 品牌知名度
  1. 测款：如果站外都放不出去，就不要做了
  2. 推新品：适合高客单/高利润/季节性产品
  3. 冲排名：站外流量大但转化低→绝对订单量可观
  4. 清库存：库存多→BD+站外方式
  5. 补单量：广告出单不够时站外补齐
  6. 品牌知名度：DO站/PR编辑网站

11.2 社媒折扣操作

阶梯式折扣策略

  1. 前5-6小时：仅50%社媒折扣
  2. 无效果：开10%优惠券（=50%+10%=60%）
  3. 仍无效果：开到20%优惠券（=50%+20%=70%）
  4. 爆单出现→立马取消
⚠️ 关键安全规则：
  • 关站外后必须等4小时才能开新优惠（防止叠加零元购）
  • 社媒促销仅美国站有；购买折扣全站点可用

11.3 渠道与自建体系

  • YouTube红人自主联系（回复率约20-30%）
  • 独立站品牌官网（成本几百元/年）
  • Facebook主页商店（免费引流）
  • 多平台布局（沃尔玛/eBay/TikTok）

第十二篇：站外配合站内打造爆款

📌 课程：012-站外配合站内打造爆款 🎯 定位：站外放量的精细化操作+20天趋势打造 ⚠️ 前置：第十一篇 站外方法 + 第一篇 秒杀策略

12.1 核心框架

  • 站外是补充型放量手段，不是主力
  • 前提：站内活动价、广告三个层面都做好后

12.2 放量效果评估与趋势打造

🔑 核心标准：第二天放量后的销量 ≥ 前一天日常销量的110%

20天周期趋势打造：
  • 每次放量都要高于上次放量值的10%
  • 站外后1-2天内必须有LD/7DD配合

12.3 渠道实操要点

  • 必须用Social Media Code（联盟生态加持）
  • Facebook群筛选：5万成员以上，保底10个群
  • Slickdeals：750美金/帖，官方编辑费，不找水军

12.4 利润测算

  • 站外亏损 = 站外定价 × 站外单量
  • 利润回赚 = 自然单量提升 × 正常定价利润
  • 利润健康线：Tcos 8%-15%

第十三篇：出单词集中型12周推品SOP

📌 课程：014-出单词集中型12周推品SOP 🎯 定位：标品/半标品的完整12周操作流程 ⚠️ 前置：第二篇 标品三阶段 + 第三篇 七大广告目的

13.1 准备阶段——八大检查项

  1. 关键词库——"没有关键词库，推品失败就是活该"
  2. 标题写法——想推的词+英文逗号+品牌名+关键词+属性+场景
  3. 图片——与10个竞品对比
  4. 高级A+——六大模块缺一不可
  5. 视频模块——五类缺一不可
  6. 定价——中高起步
  7. 类目选择——利于拿BS
  8. ASIN定投规划

13.2 第1-5周操作SOP

周次链接操作广告操作
第1周高价+购买折扣20%+种子链接自动+2个核心大词精准+首行加成50%
第2周降价10%→降价红标+专享折扣10%好词每天加5美金预算连加7天
第3周保持5大词+20长尾词+手动测词
第4周关购买折扣大词广泛+出单词再投放+准备秒杀
第5周保持ASIN定投+长尾词20-30个+核心词10-15个广泛

13.3 第6-12周操作SOP

周次核心动作
第6周BD七天秒杀→预算翻倍→第四天降→秒杀后"三不动"3-5天
第7周20-30%优惠券+出单词再投放+长尾扩展
第8周LD限时秒杀+SB广告开启
第9-12周最终架构：精准+自动+ASIN定投+SB+SBV+长尾广泛

13.4 CPC不可接受时的替代方案

  • 先走广泛路线→积累评论→SB广告→15单后报BD→回到主SOP

第十四篇：出单词分散型12周推品SOP

📌 课程：015-出单词分散型12周推品SOP 🎯 定位：非标品的完整12周操作流程 ⚠️ 前置：第二篇 非标品三阶段 + 第一篇 关联流量

14.1 核心理念

  • 出单词分散=非标品，不追求打爆款
  • 推广策略：覆盖尽可能多的流量词→每个词贡献2-3单
  • 多链接布局 > 单品打爆

14.2 四个初始周期（第1-4周）

周次名称核心动作
第1周自动开拓期多变体分组自动+2个流量大词广泛
第2周百花齐放期筛好变体+接力赛+出单词再投放+ASIN定投
第3周相同投放期相同词根归类+ASIN迭代
第4周架构形成期SB广告融入+精准测试+架构雏形

14.3 稳定出单期（第5-7周）

  • 15单是跑秒杀的门槛
  • 第六周BD+非主卖变体放站外
  • BD后"三不动"原则

14.4 最终广告架构

主要出单：自动 > 广泛/词组 > 出单词再投放
辅助出单：SBV > 品牌+关键词 > 竞品品牌
捡漏（长期配置）：0.02/0.2竞价的各组

14.5 Tcos公式

🔑 Tcos = ACos ÷ (1 + 自然单/广告单)

🏆 终极系统性总结

全部课程的核心知识体系

一、流量获取体系

1. 搜索流量（70%购买来源）
  • 广告位布局：SP/SB/SD三管齐下
  • 自然排名：点击转化率>出单量>类目排名
  • 前三页是最低要求，前20名是类目布局门槛
2. 关联流量（第二大块）
  • 自然关联（长期积累）+ 付费关联（ASIN定投）
  • 虚拟捆绑 + 商品页面防御
3. 类目/促销/榜单流量
  • 类目导航流量
  • 促销流量（Coupon/LD/BD）
  • 榜单流量（BSR/New Release/Wish For/Gift Idea）

二、广告架构体系

1. 按产品分类：
  • 标品：精准→核心词→长尾广泛→ASIN定投
  • 非标品：广泛→词组→自动→多词覆盖

2. 按目的分类（七大目的）：
  • 引流/推排名/拓词/关联/盈利/扩大/防护

3. 按广告类型：
  • SP：推排名唯一有效
  • SB：辅助+品牌+CPC低30-70%
  • SD：流失打捞+站外曝光

三、推品SOP体系

1. 出单词集中型（标品/半标品）：
  • 12周SOP → 精准打核心词 → BD秒杀拉升

2. 出单词分散型（非标品）：
  • 12周SOP → 广泛覆盖多词 → 多变体贡献

四、站外推广体系

  1. 站外是补充手段，站内基础必须做好
  2. 20天周期趋势打造
  3. Social Media Code + 阶梯折扣

五、站点差异化

  1. 美国站：大市场、高CPC、高客单用DSP
  2. 欧洲站：小市场、多语种、重精准词

🔑 关键要点回顾（TOP 20）

  1. 七大广告目的是每个广告活动的灵魂——没有目的的广告=烧钱
  2. 标品打精准、非标打广泛——这是广告架构的根本分叉点
  3. 预算递增定律——从低到高，让系统认为产品受欢迎
  4. 高低ACOS临界点 = 毛利润率 + 15%
  5. 自动广告永远不要停——能跑出特殊位置
  6. 类目垄断至少16个链接——不同位置不同目的
  7. 前20名是类目布局的门槛——进不到前三页=吃不到流量
  8. 秒杀后三不动——不动链接、不动广告、不动优惠券3-5天
  9. 接力赛广告——解决加预算变差的核心方法
  10. 玩具三反原则——反坑位、反精准、反降价
  11. 搜索量加权词频——不是简单按词频排序
  12. 旗舰店转化率10%+——高客单产品的流量承接核心
  13. 站外是催化剂不是发动机——站内基础必须做好
  14. 关站外后等4小时——防止叠加零元购
  15. Tcos 8%-15%是健康线——成熟期的终极考核指标
  16. 欧洲站≠美国站缩小版——Coupon金额vs比例、关键词分散度
  17. DSP按CPM收费，高点击率=低成本——超高客单的核心武器
  18. 非标品一定能挣钱——多链接布局、不追求爆款
  19. 数大麦筛词法——首页3000评论以上<20个=好机会
  20. 广告预算渐进式增加——每天5-10美金连加7天
