1️⃣ 最新亚马逊流量算法解析（课15）

一、A9算法核心要素

排名逻辑系统的五大要素：相关性（产品与关键词匹配）→ 转化率（订单CVR+复购）→ 销量（历史+即时）→ CPC广告排位（推广评分×出价）→ 促销（专属入口）

🔄 算法更新点
①个性化浏览（不同客户见不同结果）②PPC地位提升（广告位增多）③地理位置影响排名（消费等级+仓库距离）④站外流量受重视（ER编辑推荐）
🔍 搜索流量体系
自然搜索（自然位）+ 付费搜索（品牌广告+SP）
AC标4种获取方式：价格段/点击率/出单量/品牌维度
AC标策略：打长尾词更容易获得
📊 排名底层逻辑
曝光位轮流机制：1-3名占最好位置，4-20名六曝光位轮流，21-70名七曝光位（第二页），71-400名三曝光位（新品机会）
坑产概念：给亚马逊带来的佣金决定排名持续
💡 关联流量体系：自然关联（看了又看·相似产品·捆绑销售·FBT·Amazon Post·升级款关联·相关视频铺满）+ 付费关联（SP ASIN定位·四星推荐·品牌广告）。视频铺满5个防流量外流，Amazon Post免费关联必做。品牌广告识别：品牌logo+旗舰店入口；ASIN定位广告无logo无入口。
⚠️ 评论权重：留评率3-5%。差评权重加权——高权重买家账号差评影响大（1星=50%权重）。超3月差评影响降低。对策：锁评（5-30天订单）+ 三星卡位法压制一星差评。

六、关键词排名影响因素

阶段因素排序
新品期转化率 > 点击率 > 即时销量 > 历史销量
老品稳定期（>30单/天）单量稳定=排名稳定，关注ACoS即可
稳定期出问题先查点击率（CTR），再查转化率

关键词广告是唯一直接推关键词排名的广告类型。新品期聚焦关键词广告，稳定期再拓展其他广告类型。

2️⃣ 上万关键词查找分类实操演示（课16）

一、四象限分类矩阵

分类流量相关性推广优先级
核心大词（一级词）大强🥇 最高
精准长尾词小强🥈 次高
泛流量词大弱🥉 看情况
无效词小弱❌ 否定

流量判定标准：使用ABA搜索频率排名（非第三方搜索量）——<1万为流量大词，1-10万为中等，>10万为小词。
相关性判定：前台搜索该词→前10自然位70%以上（至少50%）为同类产品=相关性强。

二、五种查找方法

  1. 前台下拉框——免费但数量有限
  2. 竞品Listing——直接获取但人工慢
  3. 关键词工具（Sorftime/卖家精灵/H10）——数据全面但付费
  4. CPC广告报告——真实数据但需广告运行
  5. 品牌分析ABA——官方准确但数量少

三、实操步骤（以花园软管为例）

1确定核心词+类目 → 2BSR榜5+新品榜3竞品 → 3反查流量词 → 4汇总分析（5%+流量占比） → 5前台逐词验证相关性 → 6整理词表Excel

推品初期：先拿20-30个核心词做推广→最终扩展到上千个词。
词频统计：从竞品文案提取单词/双词/三词频率，指导标题写作和埋词。

二、标题三种写法

写法格式适用场景
标准写法品牌名 + 3核心词 + 功能属性 + 场景 + 参数垂直细分多款产品
核心词前置⭐最核心词 + 逗号/短横杠 + 品牌名 + 关键词...推排名·最佳实战
品牌优先品牌名 + 核心词 + 功能属性 + 场景 + 参数品牌有知名度

标题≤200字符，前90字符最重要（手机端截断）。权威链接：短横杠前的词更容易被谷歌收录。

三、图片7张图体系

图片内容
主图产品正面展示·纯白背景·"无中生有"手法增吸引力
第2-4张核心卖点（2-3个）
第5张结构/细节放大图
第6张使用说明图
第7张尺寸图+产品对比图
额外场景图 + 包装图（配件清单）

图片尺寸：1600×1600（电脑端）/1200×1500（手机端），大小200-300KB为佳。

四、视频优化

  • 主图视频：核心卖点展示+使用场景+网红视频（老外出镜）
  • 相关视频必须铺满5个——防止竞品视频占据→流量外流
  • 视频来源：买家秀 + 卖家上传 + Review自动视频

五、五点描述与Search Term

五点：总分结构（中括号总结+展开）+ 埋5-10个词 + 通俗易懂（不写技术术语）+ 数字化展示。
Search Term：写满为止·不重复标题五点已出现词·精准词优先。

六、品牌故事+A+页面

品牌故事：4个ASIN卡片做关联流量→免费流量入口。
A+页面：最多7个模块，标准对比图是关键——放3-5个相关产品做关联引流。A+图片命名埋核心关键词。

七、三大参数区域

  1. 标题与五点之间参数——必须填写！缺失可能导致排名无法显示。
  2. 搜索结果左侧栏筛选参数——不填写=被客户筛掉。
  3. Product Information——通过库存加载文件补充后台缺失项。
🔑 核心原则：竞品有的参数你也要有→列表单对比→逐一补齐。参数不完整=链接权重降低=排名受影响。

4️⃣ 高利润率推品方法介绍（课4）

一、四大前提条件

前提核心要点
① 选品新切入中低价区间·功能改良＞外观改良·不做同质化
② 定价有功能改良可卖高于主流·外观差异定中上等·不倒推利润
③ 广告运营成本≤利润额2倍可做·＞3倍不建议·新品看CVR非ACoS
④ 自然订单广告拉动自然排名·广告订单占比逐渐下降·销量曲线稳步上升
✅ 实战数据：案例产品1月内达日销100单，推广全程利润率25%。ROI 34%。第4-5天起每天赚30-50美金，第13天起每天赚100+美金。

二、广告架构核心

一拖二模式：1自动广告(紧密匹配) + 2手动广告(单词单组单活动·精准控制)
预算：自动15-20美金/天·手动建议竞价起步·每天增5-10美金
大词定义：ABA前1万名或前2万名的词（非排名第一的词）

三、标品vs非标品广告

维度标品非标品
自动匹配紧密匹配紧密+同类（必须开）
手动匹配精准匹配为主词组/广泛为主（不打精准）

5️⃣ 科学合理制定新品推广预算计划（课8）

一、推广费用四大构成

  1. 广告花费（占比最高）——核心计算公式
  2. Review获取——Vine计划200美金 + 产品成本 + Top Review 80美金/个
  3. QA费用——约5美金/个
  4. 补单费用——产品+站外放单费（弹性）

二、核心计算公式

广告花费 = 预期广告单量 ÷ 广告转化率 × 单次点击成本（CPC）

先制定单量递增计划（第1周5单→第2周10单→第3周20单→第4周30单），广告占比约50%。

📝 案例：书灯产品，目标月底日销30单（广告15单）。转化率11%（商品探测器），CPC 2.1美金（建议竞价）。预估月广告费≈4343美金。运行一周后修正为2047美金（实际CVR 20-30%，CPC 1.8）。

三、广告位优化

  • 首行加成30%让广告更多在首页产生点击
  • 每次调整幅度≤0.1-0.2美金·记录调整日志
  • 链接有权重后CPC自然下降（半个月从1.8→1.6）
  • 观察7天数据再判断·不要天天动广告

6️⃣ 预算高效利用的底层逻辑（课9）

一、广告三大目的与生命周期

时期核心目的"效益最好"定义
新品期推排名（让亚马逊收录关键词）能推动自然流量的投放·不看ACoS
成熟期要利润ACoS最低的投放
抢排名期提类目排名能推排名波动的投放

★ 判断标准看出单量，不是看上架时间！

二、广告推排名的四步机制

1相关性验证 → 2点击率（CTR） → 3转化率（CVR） → 4关键词出单量 → 排名提升

三、四个"不要纠结"

  1. 不要纠结于某一个关键词——找更多低ACoS词拉低整体
  2. 不要纠结于某一个竞价——CPC有临界值，不能硬降
  3. 不要纠结某个投放的ACoS——合理ACoS因品而异
  4. 不要纠结某个匹配方式——精准/广泛/词组可同时存在

四、测试与保护（流量转移）

自动广告跑词→ 好词拿出来打手动精准→ 保护（给足预算重点培养）

二、ACoS三维度分析

ACoS = CPC ÷（客单价 × 转化率）
维度定义应用
实际ACoS总花费÷总销售额整体表现
合理ACoS⭐建议CPC÷（客单价×出单转化率）排除无效点击后的真实竞争力
类目平均ACoS建议CPC÷（类目均价×类目CVR）市场基准

TACoS（广告占比）：广告花费÷总销售额。20%健康·30%警戒·超30%吞噬利润。

三、四类问题花费

类型特征处理
流血花费点击不出单果断关闭
低点击花费预算花不完提高竞价/收缩预算
高ACoS高于目标降竞价10-20%
低ACoS低于目标加预算扩量

9️⃣ 单量火箭式上升的SOP详解（课6）

一、定价与促销组合

  1. 中高定价+降价红标：第一周中高价出20单+→第二周降价5%+→触发红色降价标
  2. 优惠券：≥5%·专属流量入口·兑换率约50%·长期监控预算（80%及时加）
  3. 专享折扣：免费·大促有秒杀标识·比优惠券略优（家居户外类目）
  4. 购买折扣/额外购买：可叠加使用·多促销标识增转化率

二、限时秒杀LD vs 七天秒杀BD

维度LD（12小时）BD（7天）⭐核心
适用少量库存·冲瞬时排名冲销量+关键词排名+类目排名
广告不加预算翻倍！平时2倍
进度条90%加库存·100%半小时内结束第1天必须爆单·第4天降预算
优惠券关闭站外促销·开大额兜底大额优惠券兜底·必要时站外配合
频率7天1次·月4次28天1次
🔥 BD关键操作：第1天必须爆单！周末降预算是核心技巧——周六周日欧美购买欲高于平时，每天降20-30%预算，到周日降到正常水平。错过周末→秒杀结束单量腰斩。

🔟 间断流打造爆款的方法介绍（课3）

一、三种投放模式

模式特点适用
一波流投一次→排名升→掉回清库存
间断流⭐排名掉了就投→逐步稳定推新品·推荐
持续流天天投·需大量资源有资源卖家

二、站外推新品核心原则

🚫 新品做站外不能开广告！站外流量杂+站内流量干净→汇合后亚马逊无法判断→流量混乱。等自然排名到前三页后再开广告。

三、间断流操作节奏

纯折扣50-70% → 排名升 → 排名要掉 → 再投 → 逐步稳定 → 关键词前三页 → 折扣+Coupon → 开广告

每月投放10-15次·隔两天投一次·控制月站外单量≤1000单·排名底线不超300名。

四、Facebook站外渠道

折扣小组最常见（便宜·100-300元/贴·1-2天效果）。七因素影响出单：市场需求＞Listing质量＞折扣力度/折后价＞投放精准度＞竞品频率＞季节性。

💰 价格策略案例：定价35.99美金→60%折扣→折后14.4美金→每单亏1.27美金可承受。第二个月降价29.99拿降价红标→折扣+Coupon→第三个月开广告（CPC仅0.5美金因为自然排名已到前三页）→参加秒杀→六个月打造爆款。

📋 第三部分：标准SOP层

将前两部分整合为12周标准化执行时间线。标品和非标品SOP分别针对两类产品特性，逐周指南。

1️⃣1️⃣ 标品12周推品SOP详解（课18）

📦 准备阶段
查词（100+）→ 整词表 → 写Listing（12维度）→ 找对标品（新品榜前3+BSR榜）→ 定三节点目标（50名/30名/5名）→ QA 3天5对 → 测评为期3天
📌 第1周：启动期
定价：市场中高价。第1-3天优惠券15%；第4-6天加大到40-50%。
广告：自动（紧密·固定竞价·15次点击费用）+ 手动（2核心词·单词单组·建议竞价1.2-1.5倍·首行加成30-50%）。3天看一次数据。
📌 第2周：上首页
降价：中高价→中等价（10-20%幅度）→ 触发降价红标！优惠券保持40-50%。
广告：手动每天加4-5次点击预算×连加7天→停。目标：两个核心词上首页！总预算约150-160美金/天。
📌 第3-4周：拓词+控ACoS
优惠券逐步下调：每3天降一次·分两周完成·最低保持10%。
广告：新增4-10名精准·大词广泛·50长尾词广泛·预算优化。
ACoS控制：过去7天ACoS>合理1-2倍→竞价降10%；>2倍→降20%。
第4周安排第6周七天秒杀。
📌 第5-6周：精细化+秒杀
第5周：ASIN定位广告开启·长尾词持续筛选·预算优化（高目标降10%/低目标涨40%）。
第6周：七天秒杀！第一天必须爆单（30-40+单）·预算翻倍·第4天开始降预算·周末降至正常。
📌 第7-8周：多类型拓展
第7周：秒杀后开20-30%优惠券缓冲·预算转移优化。
第8周：限时秒杀·视频广告SBV·SD展示型广告·SB品牌广告（需3同小类产品）→进入常态化运营。
📌 第9-12周：常态化
循环往复：ACoS控制（竞价调10-20%）+ 预算优化（高降10%低涨10%）+ 拓展流量（每周搜索词报告·只加新词·预算10-15）

标品vs非标品快速判断

维度标品非标品
核心大词集中+精准分散+不精准
搜索结果外观相似（如书灯）款式多样（如裙子）
匹配方式精准为主词组/广泛为主
筛词标准周出5单+周出2单即可

1️⃣2️⃣ 非标品12周推品SOP详解（课19）

核心差异：流量分散·关键词多·款式多样。词组匹配比精准匹配更重要。

📌 第1周
广告：自动（紧密·建议竞价0.8倍）+ 手动（2词根广泛·10-20美金/组）+ 精准长尾词（产品了解时）。
否定策略前置：刚上来就否定不相关属性词（如卖女士包→否定"men"）。
📌 第2周
降价（红标）+ 筛词：下载搜索词报告→找出单≥2单的词→开词组匹配（预算10·CPC×1.2）。
ASIN定位：10个竞品ASIN（流量大+价比我高+评分比我低）。
📌 第3-5周
优惠券下调 + ACoS控制（下调10-20%）+ 预算优化（高降10%低涨10%）
第5周：出单词精准化——下载前四周报告→找出单最多的词→开精准匹配（总预算15美金·5个组）。
📌 第6-8周
第6周：七天秒杀！第一天爆单·优惠券加持·必要时站外。
第7周：秒杀后优惠券20-30%缓冲·广告一周内不乱调·手动精准出单词。
第8周：限时秒杀·SBV视频广告·品牌广告（词前加+号）。
📌 第9-12周
常态化运营：持续拓词（大词广泛→自动→拉出单词→词组测试→循环）·非标品不能停！

🏷️ 第四部分：类目打法层

通用方法论适配具体类目场景。每个类目都有独特性，需要针对性调整。

1️⃣3️⃣ 玩具类目产品爆单打法（课1）

🎯 类目特征
消费群体（父母）和受众群体（儿童）分离。
旺季销量是淡季10倍+。
大词流量为主（toys/gifts），精准词少。
📅 推广时间节点
最佳推广：8月中旬-9月中旬。
10月底开始爆发 → 12月顶峰。
太晚来不及推·太早成本高。
🧩 CPC认证
发货前准备好（4000-5000元）。
3岁以下要求严格。
年龄段用空格不用斜杠（"for 2 3 4 5"）。
纽扣电池必须螺丝固定。

广告策略：三周递进式

第1周手动自身精准词 → 第2周长尾词+核心词（小词带大词） → 第3周核心词+流量大词（广泛）

小词带大词：长尾词包含核心词（如"LCD writing tablet for kids 2-4"含"writing tablet"）→长尾词上首页→带动核心词自然排名→再去打大词成本低转化好。

甬道图优化框架

W1 自动广告跑词·测试相关性 → W2 手动广告一筛选60分以上（精准+词组+广泛三组） → W3 手动广告二"重点班"给足预算·要多少给多少

筛选标准：曝光>1000·CTR>7‰·CVR>10%。否定策略只做精准否定，不做词组否定。

1️⃣4️⃣ 季节性产品推品详解和避坑指南（课7）

选品三原则

≥50%利润率
≥3月销售周期
🎯流量词集中精准

款式选择与备货

类型备货策略
经典款（常规款）按主流备·量可足·参考往年数据
流行款（新款）以测为主·少备300-500套·确保50%清仓不亏

清库存策略

时机：销售周期过75%时开始清。
方法：50%社交媒体折扣 → 秒杀 → 站外 → 购买折扣 → 奥特莱斯。
底线：50%折扣清仓不亏损。

1️⃣5️⃣ 工具类目产品打法秘籍（课12）

核心逻辑：销量-排名-流量三角循环。六大流量入口协同作用。

四阶段广告布局

阶段操作
① 启动期自动广告跑词→跑出词根→手动精准（单词单组）
② 优化期否定不相关词（词组否定）·拓词（搜索词报告出单词开精准）
③ 扩展期SD展示型·SB品牌广告·视频广告·多流量入口
④ 收割期精细化管理·按ACoS分配预算·抢AC标签

1️⃣6️⃣ 服装类目的广告架构搭建演示（课13）

特殊性：SKU多·退货率高（15-30%）·尺码问题多。

退货成本定价法

定价 = （成本 + 物流 + 广告预算）÷（1 - 退货率）÷（1 - 利润率）

变体排序：按照销量降序排列变体（影响权重分配）。
品牌尺码表必须设置！

生命周期广告策略

阶段策略
新品期自动广告跑词→转化好的词转手动·控制SKU数（非标品50-100内）
成长期阶梯式增量法·多广告类型·季节性关键词提前布局
稳定期品牌词打法·主图换头技巧·反季延伸

1️⃣7️⃣ 低客单高客单推品方法介绍（课2/5）

一、低价螺旋法（适合15美金以下产品）

定义：低价→获取排名→价格上涨→获取利润。从8.99涨到13.99的主流价格。

四大适用条件：①价格敏感型产品（<15美金）②非垄断市场③标杆链接价格>成本2倍④有自然订单推动。

操作步骤：上架→半价冲量→BSR前100→涨价（每天涨0.5-1美金）→稳步上涨直到主流价格。

⚠️ 注意：低价螺旋不适合客单价高（≥50美金）的产品·做低客单+低价螺旋建议发海运降低成本·需持续关注竞品价格·不要频繁调整价格。

二、网红推荐法（适合高客单价产品）

站外通过网红/红人推广爆单的策略。核心流程：

YouTube测评→ Instagram曝光→ TikTok视频→ Facebook群组→ Deal站→ 独立站导流

六站外渠道布局：YouTube·Instagram·TikTok·Facebook群组·Deal站·独立站。

1️⃣8️⃣ 快速抢占Bestseller的方法介绍（课18）

前置条件

  • 评价基础：至少几十条评论（Vine+Top Review）
  • 高定价策略：定到类目高价区间，为站外折扣留空间（50-70%折扣后仍保本）
  • 广告基础：自动广告+核心大词手动精准（单词单组）持续测试

五重流量叠加战术

  1. 站外连续放量：5-7天连续投放Facebook·每天200-300单
  2. 七天秒杀BD：第1天爆单+广告翻倍
  3. 自动广告扩大曝光
  4. 手动广告推关键词排名
  5. 站外+站内叠加效应
🇬🇧 英国站优势：竞争较温和·CPC相对低·英国人购物习惯集中（Amazon是主要在线购物平台）·同时开通欧洲多国。

🎯 十大核心认知

  1. 排名=坑产竞争：给亚马逊带来的佣金决定排名持续
  2. 关键词广告是唯一推排名广告类型：新品期聚焦关键词广告
  3. 自然订单是盈利根本：广告订单占比需持续下降
  4. 运营成本必须提前核算：选品时就要算清，≤利润2倍可做
  5. 先规划后执行：预期单量→倒推广告预算→执行→修正
  6. 测试与保护：广告本质是"把钱花在效益最好的投放上"
  7. 数据驱动：3天看波动·7天看趋势·30天看整体
  8. 流量保护意识：视频铺满·品牌故事关联·A+对比图→锁住流量
  9. 分类施策：标品精准/非标品词组·类目打法各有差异
  10. 持续学习：一周一优化·一月一复盘·持续拓词不能停
