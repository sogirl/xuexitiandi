❓常见问题 FAQ

我是完全的小白,应该从哪里开始?
从第1课开始,按顺序学习。第1-5课建立"选品认知",第6-12课掌握"差异化方法",第13-19课是"实操演练",第20-23课是"供应链落地"。前两周集中看完理论,第三周开始对着第14-23课的真实案例边学边练,坚持一个月即可上手。
我已经做了一两年亚马逊,可以直接跳到哪里?
重点看 第6-10课(差异化方法论体系) 和 第11课(供应商),再精读 第13课(合规)。最有价值的往往是差异化和迭代方向、元素添加的精细化方法。
我是工厂型卖家,这门课对我有用吗?
非常有用。工厂型卖家的优势是开模成本低、备货灵活。课程里多次提到工厂型卖家适合做高客单价、差异化深的产品(见第5课、第11课、第23课)。重点看第11课(供应商和开模)和第23课(议价),可以反哺您对自己客户的报价策略。
如何判断一个产品值不值得做?
用本手册的三层过滤漏斗:
1️⃣ 市场层:月销≥500、品牌/卖家集中度≤70%、非季节性、非红海
2️⃣ 财务层:客单价≥$10、毛利率≥30%、投产比≥1
3️⃣ 差异化层:有真实痛点可解、有差异化方向、有30%溢价空间
三层都通过 → 进入测款阶段;任一层不达标 → pass。
为什么"差异化和迭代"是这套课程的核心?
2021年以来,亚马逊进入产能过剩+消费者主导阶段,同质化竞争激烈,价格战导致"有量没利润"。差异化是规避红海、避免低价螺旋、设立门槛的唯一路径。而迭代是让差异化持续保持优势的方法。课程第6-10课共5节课都在讲差异化,第8课专门讲迭代方向,可见其重要性。
30%溢价红线是什么?
30%溢价红线是本课程最重要的方法论之一(见第8课)。意思是:差异化带来的成本上升,市场最多能接受售价上涨30%。超过30% → 客户不接受 → 产品卖不动。案例:迷你封口机叠加多功能,采购成本30多元,但客户认知就值$9-13,卖$26-29没人买,结果赔了20万。核心教训:差异化成本必须控制,溢价红线是30%。
课程里的"产业带"是什么?
产业带 = 某类产品工厂聚集的地理区域。例如:许昌假发(90%全球假发)、阳江刀具/不锈钢、白沟箱包、洛阳办公家具、武清家纺、义乌小商品。为什么要找产业带? 一手货源、价格更低、规模真实、谈判筹码大。第11课有完整的产业带分布速查表。
投产比为什么是生命线?
投产比 = 产出 / 投入 = 利润 / (成本+头程)。投产比≥1 → 投资有正回报;投产比<1 → 卖一单亏一单,持续2-3个月小卖家就撑不住。结论:在销量和利润的取舍中,一定选利润(第4课核心)。因为利润率可以暂时低,但投产比<1是直接亏现金流,这是卖家的生命线。

🔍 市场竞争分析8大维度

维度1:市场容量与销量占比

  • 销量结构观察:第1名月销20,000+单、第22名仍能达到月销900单
  • 关键判断标准:目标月销500-1000单,根据自身资金情况确定
  • L型断崖式下降 → 直接放弃(出自第24段)
  • 第1名销量占比 >40% + 后续无销量 = 放弃

维度2:市场需求趋势(季节性)

  • 季节性产品:销售期<半年不建议
  • 季节性产品必须在高峰来临前2-3个月上架备货
  • 非季节性:销量曲线缓慢上升 = 市场增长期;缓慢下降 = 警惕市场衰落

维度3:品牌与卖家垄断

  • 品牌大卖垄断系数 > 60% → 高风险,建议排除该类目
  • 卖家垄断:看前30、前50
  • 欧乐B案例:59个链接被欧乐B占据 = 直接放弃

维度4:亚马逊自营垄断

  • 亚马逊自营自带流量优势
  • 亚马逊自营垄断 > 60%:可直接放弃整个类目

维度5:新品数量与上架时长

  • 真正的新品特征:新推广链接、无合并链接、非头部卖家重新上链接、上架时间短
  • 新品数量多 + 都是正常操作 → 新品有机会
  • 看非大卖新品在Top100中的表现

维度6:评论数量

  • 1000+评论销量占比 >70-80% → 老品占据主流 = 新品进入难
  • 1000+评论占比 <50% → 新品有机会

维度7:价格分析与饱和度

  • 饱和状态 = 选哪个价格段都竞争激烈
  • 非饱和 = 仍有差异化空间可切入
  • 核心结论:选品不是判断"能不能进",而是判断"有没有优势"(出自第62段)

维度8:评分与差评分析

  • 评分4.0以下产品有几十个 → 有较大改进空间
  • 整体都在4.3以上 → 差异化更困难(标品多见)
  • 差评分析输出:统计差评 → 发现痛点/空白点 → 找到可改进点 → 差异化方向确定

4 第4课 · 销量、利润和投产的取舍
📍 资金效率的生命线

💡 三大核心概念

销量的定义

  • 每天的出单量
  • 月销量 = 单量 × 30天

利润的定义

利润 = 销售额 - 成本 - 运费 - 优惠券/折扣 - 佣金 - 配送费

投产比的定义

投产比 = 产出 / 投入
即:投产比 = 利润 / (成本 + 头程)

关键概念区分

  • 成本:直接掏出去的钱(显性支出)
  • 投诚:产品运到亚马逊过程中产生的费用(直接支付)
  • 配送费 + 佣金:非实质性支出,每卖一单才扣除

⚖️ A销量型 vs B利润型方案对比

项目A(要销量)B(要利润)
月销量3,000单1,000单
利润率20%30%
售价$19.99$25.99
采购成本$3$5
头程$2-
配送费$4-
实际投入$27,000$15,000

关键发现:A方案实际投入 = $27,000;B方案实际投入 = $15,000;差距约2倍——但A单量仅是B的3倍,投入却是2倍。

🚨 投产比的核心地位

🚨 资金效率、生命线的核心指标 在销量和利润的取舍中,一定是选利润。因为投产比是底层的资金安全指标(出自第40-42段)。

投产比<1的严重后果

  • 投产比 < 1 = 投1元赔钱
  • 投产 < 1 = 卖一单亏一单
  • 持续2-3个月小卖家难以支撑

相同利润率下的投产比差异

  • 利润率30%的情况下:产品1投产比1.0 > 产品2投产比0.87
  • 投产比越大 = 投资回报率越高
  • 投产比 > 1 = 投资有正回报

📈 产品全生命周期策略

阶段核心目标关注指标关键动作
前期推广期推排名销量加大广告预算,要销量不要利润
中期稳定期稳定排名+获取利润销量+利润+投产排名上升继续推,稳定后减少投入
后期成熟期利润最大化利润+投产减少广告投入,靠自然流量

📋 重要阈值标准

指标阈值含义
投产比≥1资金效率底线
投产比<1卖一单亏一单
投产比负数绝对不可
利润率(低客单)20-30%基本达标
利润率(高客单)35-40%+标准要求
销量底线20-30单/天仓储费覆盖

✅ 核心结论 在销量和利润的取舍中,一定选利润。利润率可以暂时低,但投产比<1是直接亏现金流,这是卖家的生命线。

5 第5课 · 高低客单价选品方向
📍 客单价分层决定选品策略

💵 客单价分层标准

客单价区间定义转化率
高客单价>80美金3%-5%
中高客单价50-80美金-

💰 低客单产品的选品与运营

选品技巧

  1. 尺寸与重量:不适合体积大、重量大、客单价又低的产品;当前标准2磅以内(小商品计划3磅、12美金)
  2. 品类熟悉度:适合初来乍到的新卖家
  3. 定价策略:市场最低价仍能保本(案例:7.99为平衡点)
  4. 佣金优势品类:美妆、个护、母婴,佣金仅8%(vs普通品类15%),节省7% = 直接转化为利润

运营特点

  • 转化率高但Acos难看(Acos 59.30%但低客单正常现象)
  • 必须靠自然单挣钱
  • 起量路径:起步30-50单 → 中期70-90单 → 成熟期110-150单
  • 多变体布局必备:1件套引流款、2件套中等、10件套/20件套利润款
  • 市场容量要求:30单对应的排名不能太高
  • 避免低价位差竞争

差异化方案

  • 功能性升级(案例:橱柜灯加氛围灯+遥控+多颜色)
  • 简单的元素添加(卡通图案、节日元素)
  • 差异化原则:标品能加功能就加功能,但功能成本不能太高
  • 不要颠覆性改革
  • 来图定制(墙纸、相框)
  • 节日元素、爆款元素、主题元素、宠物元素、元素移植
  • 不要违背产品的根本用途(反例:茶杯做成骷髅头形状)

案例:低客单的运营打造方法

  • 不需要做种子链接
  • 定价足够低 + 市场最低价
  • 上架后第一周:优惠券 + 降价 + 大优惠券
  • 半个月-20天推大词到首页
  • 自动广告 + 两个手动
  • 广告预算:10美金起步,每次增加5美金
  • 推品节奏:先大词→二词→三四词→五六词
  • 循环赛打法

💎 高客单产品的选品与运营

两种高客单的分类

  1. 绝对高客单:金额上直接高(80美金 vs 10美金)
  2. 相对高客单:在品类内相对高(案例:78美金vs15美金,卖20美金就是相对高)

选品特点

  • 高门槛筛选:采购成本高、备货量大、资金要求高
  • 适合:工厂型卖家、资金实力强的、关系型卖家
  • 转化率特点:类目均值1%偏低、3%不错、0.x%很难做
  • 广告盈利可能性:Acos 18.43%可盈利
  • CPC优势:办公家具类CPC 0.5-1+美金,客单两三百美金

运营特点

  • 链接质量决定一切:主图视频、A+页面、QA视频、相关视频
  • 垂直细分定位:直接做垂直细分
  • 广告关联销售:用户看A也会看B
  • 运营节奏:单量10-30单/天即可
  • 自然排名 + 广告并行
  • 数据查看节奏:不要天天看,看一周数据
  • 适合自己的品类:不要跨品类

案例:办公家具活动柜(完整差异化路径图)

  • 关系型卖家(账期支持)

🎯 消费者购买决策的两大关键时刻

第一时刻——吸引点开/拿起的因素

  • 外观新颖独特、款式别致
  • 价格与折扣的吸引力
  • 标题能否解决需求
  • 图片视觉效果

  • 评论数量与质量

💡 差异化的四大动因

💸
动因一:避免价格战
2021年以来低价价格战激烈,涌入门槛低。差异化是体现不一样价值的唯一路径。
🌊
动因二:避开红海市场
案例:血海产品CPC $4.07,转化率43.5%——广告没问题但竞争太激烈。
🌀
动因三:避免低价螺旋陷阱
案例:四五十万投入呼啦圈,大家涌进价格拉低。头部卖家能扛半年,新卖家硬扛半年灰溜溜走。
🛡️
动因四:人为设立门槛
通过功能升级迭代、外观改变、受众细分,人为设立难度——"你抄我不是那么容易抄"。

8 第8课 · 产品迭代方向判定
📍 差异化的"方向判定"

🧠 底层思维:产品 = 用户 + 问题 + 解决方案

直接用户 vs 间接用户

  • 直接用户:购买者 = 使用者(自己买衣服自己穿)
  • 间接用户:购买者 ≠ 使用者(儿童产品、宠物用品)
  • 学习机案例:家长=购买者(学习需求),小孩=使用者(游戏需求)——必须双兼顾
  • 宠物服饰案例:宠物本身不需要衣服,但主人需要"漂亮"的情绪价值

市场受众大小的判断方法

  1. 直接去前台搜索,看有没有同类产品在卖
  2. 表现还不错的至少有十几个,说明需求真实存在
  3. 如果是"从来没有人做过的产品"——大概率没有市场
  4. 投入大量迭代成本前,必须先确认市场体量

⚖️ 标品 vs 非标品的双轨策略

标品的核心策略:优先功能迭代

  • 第一优先:功能迭代
  • 第二优先:外观款式、颜色变化
  • 伪需求警示:标品市场已成熟,再硬加功能会出现伪需求

标品功能迭代的判断标准

  • 这个功能成本是多少?
  • 加上后售价涨多少?市场能不能接受?
  • 30% 左右的溢价市场可以接受,过高则不接受(出自第72段)

标品经典案例

🍾
开瓶器案例
不要硬加多功能工具(带螺丝刀、刀、灯),可改为企鹅等可爱形象
🪝
挂钩案例
元素3D移植做造型差异
🥤
杯子案例
竹子外形 = 元素移植
🧈
⭐ 欧美黄油碟经典案例
痛点1:盖子放桌上粘到桌子 → 加四个角凸起
痛点2:抹黄油没有斜坡可挂 → 底部做斜坡

非标品的核心策略:款式差异化

非标品(时尚类、礼品、节日装饰、玩具)体现的是情绪价值。

🔄 元素移植与外观造型的差异化

元素移植的三种类型

  1. 文字符号(best something等文字 + 套装包装 = 礼品属性)
  2. 图案元素(波西米亚风地毯、3D抱枕)
  3. 元素3D化(把平面元素立体化,真空包装可解决体积大问题)

3D抱枕的"门槛思维"

  • 普通长方形抱枕谁都能做 → 竞争激烈
  • 3D多肉造型 → 抓住"小波人群"

布料材质的镂空差异化

  • 桌布 = 实用,桌旗 = 装饰
  • 镂空 + 大花图案 = 节日(超级碗等)装饰
  • 布料差异化难度小:不用开模,只需提供设计稿

迭代外观的成本与溢价红线

⚠️ 红线数据 改造成本预估:两三万可以接受,20-30万不考虑。
改完要跟市场比:市场上卖$10,改完$13-14(约30%溢价)可接受。

🔗 链接维度的差异化

主图差异化

  • 角度/风格不同:别人都展示正面,我们展示45度或场景化
  • "无中生有":卖花盆,里面放一盆花
  • 包装外露:精美的礼品盒 + 产品一起放主图
  • 建模优势:能建模就建模

SKU数量差异化

  • 市场大部分是10个装/20个装——我们做8个装/15个装
  • 套装混搭:1英寸2个 + 20英寸3个 = 差异化SKU

SKU颜色混搭

  • 市场清一色"5件装红/白/绿分色"
  • 我们做"6件套":2红+2白+2绿

价格阶梯引导

  • 2件装$11.89,4件装$16.99
  • 目的:引导客户买4件装,是定价策略不是低价

变体数量设置门槛

  • 时尚鞋服、配饰等变体可加到几十上百个
  • 核心目的不是销量,而是设置门槛——让竞品"算了不做了"

🚨 差异化常见误区

⚠️
误区一:质量导向的差异化
差异化不是"质量做到最好"。用铝合金 vs 银质做手机支架——提升材质档次是"伪差异化"。基本需求满足即可,不要在材质上硬提档次。
💸
误区二:功能堆砌(迷你封口机案例)
采购成本30多元,"多功能"产品。客户认知中产品就值$9-13。$26-29售价客户不买单。
🚨 踩坑代价:赔了20万(出自第176段)
核心教训:差异化成本高到市场无法接受 = 死路一条
30% 溢价是市场接受的红线
🔄
误区三:重心偏离(台灯多功能案例)
台灯 + 手机充电 + 钟表 + 音响 + 应急灯 + 笔筒。
客户买台灯的核心需求 = 照明。别人卖$30多,这款要$50,市场不接受。
核心错误:重心放在附加功能,忽略核心功能。
附加功能只能在前置功能(灯本身)做到极致后才考虑。

迷你封口机的转化率悖论

⚠️ 悖论 19% 转化率(远超大盘10%)但还是失败。
说明:转化率高不代表市场接受度高,关键是"市场体量 × 转化率 × 价格"的总销售额。

10 第10课 · 产品卖点与痛点预分析
📍 卖点 + 痛点 = 思维框架

🎯 为什么讲卖点

  • 产品能在市场上存活数年(如羽绒服卖5-7年)
  • 上架2-3个月就卖不下去的产品:缺乏卖点
  • 卖点 = 产品在市场上的存在理由

🏛️ 卖点的四个层面

📦
层面1:产品本身
功能性优势、价格优势、设计优势
🏭
层面2:工厂
工厂会主动升级产品,例:手机支架不会倾斜/往后倒/不坚固
👤
层面3:消费者
案例:羽绒服必须保暖/不掉毛;保温杯需要保温时间
🎯
层面4:目标受众
案例:嘻哈爱好者买的毯子:印花+文字+嘻哈风格

📐 卖点公式

✅ 完整定义 卖点 = 解决交易对方需求 + 满足目标受众 + 优于竞品

💬 卖点的展示方法

案例:美容院销售美白产品

  1. 发现痛点:寒暄顾客:发现气色不好、脸有点黄
  2. 提出产品功效:推荐美白产品,说明功效
  3. 激发购买欲望:详细说明产品效果
  4. 延伸推荐:主动询问:是不是没睡好?推荐助眠产品
💡 核心原则 痛点先行,引起共鸣 → 再给解决方案
亚马逊产品同样适用——listing文案要"先描述用户痛点场景,再展示产品如何解决"。

🎯 痛点的定义与分解

痛点的核心定义

超出客户对某个问题的忍受阈值,诉求得不到满足。

痛点分解的8步法

  1. 定义使用场景(产品是干什么的?适合在什么场景使用?)
  2. 诉求原因分析(多问"为什么")
  3. 5W分解(谁用/何时用/为什么用/在哪儿用/怎么用)
  4. 确定最大改变点(作为卖家,最大的改变点是什么?)
  5. 重要性分类(标品:功能>外观;非标品:外观>功能)
  6. 替代方案头脑风暴(找差异化方向)
  7. 替代方案的诉求满足度重新评估(差异化是否满足客户需求?)
  8. 替代方案的缺陷评估(差异化方案有什么缺点?)

🏷️ 痛点的四种类型

类型1:持续反馈型
用户主动反馈的普遍问题
类型2:可预见型
自己能提前发现的
类型3:认知型
用户固有认知被违背(收纳=装东西)
类型4:创造型
营销人员主动营造的

🏛️ 痛点的四个特点

  1. 普遍性(大部分用户普遍存在的问题)
  2. 先发现者抢占先机(上架时间早=抢占市场)
  3. 用户能发现但难解决(3C类带电产品)
  4. 营销人员可以主动提出(美容师引导)

🔄 痛点与卖点的关系

⚠️ 关键警示 卖家认为的痛点 ≠ 客户反馈的痛点
必须以客户反馈为准——这是差异化最容易踩的坑。

渠道用途重点

🎯 选品基调设定

核心要求

🪵 材质方向(关键约束)

可做材质(建议优先)

✅ 优先材质 编织类、金属类(铁质)、木制品、石头等
这些材质前期投入成本低,适合试款、上品、试品的要求
即使品做不好,损失也不会特别大

需谨慎/避免的材质

⚠️ 避免材质 塑料、橡胶:开模成本高、模具投入大
玻璃:运输过程不好搞,损坏率高
家具类部分材质:可能涉及认证要求

例外情况

如果对产品成功非常有信心(如已发现特别好切入点、拿到专利、有重大功能突破),塑料、玻璃也可以尝试。

  2. 没有周期性、季节性、节日属性
  3. 小类目前30名月销900单
  4. 利润达到30%以上
  5. 类目范围:厨房、家居、餐厅、客厅、工具花园
  6. 材质选择开模成本低、不需要开模的(如铁质、木质、编织类)
  7. 避免玻璃、塑料、橡胶(高开模/运输风险材质)

统计范围

样本量原则

  • 第100-105名
  • 第150-160名
  • 多统计一些,让数据更准确

📌 同款 "什么叫同款?功能是一样的,外观可能有点不一样,但根本不影响它的使用。"

实际操作

  • 从第1名开始统计,本次实操中从第15名统计到第90多名
  • 把Top 100里的同款款式都统计下来

🎯 关键结论

✅ 综合结论(浮动价值/隔板案例) 16英寸 + 木质(或木+铁组合) + 2个装或3个装 + 价格20-30美金 = 销量最好

1. 第一个结论:尺寸

  • 16英寸的销量是最高的
  • 销量最高的价格区间:20-25美金、20以下、25-30美金

2. 第二个结论:数量

  • 2个装和3个装销量最高

3. 第三个结论:材质

  • 木质和木质+铁的组合销量最高
  • 不必纠结具体木种(泡桐木、松木、复合板),整体是木质就行

🏭 1688找厂家流程

  1. 用结论去1688搜索:拿着"16英寸 + 木质/木+铁 + 2装/3装"的结论,去1688上找同款
  2. 联系厂家获取关键信息:包装后的重量和尺寸、配送费(通过尺寸和重量查到)
  3. 多家询价:前期询价要多问几家,不同厂家报价不一样(浙江生产该类产品较多;广东深圳也有但价格可能更高)
  4. 利润核算:整体算下来利润在30%左右(毛利润)
  5. 定价策略:刚上架的价格 ≠ 链接成熟后的价格。标记出热卖款的价格区间,找到自己能接受的最低价格
⚠️ 利润核算的意义 "我们最早选品时说的利润达到30%,后期还要减去广告成本,才可能会去有钱赚,所以这个就是我们整体在前面定的这个利润30%以上。"
核心逻辑:30%毛利 - 广告费 - 损耗 = 净利润
30%是底线,不能再低

💡 实操总结

  1. 多询价:至少问多家
  2. 多比较:浙江 vs 广东深圳价格差异
  3. 多维度对比:重量、尺寸、配送费
  4. 毛利率底线:30%

  2. 功能改良:在原有产品上增加新功能

1. 买家评论反馈

  • 通过买家反馈寻找痛点
  • 提炼"大家想要什么"和"大家觉得什么不好"
  • 进行改变

2. 竞品调研

  • 看其他卖家已经做的差异化
  • 借鉴优点
  • 避免重复造轮子

3. 新品榜观察

  • 打开新品榜看新上的款式
  • 找到可借鉴的新功能点

🚿 实操案例:浴室收纳款

选款理由

  • 评论数都很低(说明是比较新的款式)
  • 平均评分数较低 → 改进空间大
  • 差异化空间大

差异化方向:加收纳功能

  • 原有款式:只有上面一层置物
  • 差异化:在下面加一层(增加收纳功能)
  • 借鉴原理:功能上结合其他产品的优点

选择差异化款式的方法

  • 选择外观不太一样的产品进行模仿/借鉴
  • 不选"大差不差都是这个样子"的产品
  • 差异化要明显

📋 买家差评的统计与提炼

差评类型具体描述改进方向
重量问题比较轻、材质轻薄加重、加厚
尺寸问题尺寸不准/不合适严格把控尺寸

⚠️ 重要提醒 "评论里看到的反馈,可能也只是买家的1/10,1/10的反馈,那其实大部分都没有这个问题。"

差评只代表少数买家的声音。
多数买家是给好评的(否则链接早就卖不动了)。
差评点只是需要避开的点,不是绝对问题。
不能因为个别差评就否定整个产品。

🎯 找差异化的核心方法论

综合策略

  1. 看差评:统计差评点 → 改良产品
  2. 看新品榜:借鉴新功能
  3. 多对比:多看款式,找到外观差异大的产品
  4. 找结合点:把好产品的优点结合起来

关键认知

  • 差评 = 改进机会,但不必过度解读
  • 新品榜 = 趋势参考
  • 借鉴 ≠ 抄袭,要结合自己的产品做改良

📍 多维度决策

🏛️ 整体设计与差异化亮点

基于前两天讨论的差异化方案做出草模,核心设计特征:

  • 两块板子的结构:尺寸严格对标市场现有产品标准
  • 板子的长、厚度、高位长度、下面框架大小均参考市场上已有的产品数据
  • 差异化功能区:
    • 前面挂毛巾的位置
    • 小钩子区:用于挂浴帽、搓澡巾等浴室小件
    • 框子:用于放置瓶瓶罐罐
    • 其他浴室常用配件区

🔧 墙面固定与挂钩设计

可拆卸挂钩设计

  • 出厂时钩子直接坐在木板上
  • 木板可由客户自行定位安装
  • 考虑承重需求,预留4个螺丝孔,让用户自选

承重分级方案

  • 挂重物(湿毛巾、瓶罐)→ 建议打4个螺丝加固
  • 挂轻物(干毛巾)→ 2个螺丝即可

不可拆钩子的结构原因

  • 钩子做不可拆是为了避免单螺丝导致左右晃动
  • 采用L型折弯结构(折过去再从下边折过来),提升稳定性

📏 钩子间距与干湿分离

  • 原始钩子间距方案:相邻钩子5cm
  • 问题发现:钩子间距过近,导致干湿物品可能相互接触
  • 解决方案:
    • 把钩子数量从多减到4个
    • 钩子尺寸稍微缩小
    • 保证干湿分离的使用体验

🔩 连接铁片与螺丝数

  • 连接铁片方案:所有钩子固定在一块铁片上,铁片再与木板固定
  • 好处:客户收到货后只需把铁片钉上即可
  • 结论:考虑湿物可能造成下垂,需打3颗螺丝

⚖️ 一体式 vs 拆分式的成本权衡

对比一体式拆分式
优点减少运输体积,降低物流成本组装简单,易运输
缺点组装复杂,需拧5颗以上螺丝包装大,运费高
结论钩子做成一体不可拆(出于稳定性),其他部分可拆分框架可拆

📐 框体结构优化

  • 原始方案问题:框体过大,运输成本高
  • 改进方向:做成可拆分
  • 折叠方案设想:客户收到后只需展开+两边固定即可

📦 重量与物流成本约束

  • 平台开单率限制:7.5左右
  • 尺寸上限:不能大于2.3公斤
  • 结论:加后板会导致重量超限,倾向于不加后板

💎 包装体积优势

✅ 差异化核心优势总结 同样的功能实现,完全拆装设计 → 体积包装更小 → 配送费更低 → 利润更大 → 竞争更强
这是该产品相对市场同类用品的关键卖点。

📋 最终尺寸汇总

  • 产品尺寸:长16点多、宽6点多、高80
  • 单板重量:0点多到1公斤,总重适中
  • 五大功能区确认:12345个功能区,距离接近80
  • 门的设计:三扇门,可折叠拆卸

🪵 最终材质与线径方案

  • 安装位置:固定在上方(避免下方安装阻碍用户操作)
  • 木材材质:选用胡桃木(根据材质特性大概率判断)
  • 铁线线径规格:
    • 主铁线:5毫米线径
    • 网格线:3毫米线径
    • 挂钩:4毫米线径

📞 第一通电话:细节确认

1. 核心细节点之一:尺寸严格控制

  • 整体尺寸:详细尺寸图已发给厂家
  • 关键约束:
    • 平台要求数值不能超过45
    • 当前板子已达到42
    • 加上包装后尺寸会卡得很紧
  • 执行要求:尺寸不能做超

2. 核心细节点之二:螺丝固定

  • 关键连接点:铁架与木板之间的连接螺丝
  • 必须做紧的原因:
    • 用户要挂重物
    • 一旦脱落 → 影响产品质量和评价

3. 核心细节点之三:铁网线径

  • 初步方案:
    • 外径:5毫米
    • 内径(网格线):3毫米直径
  • 厂家反馈:5毫米可能太粗
  • 协商调整:优先满足承重需求,厂家先做一个4毫米外径 + 3毫米内径的样品

4. 核心细节点之四:板材与颜色

  • 板材厚度:已在尺寸图上标注
  • 颜色方案:参照图纸上竞品的色彩和表面处理,尽量与竞品做到一模一样

5. 打样周期协商

  • 厂家反馈:打样周期14天
  • 客户期望:
    • 7-10天出第一次样
    • 看完样品再修改
    • 争取2周内搞定样品
    • 然后直接开始生产

📞 第二通电话:样品参数采集

1. 沟通目的

  • 跟进样品是否做好
  • 索取样品打包后的尺寸重量参数(用于计算利润和运费)

2. 组装视频需求

  • 需要厂家拍摄的组装视频内容:
    • 产品从盒子里拿出的初始状态
    • 每一步安装过程
    • 所有配件展示
  • 目的:确认安装过程无bug,拆散发货给客户,必须保证用户能装上

3. 详细参数清单

  • 包装参数:包装后的尺寸、重量
  • 零件参数:板材厚度、线径规格、所有零部件
  • 测量要求:用尺子实际测量,测量时直接在对标尺子旁拍照或拍视频

💡 沟通技巧总结

  1. 先肯定再提需求:"我们这款产品看上去只是测款"
  2. 强调时间窗口的紧迫性:用"光打样就2周 + 后续修改 + 生产"链条式说理
  3. 用数据说话:5mm vs 4mm 的对比
  4. 目的导向:每提一个要求都说明原因

🎯 重点与难点

  • 重点:平台尺寸红线(45)的执行、螺丝固定的质量要求、7-10天打样周期的达成、组装视频的提前规划
  • 难点:5mm → 4mm 线径的承重验证、颜色与竞品"完全一致"的可实现度

💰 议价起点:原报价方案

1. 第一轮:试探性砍价

  • 测款阶段不会有几千套的大单
  • 7000元模具费对前期测款确实偏高
  • 不是第一次合作,希望厂家给优惠

2. 第二轮:提出历史合作模式

  • 提议复用之前的合作模式:累计到一定量后退还模具费
  • 本款产品单价高、销量小,1-2万套标准要卖三五年才回本

  • 历史标准:1-2万套返现
  • 本款提议:直接降到5000套返现,或可放宽到7000-8000套
  • 客户给出的合作价值:经常合作、推的品量不会少、长期合作双方共赢

4. 最终结果

✅ 最终结果 模具费最终定价:5000元(不是5000套,5000套返现)
即:先付5000元模具费,累计销量达5000套后退还。

💵 产品单价议价策略

1. 第一轮:报价对标

  • 客户立场:40元单价太高
  • 对标数据:
    • 已咨询其他厂家
    • 同样2.0的板材、同样做工
    • 别人报价30多元
    • 差价近10元

2. 第二轮:材料话术拆解

  • 厂家说辞:"用的材料好"
  • 客户反驳:
    • 别的厂家也说自己材料好
    • 消费者和我们自己其实分辨不出材料差异
    • 材料该用好还是用好,但别的地方能否省一点

3. 第三轮:长期合作捆绑

核心论据组合:

  • 长期合作基础
  • 这款做得好 → 后续新款还找你做
  • 卖家赚你也赚 → 双赢
  • 卖不好:把价格压下来 → 卖家承你的情
  • 后续新款开发坚定不移选你
  • 不挑其他供应商

🚨 6大核心红线

  1. 不抄袭:任何文案/图片/视频
  2. 不蹭品牌流量:用for/with介词结构
  3. 不模仿专利外观:查专利局+第三方工具
  4. 不造假:任何资料(直接关店)
  5. 不超30%溢价:差异化成本必须控制
  6. 不碰18650等敏感词:替换型号名

🎯 4个说服工厂开模条件

  1. 数量足够多:2000-3000套,多站点分摊
  2. AA开模费:起步AA,深入合作免
  3. 长期信任:1-2年合作基础
  4. VIP大圈影响力:立IP,加跨境圈子

🛠️ 4个低成本开模要点

  1. 优选材质:木材 > 不锈钢 > 塑料
  2. 局部改良:不全盘改,只改关键
  3. 同款改良:只开变动部分
  4. 造型简化:造型越简单越便宜

💬 议价5大技巧

  1. 以退为进:先承认下单量小
  2. 数据对标:用具体数字施压
  3. 未来收益绑定:与长期合作挂钩
  4. 退路预设:降低厂家防御心理
  5. 快速敲定:达成一致立即报价
