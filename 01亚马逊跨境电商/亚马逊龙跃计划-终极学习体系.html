1

产品潜力判断与高效测款

 ▼

1.1 新品推广四大痛点

  • 痛点一：转化率低 — 需区分是产品天然特性还是链接优化问题
  • 痛点二：CPC过高 — 出单词越集中、竞争越激烈，CPC越高（案例：皮卡球拍CPC近$7）
  • 痛点三：CPO过高 — CPO = CPC ÷ 转化率（示例：CPC$1，转化率10% → CPO=$10）
  • 痛点四：关键词推广难度大 — 首页曝光100%，第二页30%，第三页15%；70%客户通过搜索完成购买

1.2 产品潜力预判五维指标体系

① 新品榜前三名表现（首要指标）

新品榜第一名表现差 → 直接放弃该品类。将新品榜前三单量与自身预期对比。

② 推广手段识别

低价螺旋 前期超低价起步，前两个月亏损。站外推品 BSR排名突然跃升。黑科技 用小店铺跟卖+自发货+发红包（风险极高）。

③ CPC分层标准
CPC区间定义说明
<$1超幸福CPC家纺、箱包、假发等非标品
$1-2比较幸福2023年前$2以下算幸福
$2-4可接受需搭配客单价$30以上
>$4难受除非客单价极高，否则不建议
④ ACOS指标

合理ACOS = 毛利润率 + 15%  |  盈利ACOS < 毛利润率  |  亏损ACOS > 毛利润率

毛利润率 = (售价 - 采购成本 - 头程运费 - 佣金 - 配送费) ÷ 售价 × 100%
⑤ 转化率门槛
产品类型转化率门槛
低价标品（<$20）≥15%
非标品≥8%

1.3 CPO与TACOS深度解析

CPO（单订单广告成本）= 广告花费 ÷ 订单数 = CPC ÷ 转化率

降低CPO两条路径：压低CPC（按建议竞价初始，不要给太高）+ 提升转化率（优化Listing）。

"杀熟理论"：初始CPC给高了 → 被判定为"土豪卖家"→ 后期极难压低。

📌 无建议竞价时：有同类经验→参考平均CPC；全新品类→先给$0.5-0.6，跑3天花不出去上调10-20%
TACOS公式推导
TACOS = 广告花费 ÷ 总销售额 = ACOS ÷ (1 + 自然单/广告单)
TACOS区间状态
<8%引流不够
8%-15%合理值
>15%不利于长期盈利

降低TACOS三大方法：压低CPO、提升售价（用专享折扣而非直接提价）、增加自然单占比。

1.4 BD推品策略

类型推荐度说明
普通BD✅ 推荐通过服务商报BD，前提：至少15个评论
大促BD❌ 不推荐太明显，容易进小黑屋
无限秒杀⚠️ 谨慎容易关效率（链接被限制）
官方Woot✅ 推荐合规推品最快，但无定价权

1.5 高效测款七维测试法

  1. 测CPC — 对照CPC分层标准判断承受能力
  2. 测促销力度 — 先给10%优惠券，兑换率40-60%为合理
  3. 测转化率 — 链接优化到极致+广告词精准 → 转化率是否仍然很低？
  4. 测差评/质量 — 厂家问题可换厂家；产品通病果断放弃
  5. 测稳定出单能力 — 推品两个月日均≥10单（极高客单价除外）
  6. 测广告架构（标品） — 大词能否推动排名、精准长尾词能否转化
  7. 非标品三阶段测试 — 初始测试→扩量→精准投放
⚠️ 烘焙垫案例：客单价$11-12，转化率仅3-5%（低价标品应≥15%），降价+站外都无效→品类天然转化率低→清货放弃

关键词库搭建（标品与非标品）

 ▼

2.1 无关键词库的严重后果

  • 文案埋词靠拍脑袋 → 无依据
  • 广告选词缺乏标准 → 无法精准投放
  • 单量波动时无法追溯 → 不知哪些词的排名下降

2.2 标品 vs 非标品六大差异

维度标品非标品
外观特征一模一样或非常接近千差万别
功能属性功能性强，目标明确款式/装饰性吸引人
竞争格局同质化严重竞争相对不激烈
利润空间非常透明不透明，可达50%
出单词特征少而集中（前5词占50-60%）多而分散（每词1-3单）
CPC高低（长尾词仅$0.1-0.2）

半标品（如台灯）：外观千差万别 + 出单词集中 = 兼具两者特点。

2.3 关键词库三大核心指标

① 流量大小（ABA排名）：<2万=大词，2-5万=中上等，5-10万=中下等，>10万=小词/长尾词

② 相关性（前10自然位）：≥6个同类=强相关，3-6个=中等，0-3个=弱，0个=不相关

③ 类目转化率（商机探测器）：实际转化率通常是显示值的8-10倍

2.4 标品关键词库搭建六步法

  1. 产品画像 — 品名、驱动方式、外观、容量、场景、人群、材质
  2. 筛选10个优秀竞品ASIN — BSR前100 + 搜索结果第一页 + 新品榜前5
  3. 反查流量词 — 用卖家精灵/Sorftime等工具
  4. 筛选重点词 — 流量占比≥5%标绿色，至少保证20个重点词
  5. 词频统计 — 工具（AMZ123）或手动统计竞品标题+五点
  6. 建立完整词表 — 必须包含：关键词、ABA排名、流量大小、相关性、类目转化率、备注

2.5 标题撰写三依据 + 埋词技巧

依据：10个竞品标题参考 + 词库重点词 + 词频统计结果

📌 词频越高→越重要→越往前写。务必一次成型——标题频繁修改会导致链接被锁

2.6 ASIN分类表（广告投放专用）

类别BSR排名范围策略定位
强者ASIN比自己靠前20名以上抢流量（5个一批）
势均力敌者前后各20名竞争流量（10个一批）
弱者ASIN比自己靠后20名之后花不出多少，可忽视
出单ASIN已验证能出单持续定投

标品：ASIN表选做（加分项）；非标品：ASIN表必须做（占流量50%）。

2.7 非标品造词法（核心策略）

用高频修饰词 + 词根组合，创造更多精准长尾词：

woman + dress → woman summer dress / woman maxi dress / woman sundress / plus size summer dress

造词法广告投放原则：用修饰词造词、一个活动放5-10个词、优先开广泛匹配。

2.8 词库搭建六大铁律

  1. 流量大小必须分级（大/中上/中下/小词）
  2. 相关性只看前10自然位（≥6=强，3-6=中，0-3=弱）
  3. 词频决定标题顺序（高频前置，低频后置）
  4. 类目转化率决定广告优先级（商机探测器数据×8-10倍）
  5. 标品集中打词，非标品分散造词
  6. ASIN表与词库同等重要（尤其非标品）
3

Listing极致优化（高转化率打造）

 ▼

3.1 标题三大写法

写法公式适用场景
官方标准品牌名 + 关键词1/2/3 + 功能词 + 场景词大多数产品
高权重法核心关键词 + 逗号 + 品牌名 + 其他词标品、需推核心大词
权威链接法关键词 + 空格 + 短横杠 + 品牌名季节性产品、站外引流
⚠️ 务必一次成型——标题频繁修改会导致链接被锁

3.2 定价策略

核心原则：定价由市场决定，参考BSR前100名同类商品。新品上架先定中高价，推一周+累计出单20单以上后可降价10%（出现降价红标）。

3.3 图片优化

官方要求：主图纯白背景(RGB 255,255,255)，最长边≥1600px，商品至少占85%。

尺寸最佳实践：主图1200×1500px（手机端长图），其他图1600×1600px。

主图三大技巧：无中生有法 搭配不包含的配件提升吸引力 | 移形换影法 斜放+阴影+配件 | 背景装饰法 竞品都用时我们用了不违规

点击率决定因素：80%由图片决定，20%由标题/价格/促销决定。

3.4 五点描述与埋词

  • 可写10条（初始5条，10天半月后可编辑到10条）
  • 颜文字/图标：如灯泡图标(卖灯)、火图标(烧烤类)
  • 总分结构："Upgrade modern design"（总）+ 短横杠 + 产品词（分）
  • 埋词分配：标题至少3个词、五点至少5-10个词、A+埋词、ST写满标题五点没有的词

3.5 品牌故事

必须设置（4个ASIN模块）。要素：创始人头像 + 品牌故事文字 + 4个ASIN产品（为其他产品引流）。

3.6 高级A+模块（推荐）

  • 高级全视频 — 一个屏幕全是视频，吸引驻足
  • 优质视频图像轮播 — 视频/图片轮播
  • 包含文本的高级视频 — 视频+文字，至少2个视频
  • 高级比较表 — 关联流量，为其他产品引流
  • 高级QA — 至少5对QA，其中2对视频QA

3.7 视频体系全景

主图视频 → 高级A+视频 → 相关视频 → Post视频（免费，必须发）→ QA视频 → SBV视频（转化率最高的广告）→ SDV视频 → 3D模型（720度旋转，极大提高转化率）

3.8 参数栏三个位置

  1. 搜索结果左侧参数栏 — 不填写→客户筛选时直接过滤掉
  2. 标题和五点之间 — 对标10个竞品，缺失→类目排名出不来
  3. Product Information — 后台没有的参数→库存加载文件→大类目节点表单

3.9 免费关联流量矩阵

虚拟捆绑 老带新 | New Model 新链旧链关联 | 4个ASIN模块 为新品引流 | 高级比较表 关联入口 | Post 图片+视频

4

Review获取（快速安全获取+VINE计划）

 ▼

4.1 VINE计划深度解析

三档注册：免费档（2个）→ 高客单价/电器类；$75档 → 约10个评论；$200档 → 需要更多评论。

内推期 vs 公推期
价格区间内推期时长20天内进入公推概率
$10以下1-4天70%
$10-2010-20天50%
$20以上20天以上更低
⚠️ 种子链接合并时机（极其重要）：两个条件必须同时满足——①新链接自然排名推到第三页左右有单量有排名 ②老链接断货超过一个月。合并太早→老链权重高抢流量，新品期直接结束！

4.2 明信片获取Review

透明计划明信片（官方）：加入透明计划后可在包裹内放小卡片（品牌名/电话/地址/邮箱/官网/社媒/二维码）。

第三方直邮明信片（PII接口）：通过对接亚马逊PII开发者接口读取客户地址，直接邮寄卡片，留评率提升约5%。

4.3 后台索评

官方一键索评：送达后5-30天内的订单可索评，建议每周五执行一次，提升留评率约1-2%。

自定义索评邮件法
  1. 第一封：产品发出当天（仅自发货）→ 建立沟通
  2. 第二封：客户收到货当天 → 快速拦截差评！
  3. 第三封：收到货5-7天后 → 用了几天才有真实体验
  4. 第四封：客户确认满意后 → "是否介意分享购物经验？"附直达链接
📌 敏感词：❌ review/feedback   ✅ opinion/suggestion/experience/购物经验

4.4 差评处理四步法

  1. Step 1：分析差评→判断问题类型，确定责任方
  2. Step 2：找差评订单号
  3. Step 3：联系客户→承认错误+解释原因+给出可选方案
  4. Step 4：跟进补救→退款/补发后等3-5天→跟进确认满意度
差评类型责任方处理方式
产品瑕疵/配件缺失卖家补发/换货
物流/包装损坏亚马逊联系客服删除+向客户道歉
客户使用问题误解提供正确使用指导
5

六大促销效果最大化详解

 ▼

5.1 优惠券（Coupon）

折扣最大50% off，最小5%。费用：北美站$0.6，欧洲站£0.6/€0.6。最大时长90天。

权重机制：BSR排名越靠前→排位越靠前；领取兑换率越高→排位越靠前。

兑换率合理区间40-60%。低于40%→力度不够；高于60%→开大了应降低。

5.2 会员专享折扣（Prime Exclusive Discount）

折扣范围5%-80%。核心要求：折扣后价格必须不高于非会员非促销时价格的8折。

大促时（黑五网一）需同时满足：比最近30天最低价低 + 参考价打8折 + 会员比非会员优惠至少20%。

目标价格 = 当前售价 ÷ 0.85 × 0.8
⚠️ 专享折扣会影响秒杀价格（秒杀价必须≤专享折扣价），但优惠券不影响秒杀价

5.3 购买折扣（Purchase Discount）

必须大于5%才显示绿标。常用：最低买5件，减免6%。本质是流量入口矩阵，在商品页面形成多个展示位。

5.4 社交媒体促销（Social Media Promo）

折扣上限50%，仅美国站可用。先提价再打折（提到打5折不亏损的价格）。

⚠️ 防零元购铁律（极其重要）
① 先关优惠券，再开社媒（不能反过来）
② 社媒关掉后等4小时再开优惠券
③ 折扣叠加总额必须在可承受范围内
④ 勾选"一次结算其中一件商品"

整点设置法：每次设置最小3天，每个整点一个社媒促销（0-24点共24个），互不叠加且被红人看到概率更大。

5.5 限时秒杀（LD）

资格：近30天最低价打15%以上（黑五网一20%）、4星以上、退货率低于5%。

LD vs BD核心区别：LD只能瞬时冲量（不能推关键词排名），BD可以推动关键词自然排名。

秒杀操作节奏：开始→广告预算翻倍→目标2小时内进度条跑到90%→进度慢则开大额优惠券→达90%后取消优惠券→秒杀跳到好排位。

5.6 七天秒杀（BD）八天执行计划

阶段操作
前1-2天放站外 + 广告预算翻倍
第1天（最关键）必须爆单→不爆开大额优惠券
第2-5天每天添加广告否定词+关注排名变动
第6-7天每天降预算15-30%
秒杀后（最关键）至少3天不调广告（最好一周）+保持大额优惠券
⚠️ 如果总单量达标但关键词自然排名一动不动→BD非常失败，白跑了一次！

5.7 三大铁律

  1. 防零元购铁律：先关优惠券再开社媒，关社媒等4小时再开优惠券
  2. LD/BD不叠加原则：秒杀开始时关闭站外促销，防止亏损
  3. BD后稳字当头：秒杀结束后至少3天不动广告，保持缓冲
6

亚马逊三大广告体系全解读

 ▼

6.1 广告核心作用

流量递增原则："1+1>2"效应——预算A→流量A；预算2A→流量2A+B（递增部分B）。初始预算$10起，每天花完后逐步加（5次点击费用），当地时间零时加。

马太效应：表现越好→流量越多质量越高；表现越差→流量越少。应对：先下调竞价/budget10-20%，两次仍不好→直接关掉开新广告。

6.2 不适合打广告的六类产品

  1. 超蓝海市场小类目（Top5单量不到30单）
  2. 客单价低于$10（ACoS极难控在30%内）
  3. FBM自发货标准件
  4. 转化率超低：<$20标品<15%、<$20非标品<8%
  5. 转化差于同赛道产品（评分<4.0）
  6. 无广告资格（无购物车/受限品类）

6.3 三个30%原则

  1. 毛利润率 ≥ 30%（净利润推算：毛利30%-TACoS15%-退货5%-仓储≈7-8%）
  2. 广告订单占比 ≤ 30%（稳定期后须控制）
  3. 广告平均ACoS ≤ 30%（中高客单价<30%很健康；低客单价<45%算正常）

6.4 核心广告规律

前三页定律：广告打不到前三页都是无效的（首页100%曝光→第二页30%→第三页15%→之后几乎无）。

实际CPC = (下一位出价×下一位质量得分) ÷ 自己质量得分 + 0.01

广告质量得分 = 出价 × 产品质量得分。影响因素：相关性（高）→ 点击率（80%由主图决定）→ 转化率 → 广告历史表现。

6.5 广告投放结构原则

一产品一广告组合（绝不相互蹭广告）。单词单组单活动：ABA<2万→1活动1组1词；2-10万→1活动1组2-3词；>10万→10个左右一组打广泛。

6.6 七种广告目的

  1. 付费引流 + 2. 提高销量（所有广告都有）
  2. 推动关键词排名：最强=SP精准匹配
  3. 拓词拓流：SP广泛匹配+自动广告
  4. 提升关联流量：SP/SB/SD的ASIN定投
  5. 盈利性广告：低竞价捡漏+低预算多活动+SBV/SBP

6.7 SP/SB/SD三大广告对比

维度SP商品推广SB品牌推广SD展示型推广
占坑位✅ 核心能力✅ 辅助✅ 有限
推关键词排名✅ 最强（精准）❌ 几乎无⚠️ 极微弱
品牌宣传❌ 无✅ 有标志✅ 有宣传
预算占比建议约80-85%约5-10%约5-10%
核心目的推排名+出单卡位+品牌关联+扩曝光

6.8 SP广告详解

三种匹配方式：精准匹配（推排名，最精准）| 词组匹配（拓词）| 广泛匹配（拓词拓流，入口最大）

自动广告四种匹配：紧密匹配（上架即开）| 宽泛匹配（上架即开）| 同类商品（标品有评论后开，非标品上架即开）| 关联商品（上架2个月后再开）

6.9 SB广告加号玩法

关键词前加"+"号 = 限定必须包含该词：如"+table +lamp"→必须同时包含table和lamp才触发。用广泛匹配的价格拿精准的效果（CPC低至$0.4-0.5）。

⚠️ vCPM真相：vCPM的订单是假订单！重复记录导致ACoS极低。所有vCPM订单直接忽视，SB的vCPM不要开，SD的vCPM可少量开。

链接数据表分析优化

 ▼

10.1 六大运营表单

表单核心用途
① 类目排名表监控整体市场地位与单量趋势。大类排名1万≈20单临界点
② 关键词排名表追踪搜索流量获取能力（至少关注30个核心词）
③ 流量与CPC广告效果表分析流量结构与广告效率（含异常分析方法）

10.2 异常分析方法

单量突然上涨：排查顺序 → 看广告花费是否变化 → 看CPC → 对比自然单/广告单 → 看转化率 → 是否开了大额优惠券/BD。

BD识别特征：单量暴增到50-100+单，广告花费翻倍，访客猛增，但转化率没降低（站外会降低转化率，BD不会）。

10.3 核心原则

  1. 拒绝拍脑袋：所有决策必须基于数据
  2. 可回溯性：每个操作都要数据化记录
  3. 统一时间统计：关键词排名每天统计，其他数据每周五下午统一统计

11.2 广告位报告分析

微观分析：逐广告活动比较三个位置（搜索结果顶部/其余位置/商品页面）的CTR、转化率、ACOS。

宏观分析：汇总所有数据 → 追求高点击率→搜索结果顶部；追求增量→其余位置；追求高转化→顶部。

11.3 搜索词展示量份额报告（最重要）

五类分类法：

分类策略
高点击+低ACOS 🟢不在前三→扩大流量（纵向加预算/横向加广告类型）
高点击+高ACOS 🟠下调竞价20-30%
高点击+不转化 🔴相关性强→过段时间再看；相关性弱→直接关；200次0单→否定
低点击+低ACOS 🟩聚合后按流量大小分组（大词5个一组，小词10个一组）
低点击+不转化 ⬜找"点击少"的原因：预算不足→加预算；竞价过低→提高20%

11.4 预算报告

判断标准：平均预算活跃时间≥70%→预算充足；<70%+ACOS合理→预算不足，逐步加预算。

⚠️ 70%基准原因：一天24小时，人清醒约16小时，16÷24≈66.7%，按70%作为基准线
