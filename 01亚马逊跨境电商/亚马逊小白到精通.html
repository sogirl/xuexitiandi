0.11 划线价SOP（L开头人为设置法）

  1. 后台3个价格字段全部调成目标划线价（如11.99）
  2. 让产品在该价格出3~5单
  3. 立即将"销售价"（带开始/结束日期字段）调回实际售价（如8.99）
  4. 等待3~7天（2026年5月新规后需5~10天）→ 划线价生效

📌 划线价的两大商业价值

  • 影响秒杀价：划线价11.99+历史最低8.99 → 秒杀价会在8.89~8.99

情况处理方式
第一次下架等几小时可自行重新上架，勾选承诺

1.3 核心铁律：不是我们的原因，绝不承认（全课最重要）

🔴 为什么不能承认？

点"承认"虽然界面投诉会消失，但亚马逊会在你的"买家记号"里留下一条记录（=警察局的案底）。下一次平台扫号时，有案底的账号会被优先扫到。

亚马逊扫号的三波逻辑：

  1. 第一波：确有违规的账号
  2. 第二波：①注册资料有问题的账号；②违规且案底特别多的账号
  3. 第三波：案底少（只有2-3条），因数量不够被"拉来凑数"的账号

问题处理的优先级逻辑：①先让亚马逊客服内部申诉（成功率较高）→ ②自行申诉（先算概率）→ ③找服务商（多问几家，问清"误判"还是"承认"渠道）。

1.4 侵权投诉实战案例（服务商申诉全过程）

学员的BuyBox链接每天出几百单，被投诉文案侵权（德国独立站图文几乎一模一样）。

  • 第一家服务商：不接（败诉概率高）
  • 第二家服务商"智慧西游"：8000元解决，账号回来了

关键提醒：找服务商要问清用什么渠道解决？是"误判"还是"承认"？——误判=无案底，承认=留案底。

1.5 商品状况投诉的标准申诉流程（自行解决，无案底）

  1. 登录卖家后台 → 找到"商品状况买家投诉"申诉入口
  2. 先去帮助页面开一个 case：库存调查case
  3. 等待1-2天，仓库出调查结果（可能附照片）
  4. 回到申诉入口，选择"误判"（"买家原因"或"仓库原因"）
  5. 部分情况会让你写OPA（行动计划）

成功率：按此流程走，成功率几乎99%。自行解决=不留案底。

2.1 邮箱设置规范

2.2 电话号码设置规范

  • 必须是干净的真实手机号码，建议使用法人名下号码
  • 严禁使用虚拟电话号码（香港虚拟卡被扫号概率极大）
  • 号码归属地优先级：与营业执照注册地一致 > 与法人身份证地址一致

  • 必须用公司名义注册第三方收款账户（禁止个人）
  • 2023年封号事件教训：美国站卖家同步开通加拿大站但未绑定收款→加拿大站出单产生余额→打款失败2-3次后账号被限制销售→加拿大站限制会影响美国站

2.6 新账号养号关键期（前三个月危险期）

⏰ 危险期特征

  • 时间范围：账号注册后前三个月
  • 信任度：接近零
  • 风险特征：任何小问题都可能导致账号审核、重认证甚至封号

危险期操作禁忌：

  • ✗ 频繁更换网络 → 账号锁定
  • ✗ 使用第三方验证器 → 身份存疑
  • ✗ 大量上架产品（5-10款） → 触发审核
  • ✗ 频繁修改资料 → 启动风控

养号正确操作流程：

  1. 账号下来后第一步：进入卖家大学学习课程，模拟新手正常行为，暂不上架
  2. 完成品牌注册后再上架产品
  3. 首批上架1-2款 → 等待第一笔资金成功提款后，再上第二批

账号安全等级提升路径：

注册成功 → 危险期（信任度0）→ 第一笔提款成功 → 安全等级提升（3-5级）

2.8 香港店铺注册专题

地址验证三件套（必须索要）：①印花税单据 ②租约合同 ③水电账单

填写规范：国家/地区选"中国香港"（不是中国）；办公地址填注册地址（不是国内地址）。

两套合规注册方案：

  • 方案一：国内董事模式（推荐落地率更高）—— 国内身份证+国内信用卡+国内手机号+第三方收款（香港公司名注册）

3.1 关税问题（消除焦虑）

  • 美国关税对中小卖影响很小：头程费用最高约2~3元/公斤
  • 受影响最大：大卖（安克）、线下实体外贸、美国商超（必须实报）
  • 散装拼柜的"猫腻"机制：中小卖一个柜子有十几个品类，海关不可能逐一查验
  • 结论：即便关税加到100%，对中小卖大约只是0.5~1元/公斤的区别

  3. 侵权问题（分涉嫌侵权、知识产权侵权两种）

为什么注册时一定要选"录视频"而非"拍人脸"？

3.5 视频认证失败的两大真实案例

案例1：信用卡非法人名下——客服质问"为什么不是法人的卡？"，学员答"是朋友"，要求证明朋友关系→学员手机里没有合照→账号挂掉。

案例2：虚拟浏览器露出破绽——用资料浏览器（IP显示香港），实际在深圳。客服反复问"你现在在哪里"，坚持说"在香港"。客服问"用的什么浏览器"，答"谷歌浏览器"（与香港IP+资料浏览器矛盾）→账号挂掉。

视频认证铁律：只要有一个问题答得不顺，账号必然挂；答得没问题，几乎不会有问题。不懂就老实说"作为新手卖家不太懂"，真实回答即可，不要硬编。

3.6 商标与品牌"偷梁换柱"续命技巧

适用场景：大店（月利润10万+，年利润上百万）挂掉、找很多人救不回来。

  1. 第一店铺（已挂）持有商标"AB"，注册在24类
  2. 注册第二个店铺（或备用店铺）
  3. 第二店铺重新注册同名商标"AB"，但必须换类目（如25类）
  4. 约1周下TM标
  5. 用第二店铺跟卖第一店铺的所有产品（商标相同，可正常跟卖）
  6. 利润在第二店铺延续

第13章 选品的九大坑（不建议做的产品）

13.1 小卖家的赛道定位

小卖家的赛道是低成本、低广告费（日均20-30美金，不超过50美金）的小类目，而非大资金、上万美金广告费的模式。

选品分两个阶段：①方向要对（避开九类坑）→ ②利润的把控（通过选品报告反复练习体感）。

13.2 九大坑详解

坑核心问题典型案例
1. 高质量差异化/独家私模高质量=高成本=高售价学员砸近10万失败
2. 过度组合和赠品成本脱节+关键词跑乱夏季交流会小礼品
3. 高逼格包装增加0.5-2元成本飞机盒/礼品盒
4. 盲目追求大容量大卖"黑白通吃"轮胎修补工具
5. 突然爆火的产品发明专利带火的爆款床角产品、蜂鸟喂食器
6. 高门槛产品认证费用高+运营难医疗类、危险品、粉末、带电、带磁
7. 新奇特产品亚马逊无逛街概念+无关键词指尖陀螺
8. 投机心理/擦边球账号活不了多久明知侵权还做
9. 季节性/节日性节日一天、节后退货万圣节/圣诞节、蜂鸟喂食器

13.4 推品的五个阶段

第14章 选品模式拆解与TIO案例

14.1 选品的两个核心淘汰原则（一票否决）

序号淘汰原则核心原因
1侵权擦边球淘汰涉及店铺安全问题（资金冻结、店铺受限、资金被划走）
2没有利润淘汰利润过低（如一单仅1-5美金利润）

14.2 小卖家选品三大禁忌

  1. 不选主观意识特别强的产品（裙子审美分散）
  2. 不选选择意识特别强的产品（颜色色差导致退货）
  3. 不选多功能产品（两类买家都会跳出）

14.3 刚需产品的真正定义

刚需 = 功能性产品：买家因为产品能解决某个具体功能问题而购买，不会因为颜色、花纹、审美偏好而改变购买决策。

正面案例：锤子——因为它能锤钉子才买；厨房水槽过滤网——"刚刚好"原则。

14.4 伪刚需产品解析

虽有功能性，但每款产品都有明显的功能缺陷，买家会不断"踩坑-淘汰-再买"。案例：耳环收纳盒（旋转展示型、木质盒子、亚克力抽屉式、文件夹式，每一款都有缺陷）。

14.5 TIO侵权案例深度拆解（历时52天）

案例背景

  • 时间：2025年4月15日收到专利侵权投诉
  • 关键事实：卖家上架时已做改动，上半部分与侵权产品相同，下半部分不同

核心教训：改款产品只要与原创有50%以上相似度，对方就有权投诉。对方搞你的目的往往不是投诉成功，而是拖垮你。

完整时间轴

日期事件
4月15日收到专利侵权投诉
5月12日原告第一轮报价：75,200美金（约49万人民币）
5月22日原告第七轮报价：22,000美金
5月29日找应诉律师
5月30日应诉律师做反PI
6月7日反PI成功，和解金4,000美金 + 律师费6,000美金 = 总花费10,000美金

收到侵权投诉后的标准动作

  1. 立即提款：第一时间把后台资金提出
  2. 联系亚马逊客服获取案件号
  3. 检查邮箱：查看垃圾邮件中是否有律所邮件
  4. 决定是否停售
  5. 选择靠谱服务商/律师
  6. 联系供应商：寻找资金冻结量相近的卖家组团谈判
  7. 21天内回应：否则缺席判决，账户资金全部划走

阶段类型收费标准付费方式

第二轮律师代理约1,500美金以上必须预付

  • PI（Preliminary Injunction）：初步禁令，资金将无限期冻结
  • 反PI：申请解除资金冻结（成功后和解金大幅降低）

19.3 利润计算实操

利润 = 价格 - 采购价 - 头程费用 - 佣金 - 配送费 利润率 = 利润 / 价格

手工计算示例（售价12.99美金）：

  1. 佣金：12.99 × 0.85 = 11.04美金
  2. 扣除配送费：11.04 - 4.38 = 6.66美金
  3. 换算人民币：6.66 × 6.8 = 45.2元
  4. 扣除成本：45.2 - 10 - 4 = 31.2元利润
  5. 换算美金利润：31.2 / 6.8 = 4.6美金

利润率标准：建议30%左右。

利润 = 产品售价 - 采购价 - 头程费用 - FBA配送费 - 佣金 毛利率 = 毛利 ÷ 产品售价 × 100%

实例演算（售价9.99美金）

Step 1: 9.99 × 85% = 8.49美金（扣佣后）
Step 2: 8.49 - 3.05 = 5.44美金
Step 3: 5.44 × 6.7汇率 = 36.4元
Step 4: 36.4 - 6（采购）- 5（头程）= 25.4元利润
Step 5: 25.4 ÷ 6.7 = 3.7美金利润
盈亏平衡点 = 9.99 - 3.7 = 6.39美金

利润率底线要求：毛利率 ≥ 30%（广告费用未计入）。
