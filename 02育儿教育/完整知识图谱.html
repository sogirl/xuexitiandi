L3·1还原思想（逆推思想）
名师点金 核心思维 贯穿课
🎯 一句话心法：还原思想就是 "走回去"。一个动作的正着走与倒着走互为逆过程。
📚 概念引入：刘老师以「网上买礼物、退货」全过程开篇：要买礼物，依次打开包装纸→盒子→布袋→取出礼物；退货则把动作倒着做一遍。这就是还原思想——一个动作的"正着走"与"倒着走"互为逆过程。

📐 还原思想三大底层依据

  1. 逆运算关系：减法是加法的逆运算，除法是乘法的逆运算。
  2. 等式基本性质：等式两边同加、同减、同乘、同除，值不变。
  3. 操作的可逆性：任何正向操作都存在对应的反向操作。
🔑 必背四字口诀： 它乘我除，它减我加，它加我减，它除我乘

📋 六类问题的还原套路

问题类型还原策略代表例题
倍增类往前推一个时间单位即除以倍数虫子24小时长到20cm，求5cm需几小时
投放份数类多投放几份等价于多乘几个24份水草多少天长满
四则运算类用"它乘我除，它加我减"逆推一个数 ×4 +6 -10 ×2 = 88
分割类分段画图，逐段×2还原铁丝剪半剩14分米，求原长
卖半类同一逻辑的连续分割西瓜卖两次剩10只，求原数
多人分配类设终态为统一变量，倒推求差三人玻璃球最终同样多，求原差

📝 例题详解

例1：虫子每小时长一倍

题目：虫子每小时长一倍，24小时能长到20厘米。长到5厘米时需要多少小时？

还原过程（从20厘米往前推）：

  • 20 ÷ 2 = 10（23小时）
  • 10 ÷ 2 = 5（22小时）
例2：水草每天长一倍（变式）

题目：1份水草长满池塘需要26天。如果开始放进4份同样的水草，多少天可以长满？

分析：4份水草 = 已经多乘了2个2（即2×2=4），因此只需 26 - 2 = 24天长满。

例3：四则运算的逆推还原

题目：一个数的4倍，加上6，减去10，再乘2，得88。求这个数。

起点(88) → 还原×2（88÷2=44）→ 还原-10（44+10=54）→ 还原+6（54-6=48）→ 还原×4（48÷4=12）

验证：12×4=48 → 48+6=54 → 54-10=44 → 44×2=88 ✓

例4：铁丝问题（分割类）

题目：一根铁丝，第一次剪去一半，第二次再剪去余下的一半，这时还剩14分米。原来长多少？

关键方法：分开画图，一个单位一条线段。

  • 14分米 = 第二段的一半 → 第二段原长 = 14×2 = 28分米
  • 28分米 = 整根铁丝的一半 → 整根铁丝 = 28×2 = 56分米
例5：玻璃球问题（多人分配类）

题目：小松、小明、小航各有玻璃球若干个。如果小松给小明10个，小明给小航6个，三人玻璃球个数同样多。小松比小航原来多多少个？

还原过程（三人最终都为x）：

步骤小松小明小航
最终态xxx
还原"小明给小航6个"xx+6x-6
还原"小松给小明10个"x+10x-4x-6

答案：小松比小航多16个（差值 = 所有转移量之和 = 10+6 = 16）。

漏掉小括号（先把正向过程写出来） · 还原方向搞反（牢记"它乘我除，它加我减"） · 多人问题求各人原数过繁（直接求差：所有转移量求和） · 分割问题画图混乱（每次操作画独立线段）。
我们解决问题的时候，不仅可以正着想，还可以怎么样从结果往前想。
L3·2合理安排时间（运筹优化）
名师点金 生活迁移 并行思维
🎯 一句话心法：合理安排时间就是 运筹优化 —— 识别串行与并行的边界，让主线最短，让可并行任务充分重叠。
📚 概念搭桥：时刻与时间
  • 时刻：一个具体的"点"（如早上7点）。
  • 时间：两个时刻之间的"线段"（如7点到8点经过1小时）。
  • 形象比喻：时刻是点，时间是线段。

🎨 画图三原则（必背）

  1. 能并行的线段左端对齐；
  2. 主线任务不能错位；
  3. 总耗时 = 主线各段时间求和（不含并行部分）。
🔑 方法口诀：拉一条线 → 串行排队 → 并行嵌进 → 取最长

🏆 五大典型场景

场景1：沏茶问题（找主线+并行）

题目：给客人沏茶——烧开水12分钟，洗茶杯2分钟，买茶叶8分钟，放茶叶泡茶1分钟。

最优安排：0分钟开始烧开水（同时洗茶杯2分钟、买茶叶8分钟）→ 12分钟水开放茶叶泡茶1分钟 → 13分钟完成。

关键：泡茶必须等水烧开 → "烧水→泡茶"是主线；洗杯、买茶叶可并行。

场景2：炒鸡蛋问题（单位统一+可前置）

题目：敲蛋10秒、切葱花20秒、搅拌20秒、烧油1分钟、炒蛋2分钟、装盘10秒。最少多少秒？

Step 1：单位统一（烧油60秒，炒蛋120秒）

Step 2：主线 = 烧油→炒蛋→装盘（不可并行）

关键洞察：油可以先烧！烧油同时敲蛋、切葱花、搅蛋。

答案：主线耗时 = 60+120+10 = 190秒。

场景3：接水等候问题（加权求和最小化）

题目：四个小朋友打水，时间分别是2、4、5、3分钟，只有一个水龙头，怎样安排使接水和等候时间总和最短？

累积耗时 = 4A + 3B + 2C + 1D

最优原则：要使加权求和最小，A取最小、B取次小、C取再次、D取最大。

时间排序2、3、4、5 → 总累积耗时 = 30分钟。

核心思想：加权求和最小化——系数大的变量取较小值。

场景4：看病等候变式（多人多医生）

变式1：四名同学打针时间1、2、3、4分钟，1个医生 → 4×1+3×2+2×3+4 = 20分钟。

变式2：4名病人，2名医生 → 2A+2B+C+D = 2×1+2×2+3+4 = 13分钟。

场景5：煎饼问题（拆面法）

题目：平底锅每次最多放2个饼，煎一个饼要2分钟（正反各1分钟）。煎3个饼至少几分钟？

错误方法：1号正→1号反→3号正→3号反（4分钟，锅有空隙）

最优方法：1号正+2号正 → 1号反+3号正 → 2号反+3号反（3分钟，锅无空隙）

通用拆面法：总面数 = 饼数×2；轮数 = 总面数÷锅容量；总时间 = 轮数×每面耗时。

饼数总面数轮数总耗时

L3·3枚举总论（有序与无序）
名师点金 计数基础 铺垫乘法原理
🎯 一句话心法：分你我就是有序，不分你我就是无序。枚举是乘法原理的基础，不要急于学乘法原理、加法原理，必须先把枚举学扎实。

📐 枚举两大根本方法

  1. 标数法：数一个标一个，建立一一对应关系，避免遗漏。
  2. 分类法：按类别分组计数，每类打不同标记，避免重复。

📐 枚举两大根本原则

原则1：不重不漏 + 分类
枚举必须不重复、不遗漏。最常用工具是"分类"：把研究对象分成互不重叠的几类。
原则2：分清有序与无序
通俗判断标准：先选A再选B，和先选B再选A，一不一样？ 一样就是无序，不一样就是有序。
🔑 核心规律：单向动作（打电话） → 无序；双向动作（发微信） → 有序

📝 例题详解

例2：打电话 vs 发微信

题目1：平平、行行、乐乐互相打电话拜年，需要打几个？

分析：打电话不分你和我 → 3个电话。

例3：两个自然数乘积为12（进阶辨析）

所有算式：1×12、2×6、3×4、4×3、6×2、12×1（共6个）

场景是否区分顺序答案
求"和"（1+12=12+1）不分3个
求"算式"（1×12≠12×1）分6个
长方形（长≥宽）不分3个

核心规律：同一组数字，问的问题不同，是否区分顺序就不同。

例4：分物问题（有序无序的应用）

📋 判断有序/无序的三大关键词

关键词通常意味着
"分给"、"发给"有序（指定对象）
"分成"、"组成"无序（无归属）
"先……再……"有序（有时序）
心里默数导致遗漏（用标数法） · 分类时重复计数（不同类别打不同标记） · 同一题不区分"求和"与"求算式"（看清题目问的是什么） · 分物题默认规则（小学默认每人至少1个）。
题可以千变万化，什么是不变的？分不分你我，有序还是无序。
L3·4无序枚举（综合训练）
名师点金 枚举综合
🎯 一句话心法：无序枚举 = 不分先后，先选A再选B和先选B再选A是一样的。

📝 例题详解

例1：三人握手次数

甲、乙、丙握手：甲-乙、甲-丙、乙-丙，共3次。

口诀：先选出一个人当"主语"，再依次与剩下的人配对，避免重复。

公式雏形：n个人的无序两人组数 = n(n-1)/2。

例2：硬币凑钱

两种面值硬币各一枚（1分、2分），能组成多少种不同的钱数？

用两枚：1+2=3（1种）；用一枚：1分、2分（2种）→ 共3种（{1,2,3}）。

例3：枚举的两大核心方法

核心方法一：分类（按某个维度切分成不重叠的几类）。
核心方法二：考虑有序/无序（先选A再选B和先选B再选A算不算同一种）。

例4：1元/2元/5元凑18元的风筝付款方式

用1元、2元、5元，不多于10张，至少用两种面额，买18元风筝。

分类标准：以5元的张数分类

类别张数细节方案数
3张5元剩余3元（1张2+1张1 / 3张1）2种
2张5元剩余8元（5种组合）5种
1张5元剩余13元（3种合法组合）3种
0张5元只用2元和1元（至少两种面额）1种
总计11种

核心难点：分类要完整（5元的张数从0到3全部覆盖）+ 每类内枚举要不重不漏。

例5：从1到6中取两数和大于6

枚举（i6）：(2,5)(2,6)(3,4)(3,5)(3,6)(4,5)(4,6)(5,6)(1,6) → 9种。

心法：用"小数在前"规则避免重复。

L3·5有序枚举（排列问题）
名师点金 排列问题
🎯 一句话心法：有序枚举 = 分你我、分顺序，先选A再选B和先选B再选A不一样。

📐 有序枚举的四大通用步骤

  1. 判断有序/无序：决定计数方式（A→B与B→A是否相同）
  2. 分类：把整体切成互不重叠的几类
  3. 每类逐一枚举：在每类内部把所有可能列出来
  4. 求和：把每类的数量加起来

L3·6加减混合运算（算力核心）
算力核心 和差不变
🎯 一句话心法：算力 = 理解，不是刷题。算力的核心工具是"和差不变"的技巧。
📚 概念搭桥：位值原则
  • 123 = 1个百 + 2个十 + 3个一
  • 238 = 23个十 + 8个一（把"23"看作一个整体单位）
这是小学最重要的位值原则，所有后续简算技巧都建立在此基础上。

📚 不一样的加法交换律（核心新概念）

普通加法交换律：34 + 25 = 25 + 34（两个加数整体交换位置）。
不一样的加法交换律：不交换整个加数，而是按"相同的数位"进行交换，和也不变。
例如：34 + 25 → 35 + 24（个位4↔5，十位3↔2）。
结论：相同的数位（十位与十位、个位与个位）进行交换，和不变。

📐 五大核心定律

📌 加法「和不变」5条定律

  1. 加法交换律：A + B = B + A
  2. 加法结合律：A + B + C = A + (B + C)
  3. 不一样的加法交换律（按位交换）
  4. 来回给/来回分：和不变（不分对象如何流动，总数不变）
    例：78 + 99 → 减1给1 → 77 + 100
  5. 借来还去：为了凑整，向别人"借"一个数，算完后必须"还"回去
    例：78 + 99 = 78 + (100 − 1) = 78 + 100 − 1 = 177

📌 减法「差不变」2条定律

  1. 同加同减差不变：被减数和减数同时加上或同时减去同一个数，差不变
    例：178 − 99 = (178 + 1) − (99 + 1) = 179 − 100 = 79
  2. 连续减去两数 = 减去两数之和：A − B − C = A − (B + C)

📌 同级运算：数字和符号可以一起搬家

关键规则：没有小括号的同级运算（加减混合 或 乘除混合），可以把数字和它前面的运算符号一起"搬家"，结果不变。
例：17 + 9 − 7 = 17 − 7 + 9 = 19（搬家）

📌 A − B − C 的小括号去掉

死记口诀：A − (B − C) = A − B + C（括号前是减号要变号） 差不变解释：让括号内变0最简单 例：173 − (93 − 29) = 173 − 93 − 29

📝 例题

例1：拆数凑整
91 = 90 + 1 = 100 - 9 99 = 100 - 1 102 = 100 + 2 168 = 170 - 2 199 = 200 - 1 999 = 1000 - 1

关键思想：拆出整十、整百、整千，便于口算。

例2：加法简算·拆分法

36 + 47 = 36 + 4 + 43 = 40 + 43 = 83（把47拆成4+43，因为36+4=40是整十数）。

例3：减法简算·差不变

72 − 46：
方法1：72 − 42 − 4 = 30 − 4 = 26
方法2：72 = 70 + 2 → 70 − 46 + 2 = 24 + 2 = 26（数字带符号搬家）
方法3：63 − 28 = 65 − 30 = 35（加2）

"挣了19块、花了18块、挣了17块、花了16块、挣了15块、花了14块、又挣了13块"

等价于：(19-18) + (17-16) + (15-14) + 13 = 1 + 1 + 1 + 13 = 16

L3·7测量与最短路线
几何启蒙 点 vs 段
🎯 一句话心法：刻度是点，长度是段；两点之间线段最短；点到直线垂线段最短。
📚 概念搭桥：刻度 vs 长度
  • 刻度：尺子上标的数字（0、1、2、3、4、5……），是一个点。
  • 长度：两个刻度之间那一段线段的长度（10厘米等）。
  • 时刻 vs 时间：七点（点）→ 七点到八点（间隔）。
  • 与植树问题伏笔：树 = 刻度（点），间距 = 长度（段）。

📐 最短路线两大公理

公理一：两点之间线段最短（线段的长度叫这两点之间的距离）。 公理二：从直线外一点到这条直线所画的线段中，垂线段最短。 推论：三角形的两边之和大于第三边。

📐 比较线段长短的3种方法

  1. 一端比齐：把线段的一端对齐，看另一端。
  2. 同加同减（差不变）：起点相同，同减同样长，谁有剩余谁更长。
  3. 量化思想：分成相同小段，数段数（段数最多=最长）。

📝 例题

例1：跳远成绩测量（垂线段最短）

张强同学跳远，测量成绩时应在离起跳线最近的脚印脚跟处取一点，并向起跳线做垂直线段。
应用价值：跳远、跳高等体育测量都基于此原理。

例2：机器人走格子（A到B的距离）

机器人先南1米→东1米→南2米→东2米→南1米到B点。A到B的水平位移=1+2=3米，垂直位移=1+2+1=4米。

方法1：勾三股四弦五 → 5米。

方法2（奥数思维核心）：

AC < AB + BC = 7，所以不是7米 BC = 4是垂线段，所以AC > 4 AC > 3（因为比斜边短） 排除3、4、7，只能选 5米

老师重点讲方法2（排除法）：不靠死记"勾股定理"，靠排除与推理。

例3：方格图中黑猫白猫捉老鼠

同时出发，速度相同，谁先到？

方法1（标准单位计数）：黑猫6+3+3=12单位；白猫6+6+3=15单位 → 黑猫先到。

方法2（同加同减差不变）：不去数总长度，逐段消去。消去后黑猫先到。

结论：当路径可通过"逐段对齐消去"完全配对时，两人同时到达。

L3·8植树问题（棵数 vs 段数）
几何应用 棵数段数
🎯 一句话心法：所有植树问题，画图就迎刃而解。画最简单的图（如"两棵树一段"），判断棵数和段数的关系。

📐 概念搭桥：时刻/时间 与 刻度/长度

概念含义
时刻（点）七点、八点、九点
时间（间隔）7点到9点 = 2个小时
刻度（点）1、2、3、4
长度（段）1和3之间长度是2

核心结论：n个连续刻度之间，长度 = n − 1（n棵树 = n−1段间距的原理）

📐 植树问题的本质

本质：研究间隔数（段数）与树数（棵数）之间的关系。
把尺子画出来植树（共植6棵树）：6棵树只有 5个间隔。

📐 4种场景下的棵数与段数关系

场景棵数与段数关系例子

一端植树一端不植树棵数 = 段数2段 → 2棵树
两端都不植树棵数 = 段数 − 13段 → 2棵树

例1：两端都植树

路长200米，每隔10米种一棵树，两端都种：200÷10=20段 → 20+1=21棵。

例2：反向应用

每隔5米种一棵树，两端都种，正好种了10棵：段数 = 10-1 = 9段 → 路长 = 9×5 = 45米。

例3：方阵游行问题

10个方阵，每个长10米，间隔5米：10个方阵 → 9个间隔 → 总长 = 9×5 + 10×10 = 145米。

12点钟敲12下：12棵树 → 11个间隔 → 11秒。

玲玲家住9层，每层1分钟：1楼不需要上 → 楼高数 = 9-1 = 8 → 8分钟。
常见错误：误以为6是3的2倍，用时也2倍 = 12分钟（错的！应按楼高算）。

例6：两端都不植树（电线杆）

100米公路，每隔10米架设一根，两端都不架设：100÷10=10段 → 10-1=9根。

例7：楼梯级数

李林家住4楼，1楼到3楼走18级：1楼到3楼=2个楼高 → 1个楼高=9级 → 1楼到4楼=3个楼高 → 27级。

例8：电线杆问题（间隔与电线杆数）

第1根到第10根用9分钟，22分钟走到第几根？9个间隔用9分钟 → 1个间隔=1分钟 → 22分钟走22个间隔 → 22+1=23根。

上楼问题：1楼是起点，不用上，"楼层数 - 1"才是楼高数 · 敲钟问题：敲N下有N-1个间隔（不是N个） · 方阵问题：总长度要包含间隔长度+方阵自身长度 · 反向应用（已知棵数求长度）：先求段数（棵数-1），再求长度。

例1：锯木头问题（两端都不植树）

彩带长25分米，剪成5分米一段：25÷5 = 5段。剪5段需剪4次（次数 = 段数 − 1）。

例2：坑题辨析

李师傅把一根水管锯成3段用6分钟，他一口气锯了5根，共用多少分钟？

坑点：找"锯1次几分钟"再乘以5根的次数。

正解：每根锯3段都用6分钟（与单次时间无关） → 5×6 = 30分钟。

例3：一端植树一端不植树

街心花园一端原有1株桃树，每隔10米种海棠，共用23棵海棠：23棵海棠=23段 → 23×10 = 230米。

例4：环形植树（棵数等于段数）

人工湖周长150米，每隔3米种一棵：150÷3=50段 → 50棵。

水池周长400米栽80棵树，相邻距离 = 400÷80 = 5米。

全程60分钟，每隔5分钟一辆车。最少需要多少辆车？

L3·10行程初步（速度·时间·路程）
行程入门 三大公式
🎯 一句话心法：行程问题不是公式，而是基本概念 + 画图理解。必须先弄清速度、时间、路程的本质。

📐 行程三大基本概念

概念定义
路程物体实际经过的路径长度
速度单位时间内走的路程
时间走完路程所用的时长
三大基本关系式： 速度 × 时间 = 路程 路程 ÷ 速度 = 时间 路程 ÷ 时间 = 速度
📚 概念搭桥：距离 vs 路程
  • 距离：两点之间的线段长度（直线长度）
  • 路程：两点之间走过的轨迹长度（可走曲线）
  • 通常：距离 ≤ 路程
📚 速度的建构（重点）：比较两人快慢需统一一个量。
  • 时间相同 → 看路程（走得多者快）
  • 路程相同 → 看时间（用得少者快）
  • 都不一样 → 必须统一一个量
速度定义：单位时间内（1小时、1分钟、1秒）走的路程。

📝 例题

例1：相对位置问题（距离的多种可能）

小米家距学校1200米，学校距图书馆800米。小米家距图书馆多少米？

图书馆可能在学校的两边：

  • 情形1：图书馆在学校另一侧 → 1200+800 = 2000米
  • 情形2：图书馆在学校和小米家之间 → 1200−800 = 400米

核心思维：位置不确定时，分类讨论所有可能。

例2：画图与演示工具

小可家距学校1000米，走了250米忘戴口罩，原路返回拿。一共走了多少米？

工具：画图（画出往返轨迹）+ 演示（教具模拟走动）。

计算：250 + 250 + 1000 = 1500米。

例3-5：三大公式灵活应用

例3（求时间）：孙悟空500米，每分钟10米 → 500÷10 = 50分钟。

例4（求速度）：倒霉熊走600米用5分钟 → 600÷5 = 120米/分钟。

例5（速度×时间=路程）：7:30出发，10:30到达（3小时），每小时行8000米 → 8000×3 = 24000米 = 24千米。

例6：往返求时间

去时6000米/小时，回时6000米/小时，24千米需几小时？

24÷6 = 4小时。

例7：两段路（家→早餐店→学校）

沙和尚先步行10分钟（40米/分钟）到早餐店，再坐30分钟公交（1000米/分钟）。家到学校多远？

第一段：40×10=400米；第二段：1000×30=3000米 → 总 = 3400米。

例8：变速问题

小娜扎每分钟走60米，18分钟到校。今天前3分钟走50米/分钟，要想10分钟到校，后面每分钟走多少？

总路程：60×18=1080米；前3分钟走的：50×3=150米；剩余1080−150=930米；剩余7分钟。

L3·11等量代换（天平原则）
方程思维 曹冲称象
🎯 一句话心法：等量代换就是 天平原则 —— 等式两边同时加、同时减、同时乘（除零外）、同时除，依然平衡。
📚 概念搭桥：曹冲称象：用"船下沉到同一位置"的等量关系，把"称大象"转化成"称石头"。这就是等量代换的雏形。

📐 等量代换三大工具

  1. 等式基本性质：同加同减同乘同除不变。
  2. 直接代入法：已知关系代入未知项。
  3. 整体代换法：把某部分看作一个整体进行替换。

📝 例题

例1：天平问题·冰糖葫芦法

积木每块50克，3苹果=9积木，3桃子=6积木。一只桃比一只苹果轻多少克？

  • 3苹果=9积木 → 1苹果=9÷3=3积木
  • 3桃子=6积木 → 1桃子=6÷3=2积木
  • 1苹果比1桃子多1积木 → 1积木=50克
  • 答案：桃子比苹果轻50克

冰糖葫芦法：用竹签从上往下"穿"积木，能穿几个就是几块积木（用于图形叠加的天平问题）。

例2：进阶等量代换

"1桃=2包+40克"，"1桃+80克=4包+80克"。求1桃+4包共重多少克？

代入第二式：2包+40克+80克=4包+80克 → 2包=120克 → 1包=60克。

1桃+4包 = (2×60+40) + 4×60 = 400克。

例3-4：基础等量代换

例3：圆+圆+圆=18 → 圆=6；三角+圆=10 → 三角=4。
例4：圆=3三角，圆+三角=24 → 三角=6 → 圆=18。

例5：空瓶换水（3空瓶换1汽水）

核心思维：3空瓶=1汽水（含瓶）→ 等式两边都"去掉"1空瓶 → 2空瓶=1汽水里的"水"。

28瓶汽水 → 28空瓶 → 28÷2 = 14份水 → 总共喝的 = 28+14 = 42瓶。

例6-7：空瓶换水变式

例6（4空瓶换1汽水）：4空瓶=1汽水 → 3空瓶=1汽水里的水。18空瓶÷3=6份水 → 24瓶。

例7（7空瓶换1汽水·反向）：7人喝→买6瓶。2009人÷7=287组（287×6=1722瓶）→ 买1722瓶。

🔑 空瓶换水的本质：N空瓶换1汽水 → (N-1)空瓶 = 1汽水里的水。
不是"借来换去"或"买N送M"的技巧。本质就是等量代换。
空瓶换水不要死记"借来换去"（本质是等量代换） · 等式两边同加同减时要保持平衡 · 不同符号要严格区分 · 大数除法必须列竖式（即使三年级也要）。
等量代换是五六年级方程的基础，必须扎实掌握。
L3·12加减数字谜
数字推理 四大方法
🎯 一句话心法：数字谜的四大方法：个位分析、特殊结构、还原思想、按位交换。从最确定的位（个位）开始，逐步推到不定的位。

📐 数字谜四大分析方法

  1. 个位分析法：从个位入手，逐步考虑进位。
  2. 特殊结构法：识别特殊数字结构（如三位减两位=一位必是100-99=1）。
  3. 还原思想：减法是加法的逆运算，把减法还原成加法。
  4. 加法交换律（按位交换）：交换方框和已知数字的位置。

📝 例题

例1：个位分析法（入门）

三角+方块=85，三角+2=6。

  • 个位分析：4+2=6 → 三角=4。
  • 4+方块=85 → 方块=81。

进位专题：8+4=12（进位）→ 个位是2，注意进位1 → 9+谁=15？6，所以星=6。

例2：还原思想（减法转加法）

□−1=7 → □=8（直接算）。或"遇到减法可以还原成加法"来做。

还原口诀：遇到"加"就减，遇到"减"就加，遇到"乘"就除，遇到"除"就乘。

链式运算例：一个数先+6，再×6，再−6=30。从结果往前推：30+6=36；36÷6=6；6−6=0。

例3-5：特殊结构法

例3：三位数减两位数=1 → 特殊结构必是 100 − 99 = 1。

例4：两位数+两位数=196 → 98+98=196 ✓；97+99=196 ✓。两组。

例5：方框+方框=198 → 99+99=198 → 一组。

例6-7：加法交换律（位置互换）

例6：方框+2=8（个位），十位+3=10（进位），百位+4=7。

把方框和3交换位置 → 436+方框=708 → 方框=272。

例7：550+方框=746 → 交换5 → 方框=196。

核心思想：直接分析困难时，把方框与已知数字交换位置，用加减关系求出。

例8：汉字数字谜（爱数学）

核心方法：两个相同数相加，个位数字是6，那么这个数是8（8+8=16）或3（3+3=6）。

例9：图形相加（特殊结构）

三角+三角+三角=888。

方法1（特殊结构法）：888÷3 = 296。

方法2（个位分析法）：三角×3=个位8 → 三角=6（6×3=18）。综合 → 三角=296。

两种方法都要掌握。

进位不能漏（每位相加后都要考虑） · 借位要追踪（减法中被借走的1要传递） · 特殊结构识别要快 · 数字谜中相同字符代表相同数字。
三种方法（个位·特殊·还原·按位交换）—— 关键看题目特征灵活选用。

数字谜进阶 符号约束
🎯 一句话心法：符号谜和数字谜本质一样，只是把"方框"换成了字母/汉字/图形，相同符号代表相同数字，不同符号代表不同数字。

📝 4种常用方法

  1. 等式基本性质法：等式两边同加同减同乘同除（只有加减时最直接）。
  2. 乘法意义 + 画图：把"几×几"理解为"几个几相加"。
  3. 尾数分析法：从个位入手分析进位。
  4. 特殊结构分析法：识别位数极端值（如三位+三位=1998 → 999+999）。

📝 例题

例1：等式基本性质法

★★ − 8 = 14 → 两边都加8 → ★★=22 → ★=11。

例2：乘法意义 + 画图

18×○ − 16×○ = 48。

思路1（乘法意义）：18个○ − 16个○ = 2个○ = 48 → ○=24。

思路2（画图）：用18段表示○的18倍，16段表示16倍，差=2段=48，每段=24。

强调：乘法分配律不是必须的，三年级还没学，可以用"几个几"理解。

例3：乘除法尾数排查（系统顺序）

两个数相乘个位是2：

系统排查顺序（从小到大）： 1×2 ✓，2×1（重复） 2×6 ✓，6×2（重复） 3×4 ✓，4×3（重复） 4×8 ✓，8×4（重复） 6×7 ✓，7×6（重复） 8×9 ✓，9×8（重复） 共 6 种：1×2、2×6、3×4、4×8、6×7、8×9

心法：从小到大按顺序排查，不重复、不遗漏。

例4：特殊结构分析法

三位数 + 三位数 = 1998 → 只能是 999 + 999 = 1998。

方框 − 两位数 = 1 → 只有 100 − 99 = 1。

例5：经纬分析法（位数范围）

两位数 + 两位数 = ? → 两位数最大99，所以和最多99+99=198 → 结果只能是100-198。

例6（符号谜综合）：喜欢数学

求"喜、欢、数、学"代表的数字。

学×3=8（个位）→ 学=6（6×3=18）→ 进位1 数×3+1=0（个位）→ 数×3=9 → 数=3 → 进位1 欢×3+1=0 → 欢=9（9+1=10） 喜×3+1=? → 喜=1 答案：喜=1，欢=9，数=3，学=6
例7（恭贺新喜）：多重进位追踪

求"恭、贺、新、喜"代表的数字。

关键突破：贺这一步有两种可能（4或9），需结合上下文判断。

9×2=18+2=20（个位0，进位2），但最高位不能是0 → 贺不能是9。

4×2=8+2=10（个位0，进位1） → 贺=4 ✓。

答案：恭=1，贺=4，新=6，喜=5。

进位追踪：每步问"是否进位？进几？" · 最高位约束：结果最高位不能为0 · 不同符号约束：不同符号必代表不同数字 · 讨论要全：多种可能时都要列出再逐一排除。
数字谜和符号谜本质一样，关键是掌握分析方法。

📋 关键要点回顾（贯穿全课）

1️⃣ 算力方法（计算又快又准的核心）

  • 和不变：加法交换律、结合律、按位交换、来回给、借来还去
  • 差不变：同加同减、连续减去两数之和
  • 同级运算：数字带符号一起搬家（生活场景化：挣/花）

2️⃣ 计数方法（避免重漏的利器）

  • 分类：按某个标志把情况分成不重叠的几类
  • 有序无序：先A再B = 先B再A 吗？
  • 分类枚举：分类+类内枚举+求和
  • 最小-最大过电影法：连续范围的简化

3️⃣ 几何基础

  • 两大公理：两点之间线段最短；点到直线垂线段最短
  • 三种比较方法：一端对齐、同加同减、量小段

4️⃣ 行程基础

  • 三大概念：速度、时间、路程
  • 三个公式：速度 = 路程÷时间（基础公式）

5️⃣ 等量代换

  • 本质：等式两边同加同减同乘同除，等式仍成立
  • 空瓶换水规律：N空瓶=1汽水 → (N−1)空瓶=1汽水里的水

6️⃣ 数字谜/符号谜

  • 三大方法：特殊结构法、尾数分析法、进位分析法
  • 两个约定（符号谜）：不同符号不同数，相同符号相同数

💡 给主人的辅导建议（贯穿13讲）

  1. 重视思想方法：本套课程的核心是"思想方法 > 答案技巧"，务必引导孩子想"为什么这样做"，而不是背"怎么做"。
  2. 画图能力培养：从三年级开始培养"画简单图"的习惯，是为六年级复杂题打基础。
  3. 生活化迁移：所有问题都源自生活，老师强调"学完运筹思想要应用到生活中"。
  4. 方法不固化：没有"哪个方法一定简单"，要根据题目灵活选择。
  5. 不急于乘法原理：先把枚举学扎实，乘法原理是水到渠成的事。
