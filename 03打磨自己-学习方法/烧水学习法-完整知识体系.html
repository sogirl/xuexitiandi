1-1 烧水思维：从学渣到北大博士后的终极奥秘

总纲奠基
💬 核心命题：学习的本质目的是进步。不进步就等于没学习。进步必须是客观的、可衡量的、可以牢牢把握住的。

📖 个人逆袭故事

主讲人何俊高中四百多分（满分150的数学只考七十多分），在重点高中与满分高手差距一百多分。前两年尝试各种方法未果，反复追问"这一天的学习究竟是为了啥？"最终想通：学习的唯一目的是进步。基于此，发明了进步本技术。

最终结果：市第一名考入北大，拿到北大新生奖学金。

📒 进步本技术（核心框架）

进步本包含三个核心环节，循环运转：

环节动作关键常见失败点

🔥 烧水思维 — 学习必须一口气烧开

烧水的比喻：一壶水烧到60-70度拔掉电源 → 降温；再接上烧到70-80度又拔掉 → 又降温。这样无限循环，永远烧不开一壶水。

99.9999%的人学习失败的原因 — 持续性不行。学了忘、忘了学、学了再忘、忘了再学。

学习不直观：烧水可以通过声音或指示灯判断是否烧开，但学习过程看不见刻度、看不见结果。

🔥 威尔·史密斯："99% is the same as 0. 学习这壶水烧到99度都不行，必须一口气烧开。"

核心启示：

  1. 持续性压倒一切 — 学习绝不能中断
  2. 99% = 0 — 半途而废等于没开始
  3. 需要工具衡量进度 — 如进步本
  4. 一次烧开一壶水 — 聚焦目标，彻底掌握一门知识

1-2 对标思维：3位清北学霸逆袭背后的思维心法

明确方向
💬 核心公式：对标 = 模仿榜样 + 抠细节 + 重复到极致

🎯 对标的本质

"对标"就是深度模仿榜样。具体操作：

  1. 找到你的榜样
  2. 详细研究你的榜样
  3. 关键：必须清楚别人做成一件事情过程中的每一个环节、每一个细节，凡是能弄清楚的环节和细节，都要直接去模仿操作。这是99%的人都做不到的。

你想考清北、想考高分，但不去研究顶级高手是怎么做的？— 这就是最大的浪费。

📊 对标的三个层次

层次名称时间跨度效果核心特征
1精准对标短时间暴力突破瓶颈抓住关键一点彻底执行
2深度对标更长时间、更全面人生翻盘全面覆盖、持久坚持
3极致对标极致投入创造人生奇迹极致标准、狠劲到位

案例二：深度对标 — 袁佩佩二战考研翻盘北大

案例三：极致对标 — 张玲玲3天极限冲刺清华大学

做法具体操作

1-3 极致思维：如何给大脑编写快速进步的秘诀

核心算法
💬 核心命题：你已经会了一个东西，你只需要知道你是怎么会的，然后重复它。

🧠 焦虑的根源与解药

所有学习焦虑的根源：目标和现实有差距。解决方案分三个层次：

  1. 对标伟人 → 消除焦虑
  2. 对标自己 → 进入深度学习
  3. 深度思维 → 发现学习的核心算法

对标伟人消除焦虑

对标对象功夫对我们的启示
《中庸》人一己百，人十己千用数量弥补差距
《三国志》书读百遍重复是最基本的要求
苏东坡80万字《汉书》一字不差背诵极致的基础功夫
郑板桥每读书必千万遍要么不读，要读就读透
理查德·布兰森持续记录笔记好记性不如烂笔头

对标伟人后—还焦虑吗？不焦虑了。因为我们下的功夫跟人家相比，简直是沧海一粟。

💡 一切焦虑都源于不扎实。古人说：抓铁留痕，踏石留印。正确的事情一遍一遍扎扎实实地做，就能解决一切焦虑。

对标自己进入深度学习

遇到烦躁时，回忆你刚刚掌握的那个东西，弄清楚你是怎么学会它的，然后用同样的方法去学下一个。这个方法"百试百灵"。

原理：你已经具备学会一个东西的能力（既然能学会一个单词/公式，就具备了超级学习力的潜质）。你需要做的只是回忆并复制这个过程。核心机制不是能力问题，而是方法失忆：你忘记了自己本来就会学。

⚙️ 超级学习力的核心算法（三个环节）

① 学会一个东西（证明你有学习能力）
     ↓
② 反思如何学会的（挖掘自己的方法论）
     ↓
③ 回顾不忘 + 以此类推（复制成功经验）

为什么99.99%的人被卡在第二步？

  • 只关注"学什么"，不关注"怎么学"
  • 从不停下来反思自己成功的学习过程
  • 不知道自己本身就拥有超级学习力的钥匙

核心认知转变

认知转变从 → 到
对焦虑焦虑是能力问题 → 焦虑是方法问题
对学习学习复杂烦琐 → 学习只有三个核心环节
对自己我学不好 → 我已具备超级学习力潜质

2-1 学习：如何科学制定学习方案，获取进步捷径

学习的本质
💬 核心定义：学 = 模仿，习 = 反复刻意练习。学习的本质 = 基于模仿对象的反复刻意练习。

为什么绝大多数人学不好？

90%以上的人学了十年英语讲不出十句标准英语。根本原因：不懂学习的本质是什么。

孔子两千多年前在"学而时习之，不亦乐乎"中揭示：学和习是两回事。学是模仿，习是练习。按照学习的本质去学习，会产生学习乐趣。

要素含义常见错误
学（模仿）找到正确的模仿对象，精准模仿不知道模仿谁，模仿不到位
习（练习）反复的刻意练习只是机械重复，不是刻意练习
关键环节知道如何正确模仿和练习以为看懂了就结束了
结果产生了学习乐趣（不亦乐乎）越学越痛苦

案例：英语极致模仿

一天背10句标准句子，10天就是100句。100句可排列组合出成千上万篇文章。作者背了20多篇文章后考研英语第一名，北大读研期间写作课接近满分。

林语堂："学英语的唯一正轨，不出仿效与熟诵。"

背诵的本质：在大脑里建造高水平的精准语料库。

案例：理科极致模仿

考研统计物理95分（差5分满分）。方法：每个公式独立推导→与书上一模一样一字不差；几套真题百分之百研究透；解题步骤极其清晰标准。

案例：文科模仿标准答案

三步法：看标准答案 → 默写 → 对比差异，抠细节。

进阶：抄题 = 研究出题人的思路。掌握出题人的知识网络→做一道题顶别人做100道甚至1000道题。

案例：体育对标到极致

作者在国家皮划艇队：选男子前三名（比女子冠军快10%以上）→ 100秒比赛录像切成2400帧图片 → 每张精确分析 → 大量刻意练习。结果：第一块金牌仅在"直线滑行"一项上突破就拿到了。

科学制定学习方案

1
确定模仿对象：填空题模仿出题人思路，论述题模仿标准答案，公式模仿课本推导，语言模仿标准句子
⬇
2
确定模仿到位的标准：背到只字不差，百分之百独立做对，能独立复述
⬇
3
反复刻意练习：练习中发现漏洞→回到第一步，循环直到把壶水烧开

2-2 知识：有效问答拆解知识，提高知识消化率

知识的本质
💬 核心定义：知识的本质 = 用来解决问题的。有问才有学，没有问就没有学。

学习知识的两大困境

困境表现根源
面对资料茫然不知从何下手没有提问意识
反复读但把握不住模模糊糊，说不清学了什么只有记忆，没有理解
根本问题视学习为"采集信息"而非"解决问题"

知识的本质链

知识 → 用来解决问题 → 知识 = 学问 → 学问 = 问学 → 有问才有学

99.9%的人掉进了学习知识的大坑 — 因为他们从来不提问。

惠勒（物理学家）：没有问题，没有答案。爱因斯坦：提出一个合适的问题就等于解决问题的一半。

高效掌握知识的方法 — 拆解成问答

①
提问：问自己"它究竟想要回答什么问题？"拆到一段话、一句话，问题越小越具体越好
⬇
②
找答案：答案能否浓缩成一句话？新概念与已知概念有什么联系？
⬇
③
系统排列：去掉细枝末节，留下主线问题，按逻辑顺序排列

循环往复，直到豁然开朗。

案例：《童年的秘密》问答式拆解

作者把这本书拆解成一百多个问答，重复读了两百多遍，终于豁然贯通，发现全书核心问题："人格是如何形成的"。

问答学习法的四大优势

优势说明
🎯 回归本质知识的本质就是解决问题，问答是唯一正确姿势
🔬 深度理解步步追问直到最底层，把知识真正"消化"
🎮 游戏化学习高频收获感 → 产生学习乐趣 → 上瘾
🚀 接近创造用创造知识的方法学习知识，达到最高效能

费曼："凡我不能创造的，我就不能理解。"

本质一：社会选拔机制

考试不是某个人拍脑袋想出来的，是社会发展史创造的最公平的人才选拔机制（公元900年前后隋唐科举制度）。好的学校（北大清华）名额有限，凭考试成绩分配。心态转变：考试是压力 → 考试是机会

本质二：精准反馈

考试分数给学习状况提供精准精确的反馈，指出明确的进步方向。分数不是判决书，是指南针。分数指出：知识层面、技能层面、身体素质层面、心理素质层面的具体问题。

本质三：测试效应（Testing Effect）

心理学验证：常规被动学习正确率45%，通过测试进行学习正确率67%（提升接近50%）。

神经科学原理：记忆的本质是神经元产生新链接。产生变化只有两种途径：提高刺激强度（考试丢分让人"难受"）+ 提高刺激频率（高频测试反复刺激神经元）。

学霸的特质：高频回测、高频测试、高频考自己。

普通人高手（学霸/学神）
考一次试不回顾随时随地测试自己
漏洞积累越来越多发现一个漏洞补一个
心烦意乱随时巩固后方，随时前进
学习这壶水经常断电让学习这壶水一直在烧

考前必记的三句话

  1. 考试是最公平的资源分配方式 → 珍惜
  2. 考试分数是最精准的进步指南针 → 分析

对抗遗忘的实操方案 — 高频回测

速成原理一：学到一个东西 = 学到成千上万个东西

任何一个知识你都会成千上万次用到它。学得扎实的人一次学精准，后续每次使用都顺畅通过；学得含糊的人每次遇到都卡壳。如果在很小的知识点上不愿意下狠功夫 → 你就没法速成。

速成原理二：知识的组合爆炸特性

学习的核心：构造知识的基因。基因=最基本的概念、问题、定理、定义、原理、思维逻辑。把基因搞牢固→后面就可以组合爆炸。

速成的秘诀 — 能积微者速成（荀子）

"能积微" = 学到一点一滴的知识，要能积累住，别再忘了。用持续烧水的方式，直到知识稳定、清晰。

速成公式：扎实学一点 → 巩固住 → 再学下一点 → 再巩固 → 滚动积累 → 速成

目标粉末化 — 从痛苦到上瘾的秘诀

托尔斯泰："要有生活目标，有一辈子的目标、一个阶段的目标……乃至一分钟的目标。"

为什么是"粉末"而不是"颗粒"？食物磨成粉容易消化吸收。目标也一样：打得越碎越容易执行，越容易超额完成→产生超额成就感→养成习惯。

林语堂学英语的秘诀：不是一目十行，是每天精选两三句，回环练习。学之前把以往的内容全部巩固住，再推进。

大目标（如英语国际一流水平）
    ↓

    ↓
上瘾（欲罢不能）

《上瘾》（Hooked）的核心机制：上瘾的关键 = 有没有收获。没有明确收获→不可能全身心投入。频繁感受到收获→上瘾。

  1. 设定SMART目标
  2. 划定极小范围，目标粉末化（如一个公式独立推导出来→10-20分钟搞定）

❓ 终极一问（决定学习成败的问题）

"你究竟学到了什么？" — 永远比"你学了什么"重要。

学了1小时说不清学到了什么 → 这1小时就是慢性自杀。

普通人学霸/学神

心态自我欺骗、走形式较真、较劲

契克森米哈伊的心流理论：人们在行动中产生愉悦的共同模式 — 全力以赴为一个明确的目标努力。

从学习痛苦到心流的转变路径：学习痛苦（目标太大、没有反馈）→ 目标粉末化（设定极小的可完成目标）→ 频繁收获感（每几分钟就能检验成果）→ 学习上瘾（欲罢不能）→ 心流（全力以赴、浑然忘我）

SMART目标 + 粉末化 + 频繁问"学到了什么" = 学习上瘾 = 心流

终极技术 ⭐
💬 核心命题：进步本 = 康奈尔笔记法的极致升级。进步本的本质不是笔记本，而是进步的回路。

康奈尔笔记法 — 世界上最有名的笔记技术

一页笔记分三栏：主栏（记录）、护栏（提炼归纳）、思考栏（反思）。

五步思维操作：①记录→②简化→③背诵→④思考→⑤复习。一个学习内容要折腾5遍。

进步本 = 康奈尔笔记法的极致升级

环节康奈尔笔记法进步本（极致版）
记录记尽可能丰富的信息把当天所有进步都记进去：做错的+改过来的、不会的+弄懂的、新知识、新感悟……一个都不放过
复习每周约10分钟随时随地检测式复习—感觉模糊了立即回去复习，做过的题目重做，重要内容重复默写
检测强调较少专门的检测环节—通过做模拟题、真题发现真实水平，寻找漏洞

进步本三个环节的详细要求

① 记录环节：错题不能直接把正确答案写在空上（否则后面没法考自己）。正确答案和题目要分离开。要分析所有选项的正误—不只是正确选项，错误选项也要分析为什么错。要分析出题人的意图—抄题，在抄题中研究出题思路。能做独立出一道类似的题 = 真正掌握了。

② 复习环节：随时随地的检测式复习。感到模糊了→立即回去复习。题目会做了还不够→要熟练→要高度熟练。知识要在大脑里反复呈现→以清晰为标准。

③ 检测环节：因为我们的感觉会造假，最擅长的就是自我欺骗。必须通过真题、模拟题去检测真实水平。

为什么必须追求极致？

考试分两类：

  • 资格考试（驾照、司法考试）— 达到分数线即可
  • 竞争性考试（高考、考研、面试）— 必须奔着极端去，上不封顶，不断趋近满分

实操案例

反例：把正确答案直接填在题目空上→没法考自己；只分析正确选项→浪费知识网络；不抄题→错过深度理解。

正面案例：一位高中生把非洲地图画得五彩斑斓—不是在记笔记，而是在创造艺术品。杨柳青的进步本从图很简单→后来越画越细致，标注密密麻麻。

杨柳青："当我落笔的刹那，我感觉到它就像是我生命里的一件艺术品，仿佛成了活物，有了自己的思想和生命。纵然没有任何人欣赏，我也愿意一点一点地去打磨，去雕刻，直到我能够达到的极致。"

结果：复试第一名，以绝对优势被录取。

🍅 锦囊一：五分钟对抗拖延

拖延的本质 = 恐惧（恐惧失败 + 恐惧成功）。恐惧的根源 = 无知。

大多数人有一个经历：刚开始学一个东西时付出回报率很高，过一段时间不知不觉陷入低谷（付出回报率下降）。如果越过低谷，就会迎来指数增长。

最典型的例子 — 背单词表：英文单词表第一个词往往是"abandon"（放弃）。为什么这个词记得最熟？因为它第一个出现，你重复的次数最多。一旦发现这个秘密—任何单词你都有办法让它彻底记住。再难的单词重复到一定次数就能记住。

核心技巧：微精通。把知识打成粉末，细到不能再细，轻松掌握一个小知识点到90-100%。对知识的感觉立刻亲切→恐惧感消除→成就感产生。

具体操作：只做五分钟。不要想"把这件事做完美"。先做5分钟，只做一小点。做5分钟后，你可能不知不觉做了7分钟、8分钟、10分钟。又为难了？→再坚持5分钟。受不了了？→过一会儿再来一组。

💡 对抗拖延公式

拖延（恐惧+无知）→ 知识粉末化（微精通）→ 只做5分钟 → 进入状态

⛰️ 锦囊二：五分钟穿越学习低谷

学习低谷的三个阶段：初始期（回报率高）→ 低谷期（付出回报率降低）→ 爆发期（高速增长）

进入低谷的原因：太聪明（总以为一口气能学很多东西）、太傲慢（以为学到一点很容易）、太盲目（羡慕学霸，无视大师的笨功夫）。

大师的笨功夫：费曼读到哪卡住→从头开始读；杨振宁读数学书"读不过第一句"，卡住了只能一点一滴补；《中庸》人一能之己百之；《道德经》少则得，多则惑；荀子能积微者速成。

📝 锦囊三：五分钟消除低级错误 — 三步法

一个残酷的事实：所有人都会犯低级错误，包括清北学生。但水平越高，低级错误越少。

低级错误的根本原因：一心不得二用。大脑想解题思路时，照顾不到笔下内容的精准性。想的是对的，写出来是错的。低级错误绝不是偶然—它是一心二用带来的必然现象。

1
演算纸和答题纸一样工整 — 99%的人做不到。演算纸转着圈写，甚至作者本人都看不清。一旦做到，威力立现。
⬇
2
不要省略任何步骤 — 你以为某个步骤太简单可以省略→恰好是那个出问题。多写一行字看似浪费时间→实际是节省精力。
⬇
3
迭代回测 — 每计算两三步就回测一遍。检视完毕→再继续往下写。如此滚动迭代前进。

这样做有两个意义：再次确认思路正确；如果检查出错误—赚大了（一个小的错漏可能葬送十几分）。

💡 本质

态度好 ≠ 不犯错。操作程序正确 = 稳定产生好结果。

任何粗心，本质上还是不屑于把微小的东西做到位。一个人可以有无数理由不做简单的事情。

作用力与反作用力：你对微小环节不屑，它们对你也不客气。
