第一章 · 课程总览

📌 一句话定位

副标题为「学习与升学的完全规划」，定位为体制内升学（中高考）路线设计的体系化课程。

🏗️ 核心架构——金字塔结构

两条主线 + 一条副线

  • 主线一 · 年级主线：6–18 岁，每年一个视频，覆盖从 10 岁到 18 岁的整体路径。
  • 主线二 · 科目主线：语数英物等主课单独成片，形成「年级篇」与「科目篇」的双重视角强化。
  • 副线 · 小 tips：奥数要不要学、早读怎么读、晚上几点睡等生活细节问题。
🎯 核心输出
  • 每一年该干什么、不该干什么
  • 具体周末报课方案
  • 具体科目学习方案
  • 附讲师 1v1 顾问咨询（永久观看 + 12 年陪伴到高考）
💡 关键设计理念
  • 「学习规划 + 升学规划」合一——只讲升学路径是误区，成绩一定来自升学之前的积累。
  • 「保姆式方案」+「讲清楚为什么是这样」——家长根据自家情况决策，不是被命令。
  • 三层价值：效率层（省时省钱省心）、情绪层（防止焦虑）、执行层（避坑 + 顾问陪伴）。
💎 核心金句：年级越低看规划课越划算；四五年级才买的家长，建议不要回头看一二三年级规划（看完会悔恨）。

第二章 · 年级篇（6-18岁全时间线）

本章是课程的主线骨架——从幼升小到高中三年，每一年的「该做什么」「不该做什么」「节奏怎么走」。

年级依赖链路： 幼升小 → 一年级 → 二年级 → 三年级 → 四/五年级 → 小升初 → 初中 → 高中

📊 年级篇完整时间线

🎒 幼升小（5-6岁）

一句话定位：「幼小衔接」不是抢跑，而是习惯养成与心理准备；这一年的核心是「让孩子愿意上学」而不是「让孩子学完一年级」。

✅ 学科知识底线

  • 拼音启蒙
  • 10 以内加减法
  • 识字 200–300 字
  • 会写自己名字
⚠️ 注意：以上是「底线」，不是目标——不要拔高到「学完一年级」。

✅ 习惯养成（比知识更重要）

  • 坐得住（30 分钟）
  • 听指令
  • 收拾书包
  • 按时睡觉

✅ 心理建设

  • 提前带孩子参观目标小学
  • 模拟课堂体验
  • 不要用「上学」恐吓孩子
🚫 避坑：不要在幼儿园阶段报「拼音班」「写字班」等提前学课程——会让孩子对学习产生抵触。

📗 一年级（6-7岁）

一句话定位：一年级的核心不是成绩，是「学习兴趣 + 学习习惯 + 学习能力」三件套的奠基。

✅ 三件事比分数重要

  1. 坐得住
  2. 会听课
  3. 能完成简单作业

📚 各科重点

科目重点方法
语文识字 + 写字（一类字+二类字）听写为主，拼音跟住课堂即可
数学计算速度 + 计算准确率每天10-15分钟口算打卡
英语培养语感看动画、听儿歌，不要背单词学语法
👪 家长陪读：一年级必须陪读，核心是「观察」而非「辅导」。

📗 二年级（7-8岁）

一句话定位：二年级是「分化年」——孩子之间开始拉开差距，习惯好的开始显现优势。

📚 各科重点

科目重点方法
语文继续抓识字，目标能独立阅读简单绘本开始写「完整句」而非「短语」
数学应用题读题能力重点训练读题，不要急着做
英语分级阅读启动RAZ、牛津树等，每天15-20分钟

👪 家长角色转变

  • 从「全程陪读」逐步过渡到「抽查式陪读」
  • 培养孩子自查能力
🚫 避坑：不要报「作文班」「奥数班」——太早，孩子还没准备好。

📕 三年级（8-9岁）——分水岭

一句话定位：三年级是「分水岭」——开始有正式考试，难度提升；同时也是「奥数」「英语PET」「作文」三大主线的启动年。

📚 各科重点

科目重点时间节点
语文二类字过关 + 作文起步 + 阅读理解初步训练一整年
数学奥数正式启动，课内保持95+三年级上启动
英语PET备考启动三年级上备考，三下或四上过
📋 三件大事：奥数 + 英语PET + 语文 = 三年级三大主线，不要加其他课外班。
👪 家长角色：从「陪读」转向「监督」；建立错题本、定期小测机制。
🚫 避坑：不要在三年级报太多课外班——三件大事已经够满。

📙 四年级 / 五年级（9-11岁）

一句话定位：四年级是「巩固期」，五年级是「冲刺期」——目标是「五年级结束时，孩子的成绩单/简历/能力三件套全部 ready」。

四年级核心任务

  • 🏆 奥数进入正式体系学习（目标：五年级拿到第一个有含金量的奖）
  • 📘 英语完成 PET，开始备考 FCE
  • 📝 语文阅读理解、作文进入系统训练

五年级核心任务

  • 📄 小升初简历开始准备（详见专题篇）
  • 🏅 课内成绩保持年级前列
  • 👨‍👩‍👧‍👦 「和睦的一家人」页面所需的家庭活动素材要积累
🚫 避坑：不要临时换赛道、不要因为「别人都在搞」就加项目。五年级下学期才开始准备简历——太晚。
📋 五年级结束前的检查清单
  • ✅ 奥数至少有一个有含金量的奖项
  • ✅ FCE考试通过
  • ✅ 课内成绩稳定年级前列
  • ✅ 简历初稿已完成
  • ✅ 初一部分内容已提前接触

📔 小升初（11-12岁）

一句话定位：小升初是「路线的分叉口」——选公办对口还是择校走民办/重点班，决定了接下来三年的培养方案。

路径选择

  • 择校：五升六寒假就要把简历/证书/面试全部准备好
  • 对口入学：也建议做简历（作为家庭成长档案）

学科准备要求

科目要求
数学奥数获奖是「敲门砖」；课内必须超前学完初一内容
英语FCE通过；能听口面试基本交流
语文阅读量大、作文稳定高分
📋 小升初时间表
  • 五升六暑假：复习初中内容（数学按中考中等难度学完七年级）
  • 六年级寒假：完成简历初稿
  • 六年级春季：面试准备 + 目标学校确定
  • 六年级暑假：按自招难度重学初一内容（不可直接往前学）
💡 家长心态：不要把「择校失败」当成世界末日——公办重点班同样能出人才。

🏫 初中三年（12-15岁）

一句话定位：初中是「赛道定型期」——三年决定高中档次，高中决定大学档次。

分年级节奏

年级核心任务注意事项
初一适应新学科（多出4-5门课）；数学和英语保持领先初一暑假预习物理
初二（分水岭）物理加入；数学难度陡升；开始出现「掉队」八升九暑假电学攻坚
初三中考冲刺；关注自招体系初三暑假预习化学

学科平衡策略

  • 语文：不补就掉（详见专题·语文抢救）
  • 数学：按自招体系走（详见专题·自招体系）
  • 英语：保持FCE水平，向高考水平冲刺
  • 物理：初二必须重视，三遍法学习
  • 化学：八升九暑假开始
📋 初中三年关键节点
  • 初一暑假：物理第一遍启蒙（通读课本，建立直觉)
  • 初二：物理第二遍（中等难度做题）
  • 八升九暑假：电学攻坚（必须学到高中难度）
  • 初三：物理第三遍（学校同步+方法课）
  • 初三暑假：化学预习 + 高一物理深度预习
🚫 避坑：不要在初一就报「中考冲刺班」——太早；不要在初三放弃自招机会。

🎓 高中三年（15-18岁）

一句话定位：高中是「终极冲刺期」——一切为了高考；高一打基础、高二定方向、高三拼冲刺。

分年级节奏

年级核心任务关键动作
高一9门课齐头并进，重点「语数英+选科三科」竞赛/自招布局；暑假深度预习物理化学
高二确定选科（3+1+2或3+3）；一轮复习早期准备选科确定后不可更改；竞赛冲刺
高三全面冲刺；一模二模三模节奏心态调整 > 刷题量；志愿填报准备

选科策略

  • 不要只看分数，要看「目标专业 + 大学选科要求」
  • 物化双选已成趋势（好专业都要求物理化学）
  • 物理学不好 → 与好大学好专业无缘
📌 建议：家长买一本高考志愿填报小手册（二三十元），查看不同大学的选科要求。
🚫 避坑：高一就偏科；高二换选科（多数省份不允许）；高三临时换赛道。

第三章 · 科目篇（语数英物化生）

本章是课程的横向覆盖——六大主科，每一科的「路径」「节奏」「避坑点」。

学科依赖关系：数学 → 物理 → 化学；识字 → 阅读 → 作文；语感启蒙 → 自然拼读 → PET → FCE

📝 语文——最被忽视却最拉分的科目

一句话定位：语文靠的是长期积累，不是短期突击。它不是「玄学」，是方法问题。

📊 语文成绩的四大决定因素

板块内容学习难度
语文基础字词、拼音、成语、同反义词、近义词、时代背景基础且关键
古诗词与古文古代文体，传统文化，有专门考试板块现代人较难
阅读理解答题能否扣到点子上、能否少扣分多数人头疼
作文整体写作能力，分值占比大需系统训练
⚠️ 关键认知：四项相互影响——语文基础差影响阅读和作文；阅读差影响作文；古文影响写作；写作反过来影响古文。

📚 阅读能力的两大支柱

  1. 读懂文章的能力——通过精读课培养（三年级开始），逐字逐句做批注
  2. 答题模板与技巧——四五年级加入
💡 核心比喻——「练拳不练功，到头一场空」：套路（模板）= 拳法；读懂文章的能力 = 身体素质；只有套路没有身体素质 = 花拳绣腿。

✏️ 作文的「结构为王」理念

🏗️ 盖楼理论：结构=楼的框架；文采=房间的装修。框架搭好，装修不太差，分数就不会差。

作文训练时间线：一年级下学期看图写话 → 二年级继续 → 三年级正式作文训练

初中作文升级：记叙文→议论文；几段论套路；准备好自己的素材库

📜 小古文与古诗词

  • 小古文：三年级以上可以开始，为初中打基础，积累常见实词虚词
  • 古诗词：不需要提前背，学校学到哪就背到哪

📖 课外泛读的正确打开方式

  • 泛读是「副线」，不是「主线」
  • 两个黄金期：学前阶段 + 三四年级（好奇心爆发）
  • 泛读提升的是上限（从90分到95+），不是雪中送炭

🗺️ 语文完整规划时间线

学前
拼音 + 识字1200 + 控笔 + 多看图文书
1-2年级
只抓语文基础——字词听写是核心
3年级
语文基础 + 精读课 + 小古文
4-5年级
语文基础 + 精读 + 阅读技巧课
5-6年级
加入作文结构训练
初中
阅读技巧 + 作文模板 + 文言文逐字逐句

关键提醒

  • 一二年级不需要大量课外阅读——字词基础是第一位的
  • 「什么年级做什么事」——不要被"阅读很重要"的杂音带偏
  • 不需要晨读、拼命读课外书
  • 高中吃初中老本，无需特殊规划

🔢 数学——最需要超前学的科目

一句话定位：数学是「最需要超前学的科目」——校内进度太慢，校外必须抢跑。奥数是「数学能力的试金石」。

整体路径

核心路径：小学奥数 → 初中自招 → 高中校内紧跟

🧮 小学阶段：双线并行

线路定位方法
校内数学保持95+寒暑假预习 + 每天练习册
校外奥数思维训练 + 小升初硬通货录播课 + 方法课，三年级启动
⚠️ 校内与奥数本质上是两套体系，交集不多。一年级就要决定是否学奥数——推荐学。

🏆 初中阶段：自招体系

进入初中，停止奥数，转向「自招体系」（详见专题·自招体系）。自招介于中考和竞赛之间，包含中考体系且范围更广更深。

  • 数学必须按自招体系走
  • 自招包含部分高中知识，提前铺垫

🎓 高中阶段：校内紧跟

  • 除竞赛生外，不必碰校外高难度内容
  • 寒暑假一定要预习（高中数学太难，不预习跟不上）
📋 小升初过渡规划（垃圾时间的妙用）
  1. 六年级寒假+春季：按中考中等难度，把初中第一年（七年级）内容学一遍
  2. 暑假：回过头来，把初中第一学期按自招难度重学一遍
  3. 秋季开学后：一炮而红→好学生标签→良性循环
  4. 八升九：加快进度，八年级结束时基本学完九年级内容
注意：这是数学的专有要求，英语和语文没有这个要求。
📋 初升高过渡规划

初升高数学跨度极大，中考结束暑假千万不能玩——必须深度预习一个学期。

  • 预习要学得深（物理化学也同样需要）
  • 第一次考试决定信心→决定后续表现
  • 高中正轨后：寒暑假预习 + 学期中紧跟老师

避坑清单

  • 不要「只刷题不总结」
  • 不要「题海战术」
  • 不要因为校内成绩好就不学奥数

🔤 英语——最容易抢跑的科目

一句话定位：英语是「最容易抢跑的科目」——低投入高产出，必须提前学、超前学、超纲学。

🎯 路径时间表（关键！）

阶段时间核心任务
启蒙期幼儿园-1年级看动画、听儿歌，培养语感（半年）
入门期2年级自然拼读（1-2个月略懂即可）+ 分级阅读（RAZ/牛津树）
原版期1年级-6/7年级选一套原版体系课（Think/Unlock/Power Up）系统学习
PET阶段3年级上备考，3下-4上通过PET备考 + 考试
FCE阶段4下-5年级启动，6年级通过FCE备考 + 考试
应试期初中+吃老本 + 应试专项课（语法/翻译），向145+冲刺
💡 最核心原则：校内撑死中等分数，校外才是主战场。校外学得好→校内随便应付也能高分。

🔑 原版体系课的核心地位

  • 选一套完整的原版体系课（Think、Unlock、Power Up等）
  • 跟着体系不断往上走，学扎实、彻底掌握
  • 绝对不能东拼西凑——今天学自然拼读、明天学音标、后天报阅读班=毁了孩子
🚫 错误做法：报三套分级阅读（牛津树+RAZ+其他）=重复训练，水平原地踏步。同时上两个体系课（新概念+Think）=混乱低效。

🔄 补充策略

策略含义适用场景
以考代练报名KET/PET短期备考班体系课学4-5年，中间疲惫
大火猛攻寒暑假集中追赶进度学期中时间少，假期补上
灵活调整根据孩子实际情况调整规划不同孩子时间、基础不同

执行保障

  • 录播小课需要家长深度参与监督
  • 每天要求的学习内容必须落实、打卡
  • 规划+方法+执行=英语起飞
📋 避坑清单
  • ❌ 只紧盯校内→撑死中等分数
  • ❌ 蹦跳课作为长期学习方法→效果差
  • ❌ 自然拼读学两年→本末倒置（1-2个月即可）
  • ❌ 三套分级阅读→原地踏步
  • ❌ 两个体系课→混乱低效
  • ❌ 报了班不管→录播课特别需要监督

⚡ 物理——初中最关键的理科

一句话定位：物理是「初中最关键的理科」——初二开始加入，决定中考和高中理科基础。

🎯 物理「三遍法」学习路径

遍数时间内容难度
第一遍·启蒙六年级只理解、记忆例子，不做题补生活经验短板
第二遍·做题七年级（必做）踏实做中等/中考难度题领先学校
第三遍·同步八年级起学校同步 + 方法课拿高分八年级简单，九年级难

🔑 关键节点

🔥 八升九暑假 = 电学攻坚：电学必须学到高中难度（不是初中难度），才能做初中题飞快。
⚡ 九升高一暑假 = 物理命运：必须报班深度预习，决定高中第一次考试成绩（极其重要）。
📋 物理难学的两大原因
  1. 客观原因：物理研究世界运行规律；用数学方法学物理是错误的
  2. 主观原因：从小缺乏生活观察和自然科学积累
解决办法：从小培养对自然科学的兴趣（科技杂志、十万个为什么等），六年级启蒙补短板。

配套资源取舍

  • ❌ 专门物理实验课：不要报（撑不了一年，重点是规律应用）
  • ✅ 小学科学课：可报可不报
  • ✅ 科学杂志（《万物》《好奇号》等）：可搞可不搞

启蒙时机

🚫 铁律：不要三、四年级学物理——纯焦虑收割，最早物理启蒙=六年级。三四年级最多搞搞科学。

🏫 高中物理的极端重要性

  • 高考选科漏洞已补——好专业都要求物理
  • 物理有强筛选作用：大学用物理筛人
  • 物理学不好→与好大学好专业无缘

🧪 化学——初三才加入的科目

一句话定位：化学是「初三才加入的科目」——一年的学习时间，节奏紧，但比物理简单太多。

核心定位

  • 化学 = 理解 + 记忆
  • 内容比物理少得多
  • 不要像物理那样学三遍

🎯 时间线

阶段关键任务
八升九暑假正儿八经学一遍化学：该背的背会、该记的记好、该理解的理解好
九年级同步跟住课内，重点抓化学方程式和实验题
初升高暑假整个学完高一第一学期化学（高中一周=初中半学期内容量）
⚠️ 化学不要太早学！七八年级就开始学不推荐——理解和记忆分不开，太早学容易「学了个热闹」。

高中化学的两大支柱

  1. 记忆：知识点又细又碎，需要大量时间
  2. 理解：必须理解反应的实质，不能只靠记忆
缺一不可！

避坑清单

  • 不要死记硬背元素周期表
  • 不要忽视实验题
  • 不要等到初三下学期才准备

🧬 生物——易被忽视的翻盘科目

一句话定位：生物是「初中最容易被忽视却能在高中选科中翻盘的科目」——背诵量大，但拿分稳定。

学习建议

  • 初一或初二开始（看地区版本）
  • 课本为主，图示记忆
  • 建立「知识框架」而非孤立记忆
  • 高中选科包含生物→初三暑假要预习
🚫 避坑：不要把生物当成「副科」忽视；不要死背而不理解。

第四章 · 观念与认知（心法篇）

这是整套课程的「灵魂」——技术之外，决定成败的关键。共9篇核心观念，贯穿全周期。

🧠 观念篇知识图谱

⏳ ① 提前学到底是怎么回事

「提前学」≠「抢跑」——是用未来的视角学今天的内容，让孩子「站在高年级看低年级」。
科目提前学建议
英语✅ 必须提前学（最容易抢跑的科目）
数学✅ 适当提前，不超过1个年级
语文❌ 不需要「提前学」课本——核心是积累
物理/化学✅ 暑假预习是合理形式
判断标准：提前学的目的是「让孩子在高年级时更轻松」——如果越学越累，就错了。

🕐 ② 每日时间规划

「每天晚上4点到9点」是孩子学习的主战场——这5个小时的质量决定了一年的成绩。

典型时间表（小学高年级）

时间内容
15:30-16:30放学后自由活动（户外）
16:30-17:30写作业（在校完成更好）
17:30-18:30晚饭+休息
18:30-20:00课外班/重点学科（数学/英语）
20:00-21:00阅读/复习/错题整理
21:00洗漱睡觉（保证9-10小时睡眠）

关键原则

  • 保证睡眠——牺牲睡眠换学习=短期有效、长期崩盘
  • 每天至少1小时户外活动，对学习效率正向
  • 不要连续学习——每45分钟休息10分钟
  • 家长帮孩子「规划」而不是「监督」

🌴 ③ 寒暑假必须预习

「寒暑假」是弯道超车的唯一机会——平时大家节奏一样，寒假暑假是拉开差距的窗口。

典型寒暑假任务

阶段任务
小学寒假预习下学期数学 + 英语分级阅读 + 课外阅读5本
小学暑假预习下学期数学 + 奥数系统学习 + 英语FCE/CAE备考
初中寒假预习下学期物理/化学 + 数学拔高
初中暑假预习下学期物理/化学/数学 + 英语拔高
高中寒假高考一轮复习提前启动
高中暑假衔接下学期内容 + 竞赛/自招准备
注意：不要把寒暑假变成「第三学期」——休息+预习结合，不要全填满。

👪 ④ 低年级一定要陪读

「陪读」不是「辅导」——是「陪伴+观察+引导」，核心是「让孩子养成习惯」。

阶段性陪读策略

年级陪读方式重点
一年级全程陪坐得住、听指令、不拖拉
二年级半陪自查、时间管理
三年级起抽查式陪错题、总结
🚫 避坑：不要把陪读变成「代做」；不要变成「监督员」——孩子写作业时家长不要刷手机。

💰 ⑤ 省钱鸡娃的原则

「鸡娃」不等于「砸钱」——99%的优质教育资源是「免费+低成本」的。

核心原则

  • 能用录播课解决的，不要用直播课（直播课贵10倍）
  • 能用免费资源解决的，不要花钱（B站、YouTube等）
  • 能在家学的，不要去线下（线下机构溢价高）

哪些值得花钱

  • 一两次性的「诊断+规划」咨询
  • 关键节点的「冲刺班」（小升初简历指导、中考冲刺）

哪些不值得花钱

  • 日常辅导班（家长自己陪读即可）
  • 「同步班」「提高班」（与课内重复）
🚫 避坑：不要被「包过」「保过」忽悠；不要因为「贵=好」买单。

🏆 ⑥ 小学奥数为什么一定要尽量学

「奥数」不是「数学的延伸」——是「数学思维的体操」，不为了获奖也要学。

为什么一定要学

  1. 小升初简历的硬通货——奥数获奖是敲门砖
  2. 初高中数学的底层能力——学过奥数，初高中数学轻松30%
  3. 思维训练的不可替代性——拆解问题、多步推理，贯穿所有理科

怎么学

  • 三年级启动（详见三年级篇）
  • 不强求获奖——把「奥数七大模块」（数论、几何、应用题、组合、行程、计数、构造）系统学一遍就够
  • 录播课+刷题是主流路径
  • 线下班仅适合需要强监督的孩子
🚫 避坑：不要因为「不擅长」就放弃；不要因为「投入大」就放弃——奥数是性价比最高的课外班之一。

💻 ⑦ 编程可以学，信奥千万别搞

「编程」是兴趣+思维训练；「信奥」是极少数天才的游戏——99%的家庭不适合。

编程可以学

  • 小学低年级学Scratch，培养逻辑思维
  • 小学高年级学Python，做有趣的项目
  • 目的：兴趣+思维+简历亮点（非必填项）

信奥千万别搞的残酷真相

维度数据
成功率50（保送清北）÷ 50万（在搞信奥）= 1‱（万分之一）
政策变化从「省奖保送」→「全国前50才能保送」
投入产出信奥获奖概率<1%，投入时间是其他活动3倍
替代方案同等时间投入奥数，性价比高10倍

信奥产业链的收割套路

  1. 包装焦虑：宣传「信奥可以保送清北」
  2. 规划路径：设计从小学到高中的「清晰路径」
  3. 持续投入：从幼儿园开始报启蒙班
  4. 中途劝退：学不出来=你家孩子问题
  5. 结果兜底：课已收、钱已赚
💡 核心判断：竞赛不是「多一条路」，是「少一条路」。第50名保送清北，第51名没学上——实力差距极小，结果天壤之别。
🚫 铁律：除非孩子真有全国前50的天赋，否则不要把孩子的路走窄。
📋 什么情况可以例外

孩子表现出「对编程极度热情」+「数学成绩年级前列」时，可以谨慎尝试——但仍需清醒认知风险。

📌 ⑧ 传统意义的每日预习复习其实是伪命题

「每天预习+每天复习」是「看起来很努力其实效率很低」的反模式。

为什么是伪命题

  • 预习：如果孩子理解力够，预习是浪费时间；如果不够，预习也看不懂
  • 复习：每天复习=把当天内容再学一遍——不如周末集中+月度大复习

正确的节奏

  • 周末：集中做错题回顾+单元总结
  • 月末：做月度测试+调整下月计划
  • 寒暑假：大块时间做系统性预习和复习
🚫 避坑：不要把「每天预习复习」当成「勤奋的标志」——勤奋不等于有效。

📌 ⑨ 不要做很多无用的额外动作

「做了很多事」≠「学到了很多东西」——很多动作是「心理安慰」而非「有效投入」。

典型的无用动作

  • 每天抄课文、生字抄10遍——形式主义
  • 做完作业还要刷一套课外题——没时间总结，等于白做
  • 上太多课外班——疲于应付，没有消化时间

判断标准

一个动作如果「孩子做完后能说出学到了什么」就是有效的；说不出，就是无用的。

正确做法：「少而精」——每天只做1-2件「高密度」动作，胜过5件「低密度」动作。

第五章 · 专题篇（关键节点）

📄 加餐：小升初简历讲座

一句话定位：一份极致实操的「小升初简历制作手册」，把简历升华为「家庭规划与亲子协作的项目」。

简历形式硬规范

  • 装订方式：必须胶装（像论文/书一样）
  • 绝对避免夹子夹、铁丝环装——这些「很 low」
  • 模板可网上找，不要让打印店设计

七大内容板块

#板块内容要求
1封面积极向上的照片；标明谁的简历、来自哪个小学；投哪所就改对应学校
2第一页可放目录+自我介绍信
3自我介绍信让孩子自己写，强调「匹配」与「倾慕」，家长修改。不要把奖项写进来
4学业展示课内附成绩单（A4一页可放2-3张）；课外展示奥数课本/奖项、英语教材、作文比赛奖项
5活动展示校内：升旗手、大队长、小班干部；校外：公益、参观、活动
6荣誉展示奖状证书集中展示，可融入其他板块
7和睦的一家人写爸妈干什么的、如何教育你、家庭集体出游。学校非常在意家庭和谐度

体量控制：

理想20页，最多25页。

准备时间：

四年级或五年级就要开始准备——类比大学生求职，不能等到大四才写简历。

🎯 初中自招体系

一句话定位：「自招」是「中考之外的另一条路」——为「学科特长」的孩子准备的提前录取通道，是初中规划的核心方法论。

三条路径对比

路径难度范围适合人群
只学中考低小仅适合放弃升学的孩子
自招体系 ✅中（介于中考和奥数之间）大（中考+拓展+先修）多数孩子
初中奥数极高极大千里挑一（绝大多数人不行）
自招体系的三大作用：①校内提升（初中3年） ②高中衔接（高一开学即见效） ③升学选拔（初三自招考试拿offer）

学科策略

  • 数学：必须按自招完整体系走（加深+拓展+高中先修）
  • 物理：尽量拓展（凹面镜、温度计计算等）
  • 化学：尽量拓展，可学到接近竞赛程度
📋 可操作清单
  • 小学阶段：鼓励学奥数
  • 小升初衔接：理解「自招」概念
  • 初一：按自招体系学数学
  • 初二：深化自招体系，接触高中简单内容
  • 初三：自招体系+中考复习并行
  • 了解目标高中的自招/自主选拔政策
🚫 误区提醒：不要以为自招是少数天才的游戏——是多数孩子的最优路径。不要把自招当成「中考保底」。不要因为准备自招而忽视课内。

🏥 初中语文成绩差怎么抢救

一句话定位：「初中语文掉队」的根源在小学——抢救必须「回溯式补」，不能只盯着初中。

诊断

80%的「初中语文差」其实是「小学阅读量不够」+「作文没练过」。

抢救路径

  1. 立即开始每天30分钟课外阅读（初中难度文本）
  2. 每周一篇命题作文+一篇自由作文
  3. 整理错题本（阅读理解错题、文言文错题）
  4. 每天15分钟写字训练
⏰ 时间承诺：至少坚持一个学期才能见效。
🚫 避坑：不要寄希望于「中考冲刺班」；不要让孩子只刷题不读书。

💎 最核心的12条心法

  2. 语文英语靠积累，数学物理靠提前——语言类不能突击，理科类必须抢跑
  3. 奥数不是数学的延伸，是思维的体操——不获奖也要学
  4. 陪读的核心是「陪伴」不是「辅导」——低年级必须陪，但不是代做

  7. 「做了很多事」≠「学到了很多东西」——少而精，胜过多而杂
  8. 99%的优质教育资源是免费/低成本的——录播课>直播课，免费的资源先用起来
  9. 每天4-9点是主战场——这5小时的质量决定一年的成绩
  10. 小升初简历是「家庭项目」——不是孩子一个人的事，需要家长4年长跑

  12. 信奥99%的家庭不要碰——同等时间投入奥数性价比高10倍
