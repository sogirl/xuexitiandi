📖 发刊词：掌握自学系统方法，快速进入任何领域，学会任何知识

总纲

一、引言——自学能力缺失的时代困境

许多智力、能力和努力程度都差不多的人，学业成绩、工作成就和人生成果却相差甚远。讲师朱丹从高中起就注意到这一现象，并在随后的20年里持续探索高效学习的方法。在从事一对一学习咨询的过程中，发现人们存在各种学习问题：想学外语却无从下手、想做副业却没有方向、期望加薪却不懂技能、想教育孩子却束手无策。核心问题在于自学能力有缺失。

核心要点：努力与回报不成正比的根源在于自学能力的差异；多数人的自学能力缺失源于长期依赖他人安排学习；在AI时代，自学方法论的正确性比努力本身更重要。

二、掌握自学能力的三大好处

  1. 比别人更快掌握其他领域知识：自学力是底层能力，类似搜索力。学会自学就打通了通往其他领域的壁垒。
  2. 全面提升综合素质：自学是结合搜索、整理、逻辑、阅读、写作、记忆等多方面素质的综合能力。
  3. 将自学能力变现：在本职工作中升职加薪，同时利用知识开拓更多副业可能。

三、课程体系介绍

自学六大痛点：信息变多但思考变浅、转眼就忘、随时干扰、计划总被打乱、无法集中精力、拖延症爆发、焦虑不安、无法突破瓶颈、跨界困难。

四大板块：

  1. 入门阶段：认识自己→获取资源→找到学习对象→制定计划→付诸实践
  2. 进阶阶段：阅读技巧→信息吸收→攻克难点→管理知识→提升记忆→搭建知识体系→强化效果
  3. 应用阶段：演讲汇报→语言技能→学历提升→资格考试→投资理财
  4. 应试阶段：文科（语文/英语/历史/地理）+ 理科（数学/物理/化学/生物）全科覆盖

课程精选能帮大家提升80%学习效果的那20%精华内容，整理为"方法+技巧+工具+行动清单"。

四、为什么跟我学

理由一：讲师朱丹在香港大学完成全日制硕士，获全额奖学金攻读博士，研究信息技术如何辅助高效学习与自主学习。理由二：13年教育行业从业经验，16年高效学习方法实操经验，6年科研经验，研究过上千种学习辅助工具。理由三：覆盖考试、职场、理财、兴趣等经典自学场景，每类人都能找到自己的应用场景。

五、成功案例

  • 两周突破挂科科目，成功被清华大学录取
  • 从创业失败中转型，零基础自学自媒体，两个月涨粉近40万
  • 一个月从不知名小公司跳槽到今日头条核心部门
  • 保住全职工作还成功入读名校，硕士期间生了二胎

六、结语

自学能力就像是地基，只要掌握了强大的自学能力，想学什么就能学会什么。28天系统学习即可成为自学高手。

第01课：前期准备——认识自己，调整心态，确定合理目标

入门 认知

板块一：正确认识自己（六大方面）

  1. 价值观——你期待以什么方式在世界产生价值？你的价值追求是什么？
  2. 兴趣和爱好——能长时间让你注意力聚焦的事情
  3. 体质——了解自己的身体状况，选择适合的学习节奏
  4. 脾气秉性——了解性格选择适合的环境，关键在于扬长避短
  5. 优势——了解自己的优势是增长自信的方法之一
  6. 使命和目标——当有强烈的使命感和目标时，会在漫长人生中找到很多有趣的事情

板块二：人生罗盘模型（平衡轮）

生涯教练工具，按顺时针方向填上生命平衡与幸福最重要的八项内容。上半部分为对外发展（强目标导向），下半部分为对内发展（关系维护），最好上下平衡。讲师从20岁开始每五年以人生罗盘规划人生，每年重新梳理未来五年最重要的三个方面。

为什么以五年为一个阶段：五年时间足以让一个人从"什么都不懂的小白"做到专家级别。

板块三：方向比努力重要

两大核心建议：控制欲望（三年学一个领域 > 一年学三个领域）+ 做减法（不要被兴趣牵引忘记学习目的）。

心态调整三方面：突破难点（先学能掌握的增强自信）→ 拒绝速成心态（与刻意练习有关，与天分关系不大）→ 合理看待新知的不确定性（"学了自然就知道了"）。

板块四：学习金字塔模型（三个层次）

层次类型学习效率特征
初级听课者5%喜欢买课买书，追求"看完"
中级记录者30%记录关键词，收藏资料重复记忆
高级传播者80%分享所学，关注"会用"而非"记住"
关键转型：从关注"我学了多少"到关注"我会了多少"，从被动听课者转变为主动传播者。

第02课：获取资料——用这些方法和渠道，获取最优自学材料

入门

板块一：寻找材料前的准备工作

  1. 根据目标确认材料：目标必须非常明确，更好的做法是先做一两套真题，知道考试重点和弱点再找资料
  2. 建立初级总体印象：百科搜索（百度百科/维基百科）让对学习对象有初级印象
  3. 先做后学：找材料前先亲身操作一次，摸索时遇到不懂的记下来再展开学习

板块二：三原则两步骤

三个原则：

  1. 一手知识优先——减少信息过滤损耗（如育儿知识直接看《美国儿科学会育儿百科》原书）
  2. 兴趣导向原则——选择感兴趣的材料，过程中不会痛苦
  3. 学习区原则——刚刚好有一点难的，维持心流状态

选择难度的两个维度：自身基础 + 材料阐明的读者范围和目标。

两个步骤：第一步找自学方法（搜索顺序：知乎 > 微信搜狗 > 百度），第二步找自学材料。

板块三：五大搜索指令（核心技能）

指令格式示例用途
site中考作文 site:zhihu.com限定搜索网站
filetype2019中考真题 filetype:pdf搜索特定格式文件
时间范围人工智能 2016..2018限定时间区间
inurlinurl:gaokao.com搜索网址含指定字段页面
减号关键词 -推广排除不想要的内容

垂直搜索渠道：书籍→鸠摩搜书/多抓鱼/Library Genesis；视频→中国大学MOOC/B站/EDX/可汗学院；学术→大木虫学术导航/Edda知识搜索/谷歌学术；软件→上官方网站。

板块四：批判性吸收

所有材料都有局限性，尽信书不如无书。带着批判性思维去思考，不合适则进行二次甚至多次搜索。

第03课：学习对象——找到好的学习伙伴，让自学不再孤军奋战

入门

三种学习对象

老师：四个核心价值——及时反馈与指点、方向性指引、更准确的判断标准、答疑解惑（特别是隐性知识）。

学霸/学习小组：多角度理解、互动监督、分摊工作量、责任感促进学习质量。

学习社群：组织学习活动、随时提问、资料分享。

如何挑选

老师四准则：风格喜欢、思路清晰、能积极互动、放宽"老师"概念。

学习同伴四原则：能力差距不能太大、人数不能太多、讨论前约定任务和时长（推荐石墨文档记录）、设立组长和奖惩制度。

学习社群四要点：同一时期不超过两个、有专人运营规则明确、氛围活跃、学习社群不是社交。

搜索方法

在校生→微博/知乎/QQ；职场人士→在行APP；专家联系→邮箱搜索与推测法；线下→参加讲座/研讨会/读书会。

第04课：制定计划——用计划拆解任务，用监督克服拖延

入门

SMART法则（五个维度）

维度含义错误示范正确示范
S目标明确学好英语月考考90分以上
M可量化多读英语每天做两篇英语阅读
A可达成每天背1000单词每天背20个单词
R与角色相关脱离实际的目标与工作/学习需求关联
T有截止日期没有时间限制三个月内完成

制定步骤：确立长期目标（聚焦方向→咨询专业人士收集资料→确立目标，含课程+书籍+练习三维）→拆分为月计划和日计划。

执行记录方法

每日计划表：To do list + Inbox（干扰暂存）+ 小结（复盘反馈）。
手机APP：推荐滴答清单（大众）和时光序（免费含记录功能，有安卓/苹果/网页版）。

调整计划的六大问题

  1. 目标太多→长期目标1-3个即可
  2. 突发事件→每周留1-2天缓冲期
  3. 材料难度大→重新审视选择
  4. 无法专注→见第06课
  5. 无法抵抗诱惑→见第07课
  6. 学习状态不好→加强锻炼，疲劳期降强度

复盘总结四问

出现了什么问题？有什么心得？如何防止再出现？有什么成就感之处可以复制？

第05课：管理时间——科学利用整块时间和碎片化时间

入门

正确认识时间管理

时间管理的关键不是将每时每刻都利用起来，而是学会做选择——选出对自己最有价值的事情来做。时间的价值体现在最重要的事情上。

四象限法则

象限特征策略
第一象限重要且紧急优先处理，不能拖延
第二象限重要不紧急最具长远价值，提升自学能力的关键区域
第三象限不重要但紧急干扰源，学会甄别和减少
第四象限不重要不紧急尽量避免

时间管理具体方法

每日：四象限图记录事项，晚上写日记分析时间去向。
每月：按"固定事项→重要不紧急→休闲→空余"顺序安排，预留40%灵活变动时间。
整块时间：通过记录和分析挤出1-2小时用于深度学习。

To do list核心奥秘

  1. ABC分级法：A级（最重要）优先完成，然后B级，然后C级
  2. 排序核心：先难后易——把最难的A级项目先做完
  3. 筛选：哪些事可以授权或外包（淘宝请人处理数据等）
  4. 严格限时完成：使用倒计时或番茄钟APP

第06课：执行任务——保持专注，半天内完成所有学习任务

入门

心流理论（Flow）

由心理学家米哈里·切克森米哈赖提出。心流发生的五个条件：完全沉浸、明确目标、及时反馈、适当挑战（"踮踮脚能够到的难度"）、内在动力。

保持专注的四步执行法

第一步：执行前列代办清单。可用每日计划本（在校生）或时光序APP（职场人士）。

第二步：化整为零——番茄工作法。25分钟专注→5分钟休息→每四个番茄多休息。不能使用手机的→买机械计时器；自控力弱的→Forest（种树机制，离开APP树枯萎）；自制力好的→时光序自带专注功能。

第三步：化零为整——集中处理突发事件。先记录在计划本或备忘上再集中处理。

第四步：记录总结，根据反馈改进方法。

冥想——日常专注力练习

被科学验证的大脑运动。哈佛医学院博士Sara Zer研究表明冥想能提高专注力、减缓压力和焦虑。

操作步骤：坐好→闭眼→聚焦呼吸→走神了再回来→不纠结。建议每天5-10分钟。

工具选择速查：在校学生→每日计划本+物理计时器/Forest；大学生/职场人士→时光序APP（一站式管理）。

第07课：自学环境——内外兼修，促进自学习惯养成

入门

习惯的运作机制（习惯回路）

出发点（暗示）→ 惯常行为 → 奖赏。习惯具有自动化特性，一旦形成就会自觉执行，不会产生抗拒和排斥。

建立新习惯的方法论

常见误区：完全依赖意志力。意志力是消耗品，完全凭借意志力培养新习惯容易失败。

方法一：以旧促新。找到合适的日常习惯（如刷牙、上下班坐地铁）作为新习惯的出发点。

方法二：化大为小——微习惯策略。微小到不管发生什么一定可以完成。如想每天自学两小时，从每天五分钟开始。

方法三：设置小奖励。如喝杯咖啡、追一集综艺。

四个注意事项：一个出发点只添加一个微习惯、合理性和可操作性、建立外部监督、延迟满足（等十分钟）。

值得培养的五个好习惯

  1. 早起铺床——完成一天的第一件事，带来成就感
  2. 明确每天最重要的三件事——将80%精力用在最重要的20%事情上
  3. 记录日常开支——了解消费结构，改善财务状况
  4. 阅读——提升能力
  5. 运动——身体是革命的本钱

第08课：高效阅读——像学霸一样搞透自学材料的核心内容

进阶

高效阅读的三个基本原则

  1. 抓整体放细节：优先从整体把握资料的核心内容
  2. 化整为零：将不懂的细节知识点统一整理汇总，放在最后逐一突破
  3. 总结提炼：用自己的话语去总结对知识的理解

六步提问模型（高效阅读核心）

步骤核心问题关键动作
第一步我阅读这份材料的目的是什么？带着问题找答案，不做无用功
第二步关于这个话题我知道些什么？SWOT分析（优势/机会/劣势/威胁）
第三步这篇阅读材料的主要内容是什么？梳理框架，编织知识大网
第四步作者接下来要讲的是什么？信号词预测，与作者对话
第五步从专家角度想话题的专业问题有哪些？专业问题模板（定义/类型/关联等）
第六步针对已有信息我能提出什么问题？5W提问法（Who/What/How long/Where/Why）
实用信号词表：介绍新观点→不过/但是/然而/尽管如此；解释强调→同时/此外/不仅如此；总结→所以/因此/作为结果；提供理由→因为/基于/由于；提供清单→首先/其次/最后。

SWOT自我分析工具

S（优势）：已知道什么 W（劣势）：学起来困难的部分 O（机会）：能从书中学到什么 T（威胁）：深入研究需克服的困难。

第09课：信息吸收——高手都在巧用笔记提炼和吸收全部信息

进阶

信息吸收的三个步骤和三化状态

三步骤：收集整理知识 → 产出有效率的结果（经过自己思考）→ 用知识真实有效地解决问题。

三化状态：知识信息串联化、核心内容熟练化、转述表达简单化。

信息吸收四大注意事项

  1. 以终为始：根据目的决定哪些信息重要
  2. 调整心态：不必读完所有内容
  3. 区分信息：找重点→细嚼慢咽重要点→快速略读次要点→完全忽略其他点
  4. 分组重组：查找信息相似处和不同处

第17课：英语语言——自学英语听说读写等语言技能

应用

听力训练（英语学习第一步）

科学依据：TOEFL大数据表明听力能力决定英语整体能力上限，听力障碍会导致更严重的阅读障碍。

四步听力训练法：

  1. 裸听：不看文本听3-5遍，培养语感
  2. 文本辅助聆听：边听边读边查生词，采用"二选一"原则
  3. 循环聆听：再次聆听2-3遍
  4. 自主扩展：单词集中背诵、精听跟读、篇章背诵

听力软件：可可英语 / 每日英语听力。词典软件：有道词典 / 欧路词典 / 金山词霸。素材标准：分级有声书、以故事为主、裸听三遍听懂约50%为最合适。

口语训练（五步法）

前提：三个月听力训练基础。五步：选一分钟音频听九遍→切割20秒再听九遍→影子跟读（滞后约0.5秒）→同步跟读→录音对比。

阅读训练（三步法）

选书（生词量30%-50%）→ 设定7天目标 → 设定长期激励计划。关键：50%自适应查词原则（培养猜词能力），关注熟词僻义和习语。

写作训练

好文模仿法：模仿结构和句式。听写训练法：选TED/影视剧片段听写。四种日常实践：英文清单、英文在线评论、英文推文/博客、英文日记。

核心原则：提升写作的本质是增加输入，以输入带动输出。听力是所有英语技能的基石。

第22课：理论综述——高效掌握任意一个学科的自学方法

理论

学习公式

好的成绩 = 1%天分 + 49.5%努力 + 49.5%方法。没有方法的努力很可能是白忙一场。

两大学习原则

  1. 贪多不如嚼细：每个知识点吃透，每个定理尽量自己证明一遍。2019年最年轻院士孙斌勇："学习不能图快，要慢和细，稳最重要。"
  2. 学会主动学习：真正去运用所学知识，通过建立知识框架让大脑主动吸收消化知识

建立知识框架三步法

  1. 看课本目录，画出大体知识框架：用5W1H六个维度细化。理科用2W1H（What-Why-How），文科加Where-When-Who
  2. 复习旧知识，补充框架：学完新课后补充预习时问题的解答
  3. 吃透知识点后完善框架：标注常考内容和举一反三用得上的

三个受益终生的习惯

阅读（持续输入）、健身（精力管理）、反思日记（迭代进步）。培养方法：从微习惯开始，如每天读一页书。

第23课：理科学习——数学、物理、化学等学科自学法

理论

理科学习的本质

理科学习的本质是寻找规律并运用规律。不需要死记硬背每一个公式，而是要理解公式之间的推导关系。长方形的面积公式可推出正方形、梯形、平行四边形的面积公式。

理科自学三大步

步骤内容进入下一阶段标准
第一步吃透课本所有概念/定理/公式（用活页本整理）同桌随便问可回答出来
第二步吃透课本例题（基础性和代表性）考试成绩达到80%很有可性
第三步吃透近8-10年真题掌握出题人思路

正确的刷题方法

四步推导法：画出已知条件→推导中间信息→根据目标反向推导→找到突破口。

专题专攻：一类题型一类题型地攻克。刷到卡壳→记下卡壳地方→看答案重新做→独立做→入错题本→第二天再做。

成绩分级策略：60分以下→打基础概念定理；60-80分→刷中等难度题；80分以上→攻克难题。

现代文阅读多阅读做读书笔记（长期功夫）+ 总结题型与答题逻辑（短期提分，如记叙文线索/作用等）
作文读范文背好词佳句（积累层）+ 仿写范文（技巧层——换话题模仿结构）

英语应试补充

单词：随身单词本 + 碎片化时间记忆。语法：先理解再记忆 + 思维导图总结。阅读：课外多读（推荐百词斩爱阅读）+ 总结解题技巧（主旨题/猜词题/细节题等）。作文：积累常用句型 + 积累同义表达 + 仿写。

第25课：小科目学习——史、地、生、政等学科自学法

小科目

破除三大偏见

  1. 文科需要死记硬背？× → 要找到科学记忆方法 ✓
  2. 文科不需要重视逻辑？× → 议论文/说明文/记叙文都体现逻辑 ✓
  3. 以课本为主？× → 教考分离是趋势，需扩展学习 ✓

文科自学实操三步骤

  1. 掌握课本：用思维导图做出大纲，建构知识框架
  2. 扩展背景知识：课外阅读 + 随手查阅（"今日事今日毕"）+ 搜题软件（作业帮/小猿搜题）+ 百度百科
  3. 科学练习：首选高考真题，准备活页错题本。文科错题本关键：总结答题思路，而非像理科那样一遍遍解题。举例历史"问原因"类题型：定位时空→多角度分析（政经文+中外+内外因）→总结

科学记忆五步法（核心方法）

步骤操作
第一步判断文本类型：描述性文本（可用5W1H概括）vs 逻辑性文本（总分/分总/总分总结构）
第二步拆解文本提取关键词：描述性按5W1H拆解，逻辑性按结构拆解+二次拆分
第三步以关键词为线索回忆文本（主动回忆Active Recall）
第四步对比原文本得出反馈，标记薄弱内容
第五步利用费曼技巧，不断复述给自己听——关键：换一种描述方式

家长引导低年级孩子三阶段

示范阶段（家长做拆分示范）→ 引导阶段（提问引导孩子思考关键词）→ 独立阶段（形成习惯）。核心原则：培养意识 > 严格执行流程。

第26课：考试实操——自学通过所有科目考试，越考越好

应试

考试心态稳定

耶克斯-多德森定律：对于困难任务，中等强度的动机水平最有利于任务完成。
树立自信：列出影响自信的原因逐一反驳 + 运用积极暗示语（"我完全有能力应对考试"）。
考前脱敏训练：考前一周每天睡前细致想象从进考场到出考场的完整流程。
考场缓解紧张：深呼吸法、小口喝水法。

错题本制作7步骤（核心方法）

  1. 准备活页本
  2. 粘贴/抄写题目
  3. 弄懂题目——问老师/同学/看参考答案
  4. 用铅笔重做（隔一天后）——最关键的一步，打破"我会了"的假象
  5. 用红笔/蓝笔写技巧总结和反思
  6. 按知识点和题型归类
  7. 临考前再次回顾——擦掉答案再做一遍
错题本专区拓展：技巧总结专区、常用公式专区、常犯错误专区、高级表达转换专区等。

临场考试技巧

标注关键信息：用笔将题目关键信息逐个标注。规范打草稿：分块标题号，答案单独写。先易后难：应对舌尖现象（Tip-of-the-Tongue）——暂时跳过，后续再回看。

第27课：偏科攻克——自学搞定严重偏科的方法

应试

补差两大误区

误区一：把重心全放在弱科上，忽略优势学科的薄弱点。误区二：把重心全放在优势学科上，放弃弱势学科。在研究生阶段之前的学习，不需要用天赋为自己的成绩设限。

五步实操法

步骤操作
第一步列排名表格，最多选3科（含1科优势补差）
第二步活页本，只记知识点不记题目
第三步每天2小时逐个攻克——理科重概念定理+独立做题，文科重背诵
第四步每周末做题巩固（首选学校习题集+网上习题）
第五步1-2周后单元卷检测，循环反馈

三大注意事项

专注不贪多（攻克一个打一个勾）+ 懂得取舍（高难度知识点可暂时放弃，优先高投入产出比）+ 合理安排时间（70%课余时间弱科，30%维持其他学科）。

第28课：论文写作——自学也能写出好论文的方法

论文

论文选题四原则

题目要具体 + 题目要熟悉 + 选题前做好资料搜集 + 选题尽量有新意（创新性与熟悉性需平衡）。

论文九部分结构

①封面与声明 → ②中英文摘要与关键词（建议全文完成后再写）→ ③目录（Word自动生成）→ ④绪论（研究背景/现状/总结优劣）→ ⑤正文（章节-论点-小结三层结构）→ ⑥总结展望 → ⑦参考文献（谷歌学术/百度学术获取格式）→ ⑧附录 → ⑨致谢

文献查阅渠道

中国知网、百度学术、万方数据、谷歌学术、维普网。三点注意：搜索高影响力期刊文章、用关键词按相关性排序、中英文文献兼顾。

查重与降重

官方查重平台：中国知网、维普网、万方数据。不要使用小平台查重。

降重五种方法（按推荐程度排序）：

  1. 用自己的话复述他人观点（首选，重复率最低）
  2. 插入和转换——插入词、同义转换、转换句式
  3. 多利用国外论文资料——翻译引用
  4. 文字改为图片（图片不参与查重）
  5. 谷歌翻译"回译"法——翻译成英文再译回中文（走投无路时使用）
